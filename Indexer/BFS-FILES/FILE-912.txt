      <p>
       In celebration of the
       <a href="/wiki/Solar_eclipse_of_August_11,_1999" title="Solar eclipse of August 11, 1999">
        total solar eclipse of August 11, 1999
       </a>
       , the
       <a href="/wiki/National_Bank_of_Romania" title="National Bank of Romania">
        National Bank of Romania
       </a>
       (BNR) decided to issue a commemorative two thousand
       <a class="mw-redirect" href="/wiki/Romanian_lei" title="Romanian lei">
        Romanian lei
       </a>
       banknote. This was the last solar eclipse of the millennium and was visible across southern
       <a href="/wiki/Romania" title="Romania">
        Romania
       </a>
       .
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
       Since it was the last eclipse of the millennium, the denomination was chosen to be
       <i>
        2000
       </i>
       in respect to the upcoming year. These notes were issued as legal tender.
      </p>

      </p>

      <h2>
       <span class="mw-headline" id="Design">
        Design
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Two_thousand_lei&amp;action=edit&amp;section=1" title="Edit section: Design">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       The banknote was designed by Romanian artist Nicolae Săftoiu, who is credited with the designs on which all Romanian banknotes printed since the
       <a href="/wiki/Romanian_Revolution" title="Romanian Revolution">
        1989 Revolution
       </a>
       are based.
       <sup class="reference" id="cite_ref-2">
        <a href="#cite_note-2">
         [2]
        </a>
       </sup>
       In observance of the upcoming millennium, the obverse of the note displays a rendering of the
       <a class="mw-redirect" href="/wiki/Solar_system" title="Solar system">
        solar system
       </a>
       viewed from afar, showing all of the planets revolving around the
       <a href="/wiki/Sun" title="Sun">
        Sun
       </a>
       .
       <sup class="reference" id="cite_ref-hobby_3-0">
        <a href="#cite_note-hobby-3">
         [3]
        </a>
       </sup>
      </p>
      <p>
       The reverse of the note displays a map outline of Romania with the colors schemed to match the colors of the Romanian flag. The map marks the main points where the solar eclipse was visible in a path moving along the map from west to east.
       <sup class="reference" id="cite_ref-hobby_3-1">
        <a href="#cite_note-hobby-3">
         [3]
        </a>
       </sup>
      </p>
      <p>
       The notes were printed using a method of printing known as
       <a href="/wiki/Offset_printing" title="Offset printing">
        offset printing
       </a>
       . Offset printing is a commonly used printing technique where the inked image is transferred (or "offset") from a plate to a rubber blanket, then finally onto the printing surface.
       <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
        [
        <i>
         <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
          <span title="This claim needs references to reliable sources. (July 2009)">
           citation needed
          </span>
         </a>
        </i>
        ]
       </sup>
      </p>
      <h3>
       <span class="mw-headline" id="Main_security_features">
        Main security features
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Two_thousand_lei&amp;action=edit&amp;section=2" title="Edit section: Main security features">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <div class="thumb tright">
       <div class="thumbinner" style="width:222px;">
        <a class="image" href="/wiki/File:Rome_sec.features.JPG">
         <img alt="" class="thumbimage" data-file-height="480" data-file-width="640" height="165" src="//upload.wikimedia.org/wikipedia/en/thumb/a/a1/Rome_sec.features.JPG/220px-Rome_sec.features.JPG" srcset="//upload.wikimedia.org/wikipedia/en/thumb/a/a1/Rome_sec.features.JPG/330px-Rome_sec.features.JPG 1.5x, //upload.wikimedia.org/wikipedia/en/thumb/a/a1/Rome_sec.features.JPG/440px-Rome_sec.features.JPG 2x" width="220"/>
        </a>
        <div class="thumbcaption">
         <div class="magnify">
          <a class="internal" href="/wiki/File:Rome_sec.features.JPG" title="Enlarge">
          </a>
         </div>
         Security features, including the thread, transparent window, and shadow image
        </div>
       </div>
      </div>
      <p>
       Security features include:
      </p>
      <ul>
       <li>
        <b>
         Transparent window
        </b>
        - special in specifically polymer notes. It is an open, clear and plastic window in the note which is hard to counterfeit.
       </li>
       <li>
        <b>
         Shadow image (Watermark)
        </b>
        - includes the
        <a href="/wiki/National_Bank_of_Romania" title="National Bank of Romania">
         BNR
        </a>
        logo. This image can be seen when the banknote is held to the light and is embedded into the plastic note.
       </li>
       <li>
        <b>
         See-through registration
        </b>
        - optical see-through image on each side of the note to make sure it was not printed using a counterfeit, in which case it would not match and align
       </li>
       <li>
        <b>
         Printed thread
        </b>
        - a black thread that could be see when held up to the light
        <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
         [
         <i>
          <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
           <span title="This claim needs references to reliable sources. (July 2009)">
            citation needed
           </span>
          </a>
         </i>
         ]
        </sup>
       </li>
      </ul>
      <h2>
       <span class="mw-headline" id="Commemorative_folder">

       <span class="mw-editsection">
