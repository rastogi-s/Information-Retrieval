       Confirmed as

       <a href="/wiki/William_Shakespeare" title="William Shakespeare">

       </a>
       's play
       <i>
        <a class="mw-redirect" href="/wiki/The_Tempest_(play)" title="The Tempest (play)">
         The Tempest
        </a>
       </i>
       in August 2000.
       <sup class="reference" id="cite_ref-IAUC_7479_9-0">
        <a href="#cite_note-IAUC_7479-9">
         [9]
        </a>
       </sup>
      </p>
      <p>
       The orbital parameters suggest that it may belong to the same dynamic cluster as
       <a href="/wiki/Caliban_(moon)" title="Caliban (moon)">
        Caliban
       </a>
       , suggesting common origin.

        </a>
