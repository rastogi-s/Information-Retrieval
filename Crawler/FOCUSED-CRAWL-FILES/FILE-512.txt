      <p>
       <i>
        <b>
         Moondust
        </b>
       </i>
       is a 1983
       <a href="/wiki/Music_video_game#Generative_music_games" title="Music video game">
        generative music video game
       </a>
       created for the
       <a href="/wiki/Commodore_64" title="Commodore 64">
        Commodore 64
       </a>
       by
       <a href="/wiki/Virtual_reality" title="Virtual reality">
        virtual reality
       </a>
       pioneer,
       <a href="/wiki/Jaron_Lanier" title="Jaron Lanier">
        Jaron Lanier
       </a>
       .
       <i>
        Moondust
       </i>
       was programmed in
       <a href="/wiki/MOS_Technology_6502" title="MOS Technology 6502">
        6502 assembly
       </a>
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
       in 1982,
       <sup class="reference" id="cite_ref-2">
        <a href="#cite_note-2">
         [2]
        </a>
       </sup>
       and is considered the first
       <a class="mw-redirect" href="/wiki/Art_video_game" title="Art video game">
        art video game
       </a>
       .
       <sup class="reference" id="cite_ref-3">
        <a href="#cite_note-3">
         [3]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-4">
        <a href="#cite_note-4">
         [4]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-5">
        <a href="#cite_note-5">
         [5]
        </a>
       </sup>
       <i>
        Moondust
       </i>
       has frequently been used as an art installation piece in museum exhibitions
       <sup class="reference" id="cite_ref-6">
        <a href="#cite_note-6">
         [6]
        </a>
       </sup>
       from
       <a href="/wiki/Corcoran_Gallery_of_Art" title="Corcoran Gallery of Art">
        Corcoran Gallery of Art
       </a>
       's 1983 "ARTcade"
       <sup class="reference" id="cite_ref-7">
        <a href="#cite_note-7">
         [7]
        </a>
       </sup>
       to the
       <a href="/wiki/Smithsonian_Institution" title="Smithsonian Institution">
        Smithsonian
       </a>
       's 2012 "
       <a href="/wiki/The_Art_of_Video_Games" title="The Art of Video Games">
        The Art of Video Games
       </a>
       ".
       <sup class="reference" id="cite_ref-8">
        <a href="#cite_note-8">
         [8]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-9">
        <a href="#cite_note-9">
         [9]
        </a>
       </sup>
       It has also been used by Lanier and others in papers and lectures as an example to demonstrate the unexpected
       <a class="mw-redirect" href="/wiki/Ephemeral" title="Ephemeral">
        ephemerality
       </a>
       of digital data.
       <sup class="reference" id="cite_ref-10">
        <a href="#cite_note-10">
         [10]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-11">
        <a href="#cite_note-11">
         [11]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-12">
        <a href="#cite_note-12">
         [12]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-13">
        <a href="#cite_note-13">
         [13]
        </a>
       </sup>
      </p>
      <p>
       <i>
        Moondust
       </i>
       is also considered to be the first
       <a href="/wiki/Music_video_game" title="Music video game">
        interactive music publication
       </a>
       ,
       <sup class="reference" id="cite_ref-14">
        <a href="#cite_note-14">
         [14]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-15">
        <a href="#cite_note-15">
         [15]
        </a>
       </sup>
       and it sold quite successfully.
       <sup class="reference" id="cite_ref-16">
        <a href="#cite_note-16">
         [16]
        </a>
       </sup>
       With the profits from
       <i>
        Moondust
       </i>
       <sup class="reference" id="cite_ref-17">
        <a href="#cite_note-17">
         [17]
        </a>
       </sup>
       and additional funding from
       <a href="/wiki/Marvin_Minsky" title="Marvin Minsky">
        Marvin Minsky
       </a>
       ,
       <sup class="reference" id="cite_ref-18">
        <a href="#cite_note-18">
         [18]
        </a>
       </sup>
       Lanier formed VPL which would later go on to create the
       <a class="mw-redirect" href="/wiki/DataGlove" title="DataGlove">
        DataGlove
       </a>
       and the
       <a class="new" href="/w/index.php?title=DataSuit&amp;action=edit&amp;redlink=1" title="DataSuit (page does not exist)">
        DataSuit
       </a>
       <sup class="reference" id="cite_ref-19">
        <a href="#cite_note-19">
         [19]
        </a>
       </sup>
       and to become one of the primary innovators of virtual-reality research and development throughout the 1980s.
      </p>

      <h2>
       <span class="mw-headline" id="Gameplay">
        Gameplay
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Moondust_(video_game)&amp;action=edit&amp;section=1" title="Edit section: Gameplay">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <div class="thumb tleft">
       <div class="thumbinner" style="width:222px;">
        <a class="image" href="/wiki/File:Moon_dust_01.gif">
         <img alt="" class="thumbimage" data-file-height="200" data-file-width="320" height="138" src="//upload.wikimedia.org/wikipedia/en/thumb/7/7e/Moon_dust_01.gif/220px-Moon_dust_01.gif" srcset="//upload.wikimedia.org/wikipedia/en/7/7e/Moon_dust_01.gif 1.5x" width="220"/>
        </a>
        <div class="thumbcaption">
         <div class="magnify">
          <a class="internal" href="/wiki/File:Moon_dust_01.gif" title="Enlarge">
          </a>
         </div>
         Screenshot of gameplay.
        </div>
       </div>
      </div>
      <p>
       <i>
        Moondust'
       </i>
       s gameplay is characterized by graphical complexity,
       <sup class="reference" id="cite_ref-20">
        <a href="#cite_note-20">
         [20]
        </a>
       </sup>
       and the game features an abstract ambient score.
       <sup class="reference" id="cite_ref-21">
        <a href="#cite_note-21">
         [21]
        </a>
       </sup>
       The goal of the game is to cover the bullseye at the center of the screen with "moonjuice." To do this, the player controls
       <a href="/wiki/Astronaut" title="Astronaut">
        spaceman
       </a>
       Jose Scriabin (named in honor of
       <a class="mw-redirect" href="/wiki/Synaesthetic" title="Synaesthetic">
        synaesthetic
       </a>
       composer
       <a href="/wiki/Alexander_Scriabin" title="Alexander Scriabin">
        Alexander Scriabin
       </a>
       )
       <sup class="reference" id="cite_ref-blakeman_22-0">
        <a href="#cite_note-blakeman-22">
         [22]
        </a>
       </sup>
       as he drops a seed square and then moves around the screen in arcing paths to run over the seed, smearing it to cover the bullseye. As Jose travels, flight patterns of he and the bullet-shaped
       <a href="/wiki/Spacecraft" title="Spacecraft">
        spaceships
       </a>
       he must avoid are created and as they pass through the trails that are created, and as the moonjuice spreads and smears, the musical score is modified according to a generative algorithm.
       <sup class="reference" id="cite_ref-23">
        <a href="#cite_note-23">
         [23]
        </a>
       </sup>
       In-game scoring system assigns point-values according to an algorithm when the level is completed. Players start with three seeds but may acquire more if they have scored highly enough.
       <sup class="reference" id="cite_ref-blakeman_22-1">
        <a href="#cite_note-blakeman-22">
         [22]
        </a>
       </sup>
       The game has been compared to the works of
       <a href="/wiki/Jeff_Minter" title="Jeff Minter">
        Jeff Minter
       </a>
       .
       <sup class="reference" id="cite_ref-24">
        <a href="#cite_note-24">
         [24]
        </a>
       </sup>
      </p>
      <p>
       The game features 4 modes: Beginner, Evasive, Freestyle, and Spinsanity. In Beginner mode, the seeds that Jose must smear remain in one position wherever they had been dropped. In Evasive mode, however, the seeds seek to evade Jose by rushing to the edge of the screen. Freestyle mode greatly increases the players control over Jose, removing the physical element of momentum to connect Jose's motions directly to the joystick controller's. In Spinsanity mode, the spaceships travel in a spiral pattern making smearing of the seed more difficult for the player.
       <sup class="reference" id="cite_ref-blakeman_22-2">
        <a href="#cite_note-blakeman-22">
         [22]
        </a>
       </sup>
      </p>
      <h2>
       <span class="mw-headline" id="Reception">
        Reception
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Moondust_(video_game)&amp;action=edit&amp;section=2" title="Edit section: Reception">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       <i>
        <a href="/wiki/Ahoy!" title="Ahoy!">
         Ahoy!
        </a>
       </i>
       <span class="nowrap" style="padding-left:0.1em;">
        '
       </span>
       s review began "And now for something completely different. You have never seen anything like
       <i>
        Moondust
       </i>
       ... No mazes, no nuke the alien aggressors, no 'find ring, sit on ring'". The magazine, which gave the game a grade of B for graphics and A for gameplay, stated that it was "not for everybody", warning that "the instructions are confusing. Even the people at Creative Software didn't know what Lanier was up to". The review nonetheless concluded "Players shopping for a non-violent game that incorporates graphics and music will want to consider
       <i>
        Moondust
       </i>
       ... I recommend it".
       <sup class="reference" id="cite_ref-michaels198401_25-0">
        <a href="#cite_note-michaels198401-25">
         [25]
        </a>
       </sup>
       <i>
        The Commodore 64 Home Companion
       </i>
       agreed on
       <i>
        Moondust
       </i>
       <span class="nowrap" style="padding-left:0.1em;">
        '
       </span>
       s uniqueness, describing it as "one of the most creative new games I've seen for any computer". Describing gameplay as "more like ballet than combat", the book concluded that it "is a computer game for people who don't like computer games".
       <sup class="reference" id="cite_ref-C64HC_26-0">
        <a href="#cite_note-C64HC-26">
         [26]
        </a>
       </sup>
      </p>
