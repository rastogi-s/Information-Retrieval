      <p>
       A
       <b>
        circumlunar trajectory
       </b>
       ,
       <b>
        trans-lunar trajectory
       </b>
       or
       <b>
        lunar free return
       </b>
       is a type of
       <a class="mw-redirect" href="/wiki/Free_return_trajectory" title="Free return trajectory">
        free return trajectory
       </a>
       which takes a
       <a href="/wiki/Spacecraft" title="Spacecraft">
        spacecraft
       </a>
       from Earth, around the
       <a href="/wiki/Far_side_of_the_Moon" title="Far side of the Moon">
        far side
       </a>
       of the
       <a href="/wiki/Moon" title="Moon">
        Moon
       </a>
       , and back to Earth using only gravity once the initial trajectory is set.
      </p>
      <h2>
       <span class="mw-headline" id="Background">
        Background
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Circumlunar_trajectory&amp;action=edit&amp;section=1" title="Edit section: Background">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       The first spacecraft to fly a circumlunar trajectory was
       <a href="/wiki/Luna_3" title="Luna 3">
        Luna 3
       </a>
       . Circumlunar trajectories were also used by
       <a href="/wiki/Apollo_program" title="Apollo program">
        Apollo
       </a>
       missions prior to lunar orbit insertion,
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
       to provide a free return to Earth in the event of a propulsion system malfunction on the way to the Moon. This was used on
       <a href="/wiki/Apollo_13" title="Apollo 13">
        Apollo 13
       </a>
       , when an oxygen tank rupture necessitated return to Earth without firing the
       <a class="mw-redirect" href="/wiki/Apollo_Service_Module" title="Apollo Service Module">
        Service Module
       </a>
       engine, although a number of course corrections using the
       <a href="/wiki/Apollo_Lunar_Module" title="Apollo Lunar Module">
        Lunar Module
       </a>
       <a href="/wiki/Descent_Propulsion_System" title="Descent Propulsion System">
        descent engine
       </a>
       were required to maintain this trajectory.
       <sup class="reference" id="cite_ref-2">
        <a href="#cite_note-2">
         [2]
        </a>
       </sup>
      </p>
      <p>
       A number of manned missions were also proposed to intentionally conduct circumlunar flybys, including the Soviet
       <a href="/wiki/Soyuz_7K-L1" title="Soyuz 7K-L1">
        Soyuz 7K-L1
       </a>
       or
       <a href="/wiki/Zond_program" title="Zond program">
        Zond
       </a>
       programme, and several US proposals, including
       <a class="mw-redirect" href="/wiki/Gemini_program" title="Gemini program">
        Gemini
       </a>
       -
       <a href="/wiki/Centaur_(rocket_stage)" title="Centaur (rocket stage)">
        Centaur
       </a>
       and an early Apollo proposal.
       <sup class="reference" id="cite_ref-3">
        <a href="#cite_note-3">
         [3]
        </a>
       </sup>
      </p>
