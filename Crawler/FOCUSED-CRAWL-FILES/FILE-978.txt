       </b>
       <span class="nowrap">
        <span class="IPA nopopups noexcerpt">
         <a href="/wiki/Help:IPA/English" title="Help:IPA/English">

        trans-Neptunian object

       <a class="mw-redirect" href="/wiki/C._A._Trujillo" title="C. A. Trujillo">

       </a>
       and
       <a href="/wiki/Michael_E._Brown" title="Michael E. Brown">
        M. Brown
       </a>
       at
       <a href="/wiki/Palomar_Observatory" title="Palomar Observatory">
        Palomar
       </a>

       <a href="/wiki/Ceto" title="Ceto">
        Ceto

       from Greek mythology. The object was identified as a
       <a href="/wiki/Binary_asteroid" title="Binary asteroid">
        binary
       </a>

       <b>
        Phorcys
       </b>
       (
       <span class="nowrap">
        <span class="IPA nopopups noexcerpt">
         <a href="/wiki/Help:IPA/English" title="Help:IPA/English">

      <h2>
       <span class="mw-headline" id="Physical_characteristics">
        Physical characteristics
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=65489_Ceto&amp;action=edit&amp;section=1" title="Edit section: Physical characteristics">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       65489 Ceto is an example of a close binary TNO system in which the components are of similar size. Combined observations with the infrared
       <a href="/wiki/Spitzer_Space_Telescope" title="Spitzer Space Telescope">
        Spitzer Space Telescope
       </a>
       and the
       <a class="mw-redirect" href="/wiki/Hubble_Telescope" title="Hubble Telescope">
        Hubble Telescope
       </a>
       allow the diameter of Ceto itself to be estimated at
       <span class="nowrap">
        <span class="sortkey" style="display:none">
         7005174000000000000♠
        </span>
        174
        <span style="margin-left:0.3em;">
         <span style="display:inline-block;margin-bottom:-0.3em;vertical-align:-0.4em;line-height:1.2em;font-size:85%;text-align:right;">
          +16
          <br/>
          −18
         </span>
        </span>
        km
       </span>
       and the diameter of Phorcys at
       <span class="nowrap">
        <span class="sortkey" style="display:none">
         7005132000000000000♠
        </span>
        132
        <span style="margin-left:0.3em;">
         <span style="display:inline-block;margin-bottom:-0.3em;vertical-align:-0.4em;line-height:1.2em;font-size:85%;text-align:right;">
          +6
          <br/>
          −14
         </span>
        </span>
        km
       </span>
       , assuming equal
       <a href="/wiki/Albedo" title="Albedo">
        albedo
       </a>
       for both components.
       <sup class="reference" id="cite_ref-Grundy2007_3-7">
        <a href="#cite_note-Grundy2007-3">
         [3]
        </a>
       </sup>
      </p>
      <p>
       The binary nature of Ceto enables direct calculation of the system mass, allowing estimation of the masses of the components and providing additional constraints on their composition. The estimated
       <a href="/wiki/Density" title="Density">
        density
       </a>
       of Ceto is
       <span class="nowrap">
        <span class="sortkey" style="display:none">
         7003137000000000000♠
        </span>
        1.37
        <span style="margin-left:0.3em;">
         <span style="display:inline-block;margin-bottom:-0.3em;vertical-align:-0.4em;line-height:1.2em;font-size:85%;text-align:right;">
          +0.66
          <br/>
          −0.32
         </span>
        </span>
        g/cm
        <sup>
         3
        </sup>
       </span>
       , significantly less than that of the large TNOs (
       <a class="mw-redirect" href="/wiki/Haumea_(dwarf_planet)" title="Haumea (dwarf planet)">
        Haumea
       </a>
       : 3.0 g/cm
       <sup>
        3
       </sup>
       ,
       <a href="/wiki/Eris_(dwarf_planet)" title="Eris (dwarf planet)">
        Eris
       </a>
       : 2.26,
       <a href="/wiki/Pluto" title="Pluto">
        Pluto
       </a>
       : 2.03,
       <a href="/wiki/Charon_(moon)" title="Charon (moon)">
        Charon
       </a>
       : 1.65) but significantly more than that of smaller TNOs (e.g. 0.7 g/cm
       <sup>
        3
       </sup>
       for
       <span class="nowrap">
        <a href="/wiki/(26308)_1998_SM165" title="(26308) 1998 SM165">
         (26308) 1998 SM
         <span style="position: relative; top: 0.3em;">
          <span style="font-size:80%;">
           165
          </span>
         </span>
        </a>
       </span>
       ). Phorcys has a mass of about 1.67×10
       <sup>
        18
       </sup>
       kg.
       <sup class="reference" id="cite_ref-Grundy2007_3-8">
        <a href="#cite_note-Grundy2007-3">
         [3]
        </a>
       </sup>
       Unless the bodies are
       <a href="/wiki/Porosity" title="Porosity">
        porous
       </a>
       , the density is consistent with rock–ice composition, with rock content around 50%.
       <sup class="reference" id="cite_ref-Grundy2007_3-9">
        <a href="#cite_note-Grundy2007-3">
         [3]
        </a>
       </sup>
      </p>
      <p>
       It has been suggested that
       <a class="mw-redirect" href="/wiki/Tidal_forces" title="Tidal forces">
        tidal forces
       </a>
       , together with other potential heat sources (e.g. collisions or
       <sup>
        26
       </sup>
       <a href="/wiki/Aluminium" title="Aluminium">
        Al
       </a>
       <a href="/wiki/Radioactive_decay" title="Radioactive decay">
        decay
       </a>
       ) might have raised the temperature sufficiently to crystallise
       <a href="/wiki/Amorphous_solid" title="Amorphous solid">
        amorphous
       </a>
       ice and reduce the void space inside the object. The same tidal forces could be responsible for the quasi-circular orbits of the components of Ceto.
       <sup class="reference" id="cite_ref-Grundy2007_3-10">
        <a href="#cite_note-Grundy2007-3">
         [3]
        </a>
       </sup>
      </p>
      <p>
       Ceto is listed on
       <a href="/wiki/Michael_E._Brown" title="Michael E. Brown">
        Michael Brown
       </a>
       's website as
       <a href="/wiki/List_of_possible_dwarf_planets" title="List of possible dwarf planets">
        possibly a dwarf planet
       </a>
       .
       <sup class="reference" id="cite_ref-Brown-dplist_6-0">
        <a href="#cite_note-Brown-dplist-6">
         [6]
        </a>
       </sup>
      </p>
