   <h1 class="firstHeading" id="firstHeading" lang="en">
    Lunar orbit rendezvous
   </h1>
   <div class="mw-body-content" id="bodyContent">
    <div class="noprint" id="siteSub">
     From Wikipedia, the free encyclopedia
    </div>

    </div>

      <p>
       <b>
        Lunar orbit rendezvous
       </b>
       (
       <b>
        LOR
       </b>
       ) is a key concept for efficiently landing humans on the Moon and returning them to Earth. It was utilized for the
       <a class="mw-redirect" href="/wiki/Project_Apollo" title="Project Apollo">
        Project Apollo
       </a>
       missions in the 1960s and 1970s. In a LOR mission, a main
       <a href="/wiki/Spacecraft" title="Spacecraft">
        spacecraft
       </a>
       and a smaller lunar lander travel to
       <a href="/wiki/Lunar_orbit" title="Lunar orbit">
        lunar orbit
       </a>
       . The lunar lander then independently descends to the surface of the Moon, while the main spacecraft remains in lunar orbit. After completion of the mission there, the lander returns to lunar orbit to
       <a href="/wiki/Space_rendezvous" title="Space rendezvous">
        rendezvous
       </a>
       and re-
       <a href="/wiki/Docking_and_berthing_of_spacecraft" title="Docking and berthing of spacecraft">
        dock
       </a>
       with the main spacecraft, then is discarded after transfer of crew and payload. Only the main spacecraft returns to Earth.
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
      </p>
      <div class="thumb tright">
       <div class="thumbinner" style="width:222px;">
        <a class="image" href="/wiki/File:LOR_Gravity_Well.png">
         <img alt="" class="thumbimage" data-file-height="600" data-file-width="1200" height="110" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/3d/LOR_Gravity_Well.png/220px-LOR_Gravity_Well.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/3d/LOR_Gravity_Well.png/330px-LOR_Gravity_Well.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/3d/LOR_Gravity_Well.png/440px-LOR_Gravity_Well.png 2x" width="220"/>
        </a>
        <div class="thumbcaption">
         <div class="magnify">
          <a class="internal" href="/wiki/File:LOR_Gravity_Well.png" title="Enlarge">
          </a>
         </div>
         Representation of the lunar
         <a href="/wiki/Gravity_well" title="Gravity well">
          gravity well
         </a>
         , illustrating how resources needed only for the trip home don't have to be carried down and back up the "well"
        </div>
       </div>
      </div>
      <p>
       Lunar orbit rendezvous was first known to be proposed in 1919 by
       <a href="/wiki/Soviet_Union" title="Soviet Union">
        Soviet
       </a>
       engineer
       <a href="/wiki/Yuri_Kondratyuk" title="Yuri Kondratyuk">
        Yuri Kondratyuk
       </a>
       ,
       <sup class="reference" id="cite_ref-FOOTNOTEHarvey20076—7_2-0">
        <a href="#cite_note-FOOTNOTEHarvey20076—7-2">
         [2]
        </a>
       </sup>
       as the most economical way of sending a human on a round-trip journey to the Moon.
       <sup class="reference" id="cite_ref-FOOTNOTEWilford196941-48_3-0">
        <a href="#cite_note-FOOTNOTEWilford196941-48-3">
         [3]
        </a>
       </sup>
      </p>
      <div class="thumb tright">
       <div class="thumbinner" style="width:222px;">
        <a class="image" href="/wiki/File:Shuttleclunar.svg">
         <img alt="" class="thumbimage" data-file-height="575" data-file-width="841" height="150" src="//upload.wikimedia.org/wikipedia/commons/thumb/5/55/Shuttleclunar.svg/220px-Shuttleclunar.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/5/55/Shuttleclunar.svg/330px-Shuttleclunar.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/5/55/Shuttleclunar.svg/440px-Shuttleclunar.svg.png 2x" width="220"/>
        </a>
        <div class="thumbcaption">
         <div class="magnify">
          <a class="internal" href="/wiki/File:Shuttleclunar.svg" title="Enlarge">
          </a>
         </div>
         LOR using
         <a href="/wiki/Shuttle-Derived_Heavy_Lift_Launch_Vehicle" title="Shuttle-Derived Heavy Lift Launch Vehicle">
          Shuttle-Derived Heavy Lift Launch Vehicle
         </a>
        </div>
       </div>
      </div>
      <p>
       The most famous example involved
       <a href="/wiki/Apollo_Command/Service_Module" title="Apollo Command/Service Module">
        Apollo CSM
       </a>
       and
       <a href="/wiki/Apollo_Lunar_Module" title="Apollo Lunar Module">
        Apollo LM
       </a>
       , where they were both sent to a Translunar flight in a single rocket stack. However, variants where the landers and main spacecraft travel separately, such as the lunar landing plan proposed for
       <a href="/wiki/Shuttle-Derived_Heavy_Lift_Launch_Vehicle" title="Shuttle-Derived Heavy Lift Launch Vehicle">
        Shuttle-Derived Heavy Lift Launch Vehicle
       </a>
       and
       <a href="/wiki/Golden_Spike_Company" title="Golden Spike Company">
        Golden Spike
       </a>
       , are also considered as Lunar Orbit rendezvous.
      </p>
      <p>
      </p>

      <h2>
       <span class="mw-headline" id="Advantages_and_disadvantages">
        Advantages and disadvantages
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Lunar_orbit_rendezvous&amp;action=edit&amp;section=1" title="Edit section: Advantages and disadvantages">

        <span class="mw-editsection-bracket">

        </span>
       </span>
      </h2>
      <h3>
       <span class="mw-headline" id="Advantages">
        Advantages
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Lunar_orbit_rendezvous&amp;action=edit&amp;section=2" title="Edit section: Advantages">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <div class="thumb tright">
       <div class="thumbinner" style="width:222px;">
        <a class="image" href="/wiki/File:Comparison_of_Lander_Sizes_-_Direct_vs_LOR.gif">
         <img alt="" class="thumbimage" data-file-height="370" data-file-width="461" height="177" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/41/Comparison_of_Lander_Sizes_-_Direct_vs_LOR.gif/220px-Comparison_of_Lander_Sizes_-_Direct_vs_LOR.gif" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/41/Comparison_of_Lander_Sizes_-_Direct_vs_LOR.gif/330px-Comparison_of_Lander_Sizes_-_Direct_vs_LOR.gif 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/41/Comparison_of_Lander_Sizes_-_Direct_vs_LOR.gif/440px-Comparison_of_Lander_Sizes_-_Direct_vs_LOR.gif 2x" width="220"/>
        </a>
        <div class="thumbcaption">
         <div class="magnify">
          <a class="internal" href="/wiki/File:Comparison_of_Lander_Sizes_-_Direct_vs_LOR.gif" title="Enlarge">
          </a>
         </div>
         Comparison of lunar lander sizes, from an early Langley study
        </div>
       </div>
      </div>
      <p>
       The main advantage of LOR is the spacecraft payload saving, due to the fact that the propellant necessary to return from lunar orbit back to Earth need not be carried as dead weight down to the Moon and back into lunar orbit. This has a multiplicative effect, because each pound of "dead weight" propellant used later has to be propelled by more propellant sooner, and also because increased propellant requires increased tankage weight. The resultant weight increase would also require more thrust for lunar landing, which means larger and heavier engines.
       <sup class="reference" id="cite_ref-FOOTNOTEReeves2005_4-0">
        <a href="#cite_note-FOOTNOTEReeves2005-4">
         [4]
        </a>
       </sup>
      </p>
      <p>
       Another advantage is that the lunar lander can be designed for just that purpose, rather than requiring the main spacecraft to also be made suitable for a lunar landing. Finally, the second set of life support systems that the lunar lander requires can serve as a backup for the systems in the main spacecraft.
      </p>
      <h3>
       <span class="mw-headline" id="Risks_and_disadvantages">
        Risks and disadvantages
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Lunar_orbit_rendezvous&amp;action=edit&amp;section=3" title="Edit section: Risks and disadvantages">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       Lunar-orbit rendezvous was considered risky as of 1962, because
       <a href="/wiki/Space_rendezvous" title="Space rendezvous">
        space rendezvous
       </a>
       had not been achieved, even in Earth orbit. If the LEM could not reach the CSM, two astronauts would be stranded with no way to get back to Earth or survive
       <a href="/wiki/Atmospheric_entry" title="Atmospheric entry">
        re-entry into the atmosphere
       </a>
       . The fear proved to be unfounded, as rendezvous was successfully demonstrated in 1965 and 1966 on six
       <a href="/wiki/Project_Gemini" title="Project Gemini">
        Project Gemini
       </a>
       missions
       <sup class="reference" id="cite_ref-5">
        <a href="#cite_note-5">
         [Note 1]
        </a>
       </sup>
       with the aid of radar and on-board computers. It was also successfully done each of the eight times it was tried on Apollo missions.
       <sup class="reference" id="cite_ref-6">
        <a href="#cite_note-6">
         [Note 2]
        </a>
       </sup>
      </p>
      <h2>
       <span class="mw-headline" id="Apollo_Mission_mode_selection">
        Apollo Mission mode selection
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Lunar_orbit_rendezvous&amp;action=edit&amp;section=4" title="Edit section: Apollo Mission mode selection">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <div class="thumb tright">
       <div class="thumbinner" style="width:222px;">
        <a class="image" href="/wiki/File:Apollo_11_lunar_module.jpg">
         <img alt="" class="thumbimage" data-file-height="2306" data-file-width="2389" height="212" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/d6/Apollo_11_lunar_module.jpg/220px-Apollo_11_lunar_module.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/d6/Apollo_11_lunar_module.jpg/330px-Apollo_11_lunar_module.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/d6/Apollo_11_lunar_module.jpg/440px-Apollo_11_lunar_module.jpg 2x" width="220"/>
        </a>
        <div class="thumbcaption">
         <div class="magnify">
          <a class="internal" href="/wiki/File:Apollo_11_lunar_module.jpg" title="Enlarge">
          </a>
         </div>
         <a href="/wiki/Apollo_11" title="Apollo 11">
          Apollo 11
         </a>
         Lunar Module rendezvousing with Command Module in lunar orbit
        </div>
       </div>
      </div>
      <p>
       When the Apollo Moon landing program was started in 1961, it was assumed that the three-man
       <a href="/wiki/Apollo_Command/Service_Module" title="Apollo Command/Service Module">
        Command and Service Module combination
       </a>
       (CSM) would be used for takeoff from the lunar surface, and return to Earth. It would therefore have to be landed on the Moon by a larger rocket stage with landing gear legs, resulting in a very large spacecraft (in excess of 100,000 pounds (45,000 kg)) to be sent to the Moon.
      </p>
      <p>
       If this were done by
       <a href="/wiki/Direct_ascent" title="Direct ascent">
        direct ascent
       </a>
       (on a single
       <a href="/wiki/Launch_vehicle" title="Launch vehicle">
        launch vehicle
       </a>
       ), the rocket required would have to be extremely large, in the
       <a class="mw-redirect" href="/wiki/Nova_rocket" title="Nova rocket">
        Nova
       </a>
       class. The alternative to this would have been
       <a href="/wiki/Earth_orbit_rendezvous" title="Earth orbit rendezvous">
        Earth orbit rendezvous
       </a>
       , in which two or more rockets in the
       <a href="/wiki/Saturn_(rocket_family)" title="Saturn (rocket family)">
        Saturn class
       </a>
       would launch parts of the complete spacecraft, which would rendezvous in Earth orbit before departing for the Moon. This would possibly include a separately launched Earth departure stage, or require on-orbit refueling of the empty departure stage.
      </p>
      <p>
       <a href="/wiki/Tom_Dolan_(engineer)" title="Tom Dolan (engineer)">
        Tom Dolan
       </a>
       <sup class="reference" id="cite_ref-FOOTNOTEBrooks1979_7-0">
        <a href="#cite_note-FOOTNOTEBrooks1979-7">
         [5]
        </a>
       </sup>
       proposed the alternative of lunar orbit rendezvous, which had been studied and promoted by
       <a href="/wiki/Jim_Chamberlin" title="Jim Chamberlin">
        Jim Chamberlin
       </a>
       and
       <a href="/wiki/Owen_Maynard" title="Owen Maynard">
        Owen Maynard
       </a>
       at the
       <a href="/wiki/Space_Task_Group" title="Space Task Group">
        Space Task Group
       </a>
       in 1960 early Apollo feasibility studies.
       <sup class="reference" id="cite_ref-FOOTNOTEGainor200162-66_8-0">
        <a href="#cite_note-FOOTNOTEGainor200162-66-8">
         [6]
        </a>
       </sup>
       This mode allowed a single
       <a href="/wiki/Saturn_V" title="Saturn V">
        Saturn V
       </a>
       to launch the CSM to the Moon with a smaller
       <a href="/wiki/Apollo_Lunar_Module" title="Apollo Lunar Module">
        Lunar Excursion Module
       </a>
       (LEM).
       <sup class="reference" id="cite_ref-10">
        <a href="#cite_note-10">
         [Note 3]
        </a>
       </sup>
       When the combined spacecraft reaches
       <a href="/wiki/Lunar_orbit" title="Lunar orbit">
        lunar orbit
       </a>
       , one of the three astronauts remains with the CSM, while the other two enter the LEM, undock and descend to the surface of the Moon. They then use the ascent stage of the LEM to rejoin the CSM in lunar orbit, then discard the LEM and use the CSM for the return to Earth. This method was brought to the attention of NASA Associate Administrator
       <a href="/wiki/Robert_Seamans" title="Robert Seamans">
        Robert Seamans
       </a>
       by
       <a href="/wiki/Langley_Research_Center" title="Langley Research Center">
        Langley Research Center
       </a>
       engineer
       <a href="/wiki/John_Houbolt" title="John Houbolt">
        John C. Houbolt
       </a>
       , who led a team to develop it.
      </p>
      <p>
       Besides requiring less payload, the ability to use a lunar lander designed just for that purpose was another advantage of the LOR approach. The LEM's design gave the astronauts a clear view of their landing site through observation windows approximately 15 feet (4.6 m) above the surface, as opposed to being on their backs in a Command Module lander, at least 40 or 50 feet (12 or 15 m) above the surface, able to see it only through a television screen.
      </p>
      <p>
       Developing the LEM as a second manned vehicle provided the further advantage of redundant critical systems (electrical power, life support, and propulsion), which enabled it to be used as a "lifeboat" to keep the astronauts alive and get them home safely in the event of a critical CSM system failure. This was envisioned as a contingency, but not made a part of the LEM specifications. As it turned out, this capability proved invaluable in 1970, when just such a critical failure occurred on the
       <a href="/wiki/Apollo_13" title="Apollo 13">
        Apollo 13
       </a>
       mission when an oxygen tank failure disabled the Service Module.
      </p>
      <h3>
       <span class="mw-headline" id="Advocacy">
        Advocacy
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Lunar_orbit_rendezvous&amp;action=edit&amp;section=5" title="Edit section: Advocacy">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <div class="thumb tright">
       <div class="thumbinner" style="width:222px;">
        <a class="image" href="/wiki/File:John_C._Houbolt_-_GPN-2000-001274.jpg">
         <img alt="" class="thumbimage" data-file-height="2400" data-file-width="3000" height="176" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/4e/John_C._Houbolt_-_GPN-2000-001274.jpg/220px-John_C._Houbolt_-_GPN-2000-001274.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/4e/John_C._Houbolt_-_GPN-2000-001274.jpg/330px-John_C._Houbolt_-_GPN-2000-001274.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/4e/John_C._Houbolt_-_GPN-2000-001274.jpg/440px-John_C._Houbolt_-_GPN-2000-001274.jpg 2x" width="220"/>
        </a>
        <div class="thumbcaption">
         <div class="magnify">
          <a class="internal" href="/wiki/File:John_C._Houbolt_-_GPN-2000-001274.jpg" title="Enlarge">
          </a>
         </div>
         <a href="/wiki/John_Houbolt" title="John Houbolt">
          John Houbolt
         </a>
         explains Lunar orbit rendezvous
        </div>
       </div>
      </div>
      <p>
       Dr. John Houbolt would not let the advantages of LOR be ignored. As a member of Lunar Mission Steering Group, Houbolt had been studying various technical aspects of
       <a href="/wiki/Space_rendezvous" title="Space rendezvous">
        space rendezvous
       </a>
       since 1959 and was convinced, like several others at
       <a href="/wiki/Langley_Research_Center" title="Langley Research Center">
        Langley Research Center
       </a>
       , that LOR was not only the most feasible way to make it to the Moon before the decade was out, it was the only way. He had reported his findings to
       <a href="/wiki/NASA" title="NASA">
        NASA
       </a>
       on various occasions but felt strongly that the internal task forces (to which he made presentations) were following arbitrarily established "ground rules." According to Houbolt, these ground rules were constraining NASA's thinking about the lunar mission—and causing LOR to be ruled out before it was fairly considered.
       <sup class="reference" id="cite_ref-:0_11-0">
        <a href="#cite_note-:0-11">
         [8]
        </a>
       </sup>
      </p>
      <p>
       In November 1961, Houbolt took the bold step of skipping proper channels and writing a private letter, nine pages long, directly to
       <a class="mw-redirect" href="/wiki/Robert_C._Seamans" title="Robert C. Seamans">
        Robert C. Seamans
       </a>
       , the associate administrator. "Somewhat as a voice in the wilderness," Houbolt protested LOR's exclusion. "Do we want to go to the Moon or not?" the Langley engineer asked. "Why is Nova, with its ponderous size simply just accepted, and why is a much less grandiose scheme involving rendezvous ostracized or put on the defensive? I fully realize that contacting you in this manner is somewhat unorthodox," Houbolt admitted, "but the issues at stake are crucial enough to us all that an unusual course is warranted."
       <sup class="reference" id="cite_ref-FOOTNOTETennant2009_12-0">
        <a href="#cite_note-FOOTNOTETennant2009-12">
         [9]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-FOOTNOTEHansen1995_13-0">
        <a href="#cite_note-FOOTNOTEHansen1995-13">
         [10]
        </a>
       </sup>
      </p>
      <p>
       It took two weeks for Seamans to reply to Houbolt's extraordinary letter. The associate administrator agreed that "it would be extremely harmful to our organization and to the country if our qualified staff were unduly limited by restrictive guidelines." He assured Houbolt that NASA would in the future be paying more attention to LOR than it had up to this time.
      </p>
      <p>
       In the following months, NASA did just that, and to the surprise of many both inside and outside the agency, the dark horse candidate, LOR, quickly became the front runner. Several factors decided the issue in its favor. First, there was growing disenchantment with the idea of
       <a href="/wiki/Direct_ascent" title="Direct ascent">
        direct ascent
       </a>
       due to the time and money it was going to take to develop a 50-foot (15 m) diameter
       <a href="/wiki/Nova_(rocket)" title="Nova (rocket)">
        Nova rocket
       </a>
       , compared to the 33-foot (10 m) diameter
       <a href="/wiki/Saturn_V" title="Saturn V">
        Saturn V
       </a>
       . Second, there was increasing technical apprehension over how the relatively large spacecraft demanded even by Earth-orbit rendezvous would be able to maneuver to a soft landing on the Moon. As one NASA engineer who changed his mind explained:
      </p>
      <blockquote>
       <p>
        The business of eyeballing that thing down to the Moon really didn't have a satisfactory answer. The best thing about LOR was that it allowed us to build a separate vehicle for landing.
       </p>
      </blockquote>
      <p>
       The first major group to break camp in favor of LOR was
       <a href="/wiki/Robert_R._Gilruth" title="Robert R. Gilruth">
        Robert Gilruth
       </a>
       's Space Task Group, which was still located at Langley but was soon to move to
       <a class="mw-redirect" href="/wiki/Lyndon_B._Johnson_Space_Center" title="Lyndon B. Johnson Space Center">
        Houston
       </a>
       . The second to come over was the
       <a href="/wiki/Wernher_von_Braun" title="Wernher von Braun">
        Von Braun
       </a>
       team at the
       <a href="/wiki/Marshall_Space_Flight_Center" title="Marshall Space Flight Center">
        Marshall Space Flight Center
       </a>
       in
       <a href="/wiki/Huntsville,_Alabama" title="Huntsville, Alabama">
        Huntsville, Alabama
       </a>
       . Then these two powerful groups of converts, along with the original true believers at Langley, persuaded key officials at NASA Headquarters, notably Administrator
       <a href="/wiki/James_E._Webb" title="James E. Webb">
        James Webb
       </a>
       , who had been holding out for direct ascent, that LOR was the only way to land on the Moon by 1969. With the key players inside NASA lined up behind the concept, Webb approved LOR in July 1962.
       <sup class="reference" id="cite_ref-FOOTNOTEWitkin1962_14-0">
        <a href="#cite_note-FOOTNOTEWitkin1962-14">
         [11]
        </a>
       </sup>
       The decision was officially announced at a press conference on July 11, 1962.
       <sup class="reference" id="cite_ref-FOOTNOTENASA19621_15-0">
        <a href="#cite_note-FOOTNOTENASA19621-15">
         [12]
        </a>
       </sup>
       President Kennedy's science adviser,
       <a href="/wiki/Jerome_Wiesner" title="Jerome Wiesner">
        Jerome Wiesner
       </a>
       , remained firmly opposed to LOR.
       <sup class="reference" id="cite_ref-FOOTNOTENelson2009209—210_16-0">
        <a href="#cite_note-FOOTNOTENelson2009209—210-16">
         [13]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-:0_11-1">
        <a href="#cite_note-:0-11">
         [8]
        </a>
       </sup>
      </p>
      <h2>
       <span class="mw-headline" id="Other_plans_using_LOR">
        Other plans using LOR
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Lunar_orbit_rendezvous&amp;action=edit&amp;section=6" title="Edit section: Other plans using LOR">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <ul>
       <li>
        The proposed Soviet Lunar Landing plan, using the
        <a href="/wiki/N1_(rocket)" title="N1 (rocket)">
         N1 rocket
        </a>
        ,
        <a href="/wiki/LK_(spacecraft)" title="LK (spacecraft)">
         LK Lander
        </a>
        and
        <a href="/wiki/Soyuz_7K-LOK" title="Soyuz 7K-LOK">
         Soyuz 7K-LOK
        </a>
        would have used a similar LOR mission profile.
       </li>
       <li>
        The cancelled
        <a href="/wiki/Constellation_program" title="Constellation program">
         Constellation program
        </a>
        Lunar Landing would have used a combination of
        <a class="mw-redirect" href="/wiki/Earth_Orbit_Rendezvous" title="Earth Orbit Rendezvous">
         Earth Orbit Rendezvous
        </a>
        and Lunar Rendezvous.
       </li>
       <li>
        A robotic lunar orbit rendezvous is planned for use by the Chinese
        <a href="/wiki/Chang%27e_5" title="Chang'e 5">
         Chang'e 5
        </a>
        lunar sample return mission scheduled in 2017.
        <sup class="reference" id="cite_ref-FOOTNOTELaxman2012_17-0">
         <a href="#cite_note-FOOTNOTELaxman2012-17">
          [14]
         </a>
        </sup>
       </li>
      </ul>
