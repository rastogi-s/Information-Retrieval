      <p>
       <b>
        KStars
       </b>
       is a
       <a href="/wiki/Free_software" title="Free software">
        freely licensed
       </a>
       <a href="/wiki/Planetarium" title="Planetarium">
        planetarium
       </a>
       program using the
       <a class="mw-redirect" href="/wiki/KDE_Platform" title="KDE Platform">
        KDE Platform
       </a>
       . It can be used on most
       <a href="/wiki/Unix-like" title="Unix-like">
        Unix-like
       </a>
       computer
       <a href="/wiki/Operating_system" title="Operating system">
        operating systems
       </a>
       , as well as on the
       <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">
        Microsoft Windows
       </a>
       platform using 'KDE for Windows'.
       <sup class="reference" id="cite_ref-2">
        <a href="#cite_note-2">
         [2]
        </a>
       </sup>
       It provides an accurate graphical representation of the night sky, from any location on Earth, at any date and time. The display includes up to 100 million
       <a href="/wiki/Star" title="Star">
        stars
       </a>
       (with additional addons), 13,000
       <a class="mw-redirect" href="/wiki/Deep_sky_object" title="Deep sky object">
        deep sky objects
       </a>
       ,
       <a href="/wiki/Constellation" title="Constellation">
        constellations
       </a>
       from different cultures, all 8 planets, the
       <a href="/wiki/Sun" title="Sun">
        Sun
       </a>
       and
       <a href="/wiki/Moon" title="Moon">
        Moon
       </a>
       , and thousands of
       <a href="/wiki/Comet" title="Comet">
        comets
       </a>
       and
       <a href="/wiki/Asteroid" title="Asteroid">
        asteroids
       </a>
       . It has features to appeal to users of all levels, from informative hypertext articles about astronomy, to robust control of
       <a href="/wiki/Telescope" title="Telescope">
        telescopes
       </a>
       and
       <a href="/wiki/Charge-coupled_device" title="Charge-coupled device">
        CCD
       </a>
       cameras, and logging of observations of specific objects.
      </p>
      <p>
       KStars supports adjustable simulation speeds in order to view phenomena that happen over long timescales. For astronomical calculations, Astrocalculator can be used to predict conjunctions, and perform many common astronomical calculations. The following tools are included:
      </p>
      <ul>
       <li>
        Observation planner
       </li>
       <li>
        Sky calendar tool
       </li>
       <li>
        Script Builder
       </li>
       <li>
        Solar System
       </li>
       <li>
        Jupiter Moons
       </li>
       <li>
        Flags: Custom flags superimposed on the sky map.
       </li>
       <li>
        FOV editor to calculate field of view of equipment and display them.
       </li>
       <li>
        Altitude vs. Time tool to plot altitude vs. time graphs for any object.
       </li>
       <li>
        High quality print outs for sky charts.
       </li>
       <li>
        Ekos is an astrophotography suite, a complete astrophotography solution that can control all
        <a href="/wiki/Instrument_Neutral_Distributed_Interface" title="Instrument Neutral Distributed Interface">
         INDI
        </a>
        devices including numerous telescopes, CCDs,
        <a class="mw-redirect" href="/wiki/DSLR" title="DSLR">
         DSLRs
        </a>
        , focusers, filters, and a lot more. Ekos supports highly accurate tracking using online and offline
        <a href="/wiki/Astrometry" title="Astrometry">
         astrometry
        </a>
        solver, auto-focus and auto-guiding capabilities, and capture of single or multiple images using the powerful built in sequence manager.
       </li>
      </ul>
      <p>
       KStars has been packaged by many Linux/BSD distributions, including
       <a href="/wiki/Red_Hat_Linux" title="Red Hat Linux">
        Red Hat Linux
       </a>
       ,
       <a href="/wiki/OpenSUSE" title="OpenSUSE">
        OpenSUSE
       </a>
       ,
       <a href="/wiki/Mandriva_Linux" title="Mandriva Linux">
        Mandriva Linux
       </a>
       , and
       <a href="/wiki/Debian" title="Debian">
        Debian GNU/Linux
       </a>
       . Some distributions package KStars as a separate application, some just provide a
       <a class="mw-redirect" href="/wiki/Kdeedu" title="Kdeedu">
        kdeedu
       </a>
       package, which includes KStars. KStars is distributed with the
       <a href="/wiki/KDE_Software_Compilation" title="KDE Software Compilation">
        KDE Software Compilation
       </a>
       as part of the kdeedu "Edutainment" module.
      </p>
      <p>
       KStars participated in
       <a href="/wiki/Google_Summer_of_Code" title="Google Summer of Code">
        Google Summer of Code
       </a>
       in 2008,
       <sup class="reference" id="cite_ref-3">
        <a href="#cite_note-3">
         [3]
        </a>
       </sup>
       2009,
       <sup class="reference" id="cite_ref-4">
        <a href="#cite_note-4">
         [4]
        </a>
       </sup>
       2010,
       <sup class="reference" id="cite_ref-5">
        <a href="#cite_note-5">
         [5]
        </a>
       </sup>
       2011
       <sup class="reference" id="cite_ref-6">
        <a href="#cite_note-6">
         [6]
        </a>
       </sup>
       2012,
       <sup class="reference" id="cite_ref-7">
        <a href="#cite_note-7">
         [7]
        </a>
       </sup>
       2015
       <sup class="reference" id="cite_ref-8">
        <a href="#cite_note-8">
         [8]
        </a>
       </sup>
       and 2016.
       <sup class="reference" id="cite_ref-9">
        <a href="#cite_note-9">
         [9]
        </a>
       </sup>
       It has also participated in the first run of
       <a class="mw-redirect" href="/wiki/ESA" title="ESA">
        ESA
       </a>
       's Summer of Code in Space in 2011.
       <sup class="reference" id="cite_ref-10">
        <a href="#cite_note-10">
         [10]
        </a>
       </sup>
      </p>
      <p>
       It has been identified as one of the three best "Linux stargazing apps" in a Linux.com review.
       <sup class="reference" id="cite_ref-11">
        <a href="#cite_note-11">
         [11]
        </a>
       </sup>
      </p>
      <p>
       The latest version of KStars is 2.8.9 which included additional support for guide star images, drift errors, and RMS values.
       <sup class="reference" id="cite_ref-12">
        <a href="#cite_note-12">
         [12]
        </a>
       </sup>
      </p>
