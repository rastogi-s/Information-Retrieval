   <h1 class="firstHeading" id="firstHeading" lang="en">
    Easting and northing
   </h1>

    <div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en">
     <div class="mw-parser-output">
      <p>
       The terms
       <b>
        easting
       </b>
       and
       <b>
        northing
       </b>
       are
       <a href="/wiki/Geography" title="Geography">
        geographic
       </a>
       <a href="/wiki/Cartesian_coordinate_system" title="Cartesian coordinate system">
        Cartesian coordinates
       </a>
       for a point. Easting refers to the
       <a href="/wiki/East" title="East">
        eastward
       </a>
       -measured distance (or the
       <i>
        x
       </i>
       -
       <a class="mw-redirect" href="/wiki/Coordinate" title="Coordinate">
        coordinate
       </a>
       ), while northing refers to the
       <a href="/wiki/North" title="North">
        northward
       </a>
       -measured distance (or the
       <i>
        y
       </i>
       -coordinate). When using common projections such as the
       <a href="/wiki/Transverse_Mercator_projection" title="Transverse Mercator projection">
        transverse Mercator projection
       </a>
       , these are distances
       <a href="/wiki/Map_projection" title="Map projection">
        projected
       </a>
       on an imaginary surface similar to a bent sheet of paper, and are not the same as distances measured on the
       <a href="/wiki/Figure_of_the_Earth" title="Figure of the Earth">
        curved surface of the Earth
       </a>
       .
      </p>
      <div class="thumb tright">
       <div class="thumbinner" style="width:222px;">
        <a class="image" href="/wiki/File:Utm-zones.jpg">
         <img alt="" class="thumbimage" data-file-height="1024" data-file-width="2048" height="110" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ed/Utm-zones.jpg/220px-Utm-zones.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ed/Utm-zones.jpg/330px-Utm-zones.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ed/Utm-zones.jpg/440px-Utm-zones.jpg 2x" width="220"/>
        </a>
        <div class="thumbcaption">
         <div class="magnify">
          <a class="internal" href="/wiki/File:Utm-zones.jpg" title="Enlarge">
          </a>
         </div>
         UTM-zones
        </div>
       </div>
      </div>
      <p>
       Easting and northing coordinates are commonly measured in metres from the axes of some
       <a class="mw-redirect" href="/wiki/Datum_(geodesy)#Horizontal_datum" title="Datum (geodesy)">
        horizontal datum
       </a>
       . However, other units (e.g.,
       <a class="mw-redirect" href="/wiki/Survey_feet" title="Survey feet">
        survey feet
       </a>
       ) are also used. The coordinates are most commonly associated with the
       <a href="/wiki/Universal_Transverse_Mercator_coordinate_system#Locating_a_position_using_UTM_coordinates" title="Universal Transverse Mercator coordinate system">
        Universal Transverse Mercator coordinate system
       </a>
       (UTM), which has unique zones that cover the Earth to provide detailed referencing.
      </p>
      <p>
      </p>

      <h2>
       <span class="mw-headline" id="Notation_and_conventions">
        Notation and conventions
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Easting_and_northing&amp;action=edit&amp;section=1" title="Edit section: Notation and conventions">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       Locations can be found using easting/northing (or
       <i>
        x
       </i>
       ,
       <i>
        y
       </i>
       ) pairs. The pair is usually represented conventionally with easting first, northing second.
      </p>
      <p>
       For example, the peak of
       <a href="/wiki/Mount_Assiniboine" title="Mount Assiniboine">
        Mount Assiniboine
       </a>
       (at
       <span class="plainlinks nourlexpansion">
        <a class="external text" href="//tools.wmflabs.org/geohack/geohack.php?pagename=Easting_and_northing&amp;params=50_52_10_N_115_39_03_W_">
         <span class="geo-default">
          <span class="geo-dms" title="Maps, aerial photos, and other data for this location">
           <span class="latitude">
            50°52′10″N
           </span>
           <span class="longitude">
            115°39′03″W
           </span>
          </span>
         </span>
         <span class="geo-multi-punct">
          ﻿ / ﻿
         </span>
         <span class="geo-nondefault">
          <span class="geo-dec" title="Maps, aerial photos, and other data for this location">
           50.86944°N 115.65083°W
          </span>
          <span style="display:none">
           ﻿ /
           <span class="geo">
            50.86944; -115.65083
           </span>
          </span>
         </span>
        </a>
       </span>
       ) in UTM Zone 11 is represented by
       <code>
        11U 594934 5636174
       </code>
       . Other conventions can also be used, such as a truncated grid reference,
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
       which would shorten the example coordinates to
       <code>
        949-361
       </code>
       .
      </p>
      <p>
       Negative northing and easting values indicate a position due south and west of the origin, respectively.
      </p>
      <p>
       <span id="False_origin">
       </span>
       <span id="Natural_origin">
       </span>
       Usually associated with a map projection is a
       <i>
        natural origin
       </i>
       , e.g., at which the ellipsoid and flat map surfaces coincide.
       <sup class="reference" id="cite_ref-OGP7_2_2-0">
        <a href="#cite_note-OGP7_2-2">
         [2]
        </a>
       </sup>
       To ensure that the northing and easting coordinates on a map are not negative, map projections may set up a
       <i>
        false origin
       </i>
       , specified in terms of
       <i>
        false northing
       </i>
       and
       <i>
        false easting
       </i>
       values, that offset the true origin.
      </p>
      <h2>
       <span class="mw-headline" id="Other_usage">
        Other usage
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Easting_and_northing&amp;action=edit&amp;section=2" title="Edit section: Other usage">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       The term
       <b>
        northing
       </b>
       has also been used by explorers to describe a general progress toward the
       <a href="/wiki/North_Pole" title="North Pole">
        North Pole
       </a>
       .
       <a href="/wiki/Isaac_Israel_Hayes" title="Isaac Israel Hayes">
        Isaac Israel Hayes
       </a>
       used this term in an 1861 address to the New-York Geographical and Statistical Society saying, "The want of steam power curtailed my northing."
       <sup class="reference" id="cite_ref-3">
        <a href="#cite_note-3">
         [3]
        </a>
       </sup>
      </p>
