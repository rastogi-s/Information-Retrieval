      <p>
       The
       <b>
        Paleohispanic languages
       </b>
       were the languages of the
       <a class="mw-redirect" href="/wiki/Pre-Roman_peoples_of_the_Iberian_Peninsula" title="Pre-Roman peoples of the Iberian Peninsula">
        Pre-Roman peoples of the Iberian Peninsula
       </a>
       , excluding languages of foreign colonies, such as
       <a href="/wiki/Ancient_Greek" title="Ancient Greek">
        Greek
       </a>
       in
       <a href="/wiki/Emp%C3%BAries" title="Empúries">
        Emporion
       </a>
       and
       <a href="/wiki/Phoenician_language" title="Phoenician language">
        Phoenician
       </a>
       in
       <a href="/wiki/Cartagena,_Spain" title="Cartagena, Spain">
        Qart Hadast
       </a>
       . After the
       <a href="/wiki/Roman_conquest_of_Hispania" title="Roman conquest of Hispania">
        Roman conquest of Hispania
       </a>
       the Paleohispanic languages, with the exception of
       <a href="/wiki/Proto-Basque_language" title="Proto-Basque language">
        Proto-Basque
       </a>
       , were replaced by
       <a href="/wiki/Latin" title="Latin">
        Latin
       </a>
       , the ancestor of the modern
       <a href="/wiki/Iberian_Romance_languages" title="Iberian Romance languages">
        Iberian Romance languages
       </a>
       .
      </p>
      <p>
      </p>

      <h2>
       <span class="mw-headline" id="Languages">
        Languages
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Paleohispanic_languages&amp;action=edit&amp;section=1" title="Edit section: Languages">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       Some of these languages were documented directly through inscriptions, mainly in
       <a href="/wiki/Paleohispanic_scripts" title="Paleohispanic scripts">
        Paleohispanic scripts
       </a>
       , that date for sure between the 5th century BC, maybe from the 7th century in the opinion of some researchers, until the end of the 1st century BC or the beginning of the 1st century AD.
      </p>
      <ul>
       <li>
        <a class="mw-redirect" href="/wiki/Proto-Basque" title="Proto-Basque">
         Proto-Basque
        </a>
       </li>
       <li>
        <a href="/wiki/Aquitanian_language" title="Aquitanian language">
         Aquitanian language
        </a>
       </li>
       <li>
        <a href="/wiki/Iberian_language" title="Iberian language">
         Iberian language
        </a>
       </li>
       <li>
        <a href="/wiki/Celtiberian_language" title="Celtiberian language">
         Celtiberian
        </a>
       </li>
       <li>
        <a href="/wiki/Gallaecian_language" title="Gallaecian language">
         Gallaecian
        </a>
       </li>
       <li>
        <a href="/wiki/Tartessian_language" title="Tartessian language">
         Tartessian
        </a>
       </li>
       <li>
        <a href="/wiki/Lusitanian_language" title="Lusitanian language">
         Lusitanian
        </a>
       </li>
       <li>
        <a href="/wiki/Aquitanian_language" title="Aquitanian language">
         Aquitanian
        </a>
       </li>
       <li>
        <a href="/wiki/Sorothaptic_language" title="Sorothaptic language">
         Sorothaptic
        </a>
       </li>
      </ul>
      <p>
       Other Paleohispanic languages can only be identified indirectly through
       <a class="mw-redirect" href="/wiki/Toponym" title="Toponym">
        toponyms
       </a>
       ,
       <a class="mw-redirect" href="/wiki/Anthroponym" title="Anthroponym">
        anthroponyms
       </a>
       or
       <a class="mw-redirect" href="/wiki/Theonym" title="Theonym">
        theonyms
       </a>
       cited by
       <a href="/wiki/Ancient_Rome" title="Ancient Rome">
        Roman
       </a>
       and
       <a href="/wiki/Ancient_Greece" title="Ancient Greece">
        Greek
       </a>
       sources.
      </p>
      <h2>
       <span class="mw-headline" id="Classification">
        Classification
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Paleohispanic_languages&amp;action=edit&amp;section=2" title="Edit section: Classification">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       Of these languages, Celtiberian, Gallaecian, Lusitanian, and presumably Sorothaptic were
       <a href="/wiki/Indo-European_languages" title="Indo-European languages">
        Indo-European languages
       </a>
       ; Celtiberian and Gallaecian were
       <a href="/wiki/Celtic_languages" title="Celtic languages">
        Celtic languages
       </a>
       , and Lusitanian might have been, but the hypothetical Sorothaptic was not. Aquitanian was a precursor of
       <a href="/wiki/Basque_language" title="Basque language">
        Basque
       </a>
       , while Tartessian and Iberian remain
       <a href="/wiki/Unclassified_language" title="Unclassified language">
        unclassified
       </a>
       .
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
      </p>
