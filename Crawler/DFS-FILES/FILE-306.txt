      <p>
       <b>
        Spanish Braille
       </b>
       is the
       <a href="/wiki/Braille" title="Braille">
        braille
       </a>
       alphabet of
       <a href="/wiki/Spanish_language" title="Spanish language">
        Spanish
       </a>
       and
       <a href="/wiki/Galician_language" title="Galician language">
        Galician
       </a>
       . It is very close to
       <a href="/wiki/French_Braille" title="French Braille">
        French Braille
       </a>
       , with the addition of a letter for
       <i>
        ñ
       </i>
       , slight modification of the accented letters, and some differences in punctuation. Further conventions have been unified by the Latin American Blind Union, but differences with Spain remain.
      </p>

      <h2>
       <span class="mw-headline" id="Alphabet">
        Alphabet
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Spanish_Braille&amp;action=edit&amp;section=1" title="Edit section: Alphabet">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       The French Braille letters for vowels with a grave accent,
       <i>
        à è ì ò ù,
       </i>
       are used in Spanish Braille for vowels with an acute accent,
       <i>
        á é í ó ú
       </i>
       . In addition, French
       <span style="background: #f4f4f4; font-size:100%; margin:1px 2px 0 0;" title=" ⠻ (braille pattern dots-12456) U+283B">
        ⠻
       </span>
       <i>
        ï
       </i>
       is reassigned to Spanish
       <i>
        ñ
       </i>
       . Thus, in numerical order, the letters are:
      </p>
      <dl>
       <dd>
        <table class="wikitable" style="line-height: 1.2">
         <tr align="center">

      <h2>
       <span class="mw-headline" id="Other_languages">
        Other languages
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Spanish_Braille&amp;action=edit&amp;section=5" title="Edit section: Other languages">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       <span id="Galician">
       </span>
       <span id="Basque">
       </span>
       The full Spanish Braille alphabet is used for
       <a href="/wiki/Galician_language" title="Galician language">
        Galician
       </a>
       as well. The letter
       <span style="background: #f4f4f4; font-size:100%; margin:1px 2px 0 0;" title=" ⠻ (braille pattern dots-12456) U+283B">
        ⠻
       </span>
       for
       <i>
        ñ
       </i>
       is shared with
       <a class="mw-redirect" href="/wiki/Basque_Braille" title="Basque Braille">
        Basque Braille
       </a>
       (which has no additional letters) and with
       <a href="/wiki/Guarani_Braille" title="Guarani Braille">
        Guarani Braille
       </a>
       (which does). It is not, however, used for the languages of the Philippines, which instead use an accent dot of
       <a href="/wiki/English_Braille" title="English Braille">
        English Braille
       </a>
       with
       <i>
        n
       </i>
       ,
       <span style="background: #f4f4f4; font-size:100%; margin:1px 2px 0 0;" title=" ⠈ (braille pattern dots-4) U+2808">
        ⠈
       </span>
       <span style="background: #f4f4f4; font-size:100%; margin:1px 2px 0 0;" title=" ⠝ (braille pattern dots-1345) U+281D">
        ⠝
       </span>
       , for
       <i>
        ñ
       </i>
       . (See
       <a class="mw-redirect" href="/wiki/Basic_braille" title="Basic braille">
        basic braille
       </a>
       .)
      </p>
      <p>
       Punctuation for Galician and Basque Braille is the same as that of Spanish Braille.
      </p>
