      <p>
       <b>
        Ñ
       </b>
       (lower case
       <b>
        ñ
       </b>
       ,
       <a href="/wiki/Spanish_language" title="Spanish language">
        Spanish
       </a>
       :
       <i>
        <span lang="es" xml:lang="es">
         eñe
        </span>
       </i>
       ,
       <a href="/wiki/International_Phonetic_Alphabet" title="International Phonetic Alphabet">
        Phonetic Alphabet
       </a>
       :
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        /ˈeɲe/
       </span>
       "
       <i>
        énye
       </i>
       ",
       <span class="unicode haudio">
        <span class="fn">
         <span style="white-space:nowrap">
          <a href="/wiki/File:Letter_%C3%B1_es_es.flac" title="About this sound">
           <img alt="About this sound" data-file-height="20" data-file-width="20" height="11" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/8a/Loudspeaker.svg/11px-Loudspeaker.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/8a/Loudspeaker.svg/17px-Loudspeaker.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/8a/Loudspeaker.svg/22px-Loudspeaker.svg.png 2x" width="11"/>
          </a>
         </span>
         <a class="internal" href="//upload.wikimedia.org/wikipedia/commons/e/e0/Letter_%C3%B1_es_es.flac" title="Letter ñ es es.flac">
          pronunciation
         </a>
        </span>
        <small class="metadata audiolinkinfo" style="cursor:help;">
         (
         <a class="mw-redirect" href="/wiki/Wikipedia:Media_help" title="Wikipedia:Media help">
          <span style="cursor:help;">
           help
          </span>
         </a>
         ·
         <a href="/wiki/File:Letter_%C3%B1_es_es.flac" title="File:Letter ñ es es.flac">
          <span style="cursor:help;">
           info
          </span>
         </a>
         )
        </small>
       </span>
       ) is a letter of the modern
       <a href="/wiki/Latin_alphabet" title="Latin alphabet">
        Latin alphabet
       </a>
       , formed by placing a
       <a href="/wiki/Tilde" title="Tilde">
        tilde
       </a>
       (called a
       <i>
        virgulilla
       </i>
       in Spanish) on top of an upper- or lowercase
       <i>
        N
       </i>
       .
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
       It became part of the
       <a class="mw-redirect" href="/wiki/Spanish_alphabet" title="Spanish alphabet">
        Spanish alphabet
       </a>
       in the eighteenth century when it was first formally defined, but it is also used in other languages such as
       <a class="mw-redirect" href="/wiki/Galician_alphabet" title="Galician alphabet">
        Galician
       </a>
       ,
       <a class="mw-redirect" href="/wiki/Asturian_alphabet" title="Asturian alphabet">
        Asturian
       </a>
       , the
       <a href="/wiki/Aragonese_language" title="Aragonese language">
        Aragonese
       </a>
       <i>
        Grafía de Uesca
       </i>
       ,
       <a class="mw-redirect" href="/wiki/Chavacano_language" title="Chavacano language">
        Chavacano
       </a>
       ,
       <a href="/wiki/Filipino_orthography" title="Filipino orthography">
        Filipino
       </a>
       ,
       <a href="/wiki/Chamorro_language" title="Chamorro language">
        Chamorro
       </a>
       ,
       <a href="/wiki/Guarani_language" title="Guarani language">
        Guarani
       </a>
       ,
       <a href="/wiki/Quechua_alphabet" title="Quechua alphabet">
        Quechua
       </a>
       ,
       <a href="/wiki/Mapudungun_alphabet" title="Mapudungun alphabet">
        Mapudungun
       </a>
       ,
       <a href="/wiki/Mandinka_language" title="Mandinka language">
        Mandinka
       </a>
       , and
       <a href="/wiki/Tetum_alphabet" title="Tetum alphabet">
        Tetum
       </a>
       alphabets, as well as in Latin transliteration of
       <a href="/wiki/Tocharian_languages" title="Tocharian languages">
        Tocharian
       </a>
       and
       <a href="/wiki/Sanskrit" title="Sanskrit">
        Sanskrit
       </a>
       , where it represents
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [
        <a href="/wiki/Palatal_nasal#Palatal_or_alveolo-palatal" title="Palatal nasal">
         ɲ
        </a>
        ]
       </span>
       . It represents
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [
        <a href="/wiki/Velar_nasal" title="Velar nasal">
         ŋ
        </a>
        ]
       </span>
       in
       <a href="/wiki/Crimean_Tatar_language" title="Crimean Tatar language">
        Crimean Tatar
       </a>
       . In
       <a href="/wiki/Breton_language" title="Breton language">
        Breton
       </a>
       and in
       <a href="/wiki/Rohingya_language" title="Rohingya language">
        Rohingya
       </a>
       , it denotes
       <a href="/wiki/Nasalization" title="Nasalization">
        nasalization
       </a>
       of the preceding vowel.
      </p>
      <p>
       Unlike many other letters that use diacritic marks (such as
       <i>
        Ü
       </i>
       in
       <a href="/wiki/Spanish_language" title="Spanish language">
        Spanish
       </a>
       and
       <a href="/wiki/German_language" title="German language">
        German
       </a>
       and
       <i>
        Ç
       </i>
       in
       <a href="/wiki/French_language" title="French language">
        French
       </a>
       and
       <a href="/wiki/Portuguese_language" title="Portuguese language">
        Portuguese
       </a>
       ),
       <i>
        Ñ
       </i>
       in
       <a href="/wiki/Spanish_language" title="Spanish language">
        Spanish
       </a>
       ,
       <a href="/wiki/Galician_language" title="Galician language">
        Galician
       </a>
       ,
       <a href="/wiki/Asturian_language" title="Asturian language">
        Asturian
       </a>
       ,
       <a class="mw-redirect" href="/wiki/Leonese_language" title="Leonese language">
        Leonese
       </a>
       , and
       <a href="/wiki/Filipino_language" title="Filipino language">
        Filipino
       </a>
       is considered a letter in its own right, has its own name (in Spanish:
       <i>
        <span lang="es" xml:lang="es">
         <a class="extiw" href="https://en.wiktionary.org/wiki/e%C3%B1e#Spanish" title="wikt:eñe">
          eñe
         </a>
        </span>
       </i>
       ), and its own
       <a href="/wiki/Alphabetical_order" title="Alphabetical order">
        place in the alphabet
       </a>
       (after
       <i>
        N
       </i>
       ). Its alphabetical independence is similar to the English
       <i>
        <a href="/wiki/W" title="W">
         W
        </a>
       </i>
       , which historically came from a doubled
       <i>
        V
       </i>
       .
      </p>

      <h2>
       <span class="mw-headline" id="History">
        History
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=%C3%91&amp;action=edit&amp;section=1" title="Edit section: History">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       Historically,
       <i>
        ñ
       </i>
       arose as a
       <a href="/wiki/Typographic_ligature" title="Typographic ligature">
        ligature
       </a>
       of
       <i>
        nn
       </i>
       ; the tilde was
       <a href="/wiki/Shorthand" title="Shorthand">
        shorthand
       </a>
       for the second
       <i>
        n
       </i>
       , written over the first.
       <sup class="reference" id="cite_ref-2">
        <a href="#cite_note-2">
         [2]
        </a>
       </sup>
       This is a letter in the Spanish alphabet that is used for many words, for example, the Spanish word
       <i>
        año
       </i>
       (
       <i>
        anno
       </i>
       in Old Spanish) meaning "year" and derived from
       <a href="/wiki/Latin" title="Latin">
        Latin
       </a>
       <small>
        ANNVS
       </small>
       . Other languages used the
       <a href="/wiki/Macron_(diacritic)" title="Macron (diacritic)">
        macron
       </a>
       over an
       <i>
        n
       </i>
       or
       <i>
        m
       </i>
       to indicate simple doubling.
      </p>
      <p>
       Already in
       <a href="/wiki/Medieval_Latin" title="Medieval Latin">
        medieval Latin
       </a>
       palaeography, the sign that in Spanish came to be called
       <i>
        virgulilla
       </i>
       (meaning "little comma") was used over a vowel to indicate a following nasal consonant (
       <i>
        n
       </i>
       or
       <i>
        m
       </i>
       ) that had been omitted, as in
       <i>
        tãtus
       </i>
       for
       <i>
        tantus
       </i>
       or
       <i>
        quã
       </i>
       for
       <i>
        quam
       </i>
       . This usage was passed on to other languages using the Latin alphabet although it was subsequently dropped by most. Spanish retained it, however, in some specific cases, particularly to indicate the
       <a href="/wiki/Palatal_nasal" title="Palatal nasal">
        palatal nasal
       </a>
       , the sound that is now spelt as
       <i>
        ñ
       </i>
       . The word
       <i>
        tilde
       </i>
       comes from Spanish, derived by metathesis of the word
       <i>
        título
       </i>
       as
       <i>
        tidlo
       </i>
       , this originally from Latin
       <small>
        TITULUS
       </small>
       "title" or "heading"; compare
       <i>
        cabildo
       </i>
       with Latin
       <small>
        CAPITULUM
       </small>
       .
       <sup class="reference" id="cite_ref-3">
        <a href="#cite_note-3">
         [3]
        </a>
       </sup>
      </p>
      <p>
       From spellings of
       <i>
        anno
       </i>
       abbreviated as
       <i>
        ãno
       </i>
       , as explained above, the tilde was thenceforth transferred to the
       <i>
        n
       </i>
       and kept as a useful expedient to indicate the new palatal nasal sound that Spanish had developed in that position:
       <i>
        año
       </i>
       . The sign was also adopted for the same palatal nasal in all other cases, even when it did not derive from an original
       <i>
        nn
       </i>
       , as in
       <i>
        leña
       </i>
       (from Latin
       <i>
        ligna
       </i>
       ) or
       <i>
        señor
       </i>
       (from Latin
       <i>
        senior
       </i>
       ).
      </p>
      <p>
       The palatal nasal sound is roughly reminiscent of the English
       <a href="/wiki/Consonant_cluster" title="Consonant cluster">
        consonant cluster
       </a>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        /nj/
       </span>
       <i>
        ca
        <b>
         ny
        </b>
        on
       </i>
       <span class="nowrap">
        <span class="IPA nopopups noexcerpt">
         <a href="/wiki/Help:IPA/English" title="Help:IPA/English">
          /
          <span style="border-bottom:1px dotted">
           <span title="/ˈ/: primary stress follows">
            ˈ
           </span>
           <span title="'k' in 'kind'">
            k
           </span>
           <span title="/æ/: 'a' in 'bad'">
            æ
           </span>
           <span title="'n' in 'nigh'">
            n
           </span>
           <span title="/j/: 'y' in 'yes'">
            j
           </span>
           <span title="/ə/: 'a' in 'about'">
            ə
           </span>
           <span title="'n' in 'nigh'">
            n
           </span>
          </span>
          /
         </a>
        </span>
       </span>
       . While this common description is enough to give a rough idea of the sound, it is not precise. It is analogous to giving the pronunciation of the English word
       <i>
        shot
       </i>
       as "syot".
      </p>
      <p>
       Other
       <a href="/wiki/Romance_languages" title="Romance languages">
        Romance languages
       </a>
       have different spellings for this sound:
       <a href="/wiki/Italian_language" title="Italian language">
        Italian
       </a>
       and
       <a href="/wiki/French_language" title="French language">
        French
       </a>
       use
       <i>
        gn
       </i>
       , a consonant cluster that had evolved from Latin, whereas
       <a href="/wiki/Occitan_language" title="Occitan language">
        Occitan
       </a>
       and
       <a href="/wiki/Portuguese_language" title="Portuguese language">
        Portuguese
       </a>
       chose
       <i>
        nh
       </i>
       and
       <a href="/wiki/Catalan_language" title="Catalan language">
        Catalan
       </a>
       (
       <i>
        ny
       </i>
       ) even though these digraphs had no etymological precedent.
      </p>
      <p>
       When
       <a href="/wiki/Morse_code" title="Morse code">
        Morse code
       </a>
       was extended to cover languages other than English, the sequence ( — — · — — ) was allotted for this character.
      </p>
      <p>
       Although
       <i>
        ñ
       </i>
       is used by other languages whose spellings were influenced by Spanish, it has recently been chosen to represent the identity of the Spanish language, especially as a result of the battle against its obliteration from computer keyboards by an English-led industry.
       <sup class="reference" id="cite_ref-GGM_4-0">
        <a href="#cite_note-GGM-4">
         [4]
        </a>
       </sup>
      </p>
      <h2>
       <span class="mw-headline" id="Cross-linguistic_usage">
        Cross-linguistic usage
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=%C3%91&amp;action=edit&amp;section=2" title="Edit section: Cross-linguistic usage">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <div class="thumb tleft">
       <div class="thumbinner" style="width:222px;">
        <a class="image" href="/wiki/File:Pi%C3%B1ata.svg">
         <img alt="" class="thumbimage" data-file-height="587" data-file-width="496" height="260" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/fa/Pi%C3%B1ata.svg/220px-Pi%C3%B1ata.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/fa/Pi%C3%B1ata.svg/330px-Pi%C3%B1ata.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/f/fa/Pi%C3%B1ata.svg/440px-Pi%C3%B1ata.svg.png 2x" width="220"/>
        </a>
        <div class="thumbcaption">
         <div class="magnify">
          <a class="internal" href="/wiki/File:Pi%C3%B1ata.svg" title="Enlarge">
          </a>
         </div>
         The letter
         <i>
          ñ
         </i>
         used in the word
         <i>
          <a href="/wiki/Pi%C3%B1ata" title="Piñata">
           piñata
          </a>
         </i>
        </div>
       </div>
      </div>
      <p>
       In Spanish and some other languages (such as
       <a href="/wiki/Philippine_languages" title="Philippine languages">
        Filipino languages
       </a>
       ,
       <a href="/wiki/Aymara_language" title="Aymara language">
        Aymara
       </a>
       ,
       <a class="mw-redirect" href="/wiki/Quechua_language" title="Quechua language">
        Quechua
       </a>
       ,
       <a class="mw-redirect" href="/wiki/Mapudung%C3%BAn" title="Mapudungún">
        Mapudungún
       </a>
       ,
       <a href="/wiki/Guarani_language" title="Guarani language">
        Guarani
       </a>
       ,
       <a href="/wiki/Basque_language" title="Basque language">
        Basque
       </a>
       ,
       <a href="/wiki/Chamorro_language" title="Chamorro language">
        Chamorro
       </a>
       ,
       <a class="new" href="/w/index.php?title=Leonesfe_language&amp;action=edit&amp;redlink=1" title="Leonesfe language (page does not exist)">
        Leonese
       </a>
       ,
       <a href="/wiki/Yavapai_language" title="Yavapai language">
        Yavapai
       </a>
       and
       <a class="mw-redirect" href="/wiki/Tetum" title="Tetum">
        Tetum
       </a>
       ), whose orthographies have some basis in that of Spanish, it represents a
       <a href="/wiki/Palatal_nasal" title="Palatal nasal">
        palatal nasal
       </a>
       .
      </p>
      <p>
       In
       <a href="/wiki/Galician_language" title="Galician language">
        Galician
       </a>
       , it represents such a sound, but it was probably not adopted from Spanish, as evidenced by its presence in the first
       <a href="/wiki/Galician-Portuguese" title="Galician-Portuguese">
        Galician-Portuguese
       </a>
       written in
       <a class="mw-redirect" href="/wiki/Galicia,_Spain" title="Galicia, Spain">
        Galicia
       </a>
       conserved text (
       <i>
        <a class="extiw" href="https://gl.wikipedia.org/wiki/Foro_do_bo_burgo_de_Castro_Caldelas" title="gl:Foro do bo burgo de Castro Caldelas">
         Foro do bo burgo de Castro Caldelas
        </a>
       </i>
       , written in 1228
       <sup class="reference" id="cite_ref-5">
        <a href="#cite_note-5">
         [5]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-6">
        <a href="#cite_note-6">
         [6]
        </a>
       </sup>
       ).
      </p>
      <p>
       Other
       <a class="mw-redirect" href="/wiki/Romance_language" title="Romance language">
        Romance languages
       </a>
       also have the sound, written
       <i>
        nh
       </i>
       in
       <a href="/wiki/Occitan_language" title="Occitan language">
        Occitan
       </a>
       (
       <i>
        espahnòl
       </i>
       /
       <i>
        espanhòu
       </i>
       ) and
       <a href="/wiki/Portuguese_language" title="Portuguese language">
        Portuguese
       </a>
       (
       <i>
        espanhol
       </i>
       ),
       <i>
        gn
       </i>
       in
       <a href="/wiki/Italian_language" title="Italian language">
        Italian
       </a>
       (
       <i>
        spagnolo
       </i>
       ) and
       <a href="/wiki/French_language" title="French language">
        French
       </a>
       (
       <i>
        espagnol
       </i>
       ), and
       <i>
        ny
       </i>
       in
       <a href="/wiki/Catalan_language" title="Catalan language">
        Catalan
       </a>
       and
       <a href="/wiki/Aragonese_language" title="Aragonese language">
        Aragonese
       </a>
       (
       <i>
        espanyol
       </i>
       ).
      </p>
      <p>
       The accented letter
       <i>
        <a href="/wiki/%C5%83" title="Ń">
         ń
        </a>
       </i>
       used in
       <a href="/wiki/Polish_language" title="Polish language">
        Polish
       </a>
       and the character
       <i>
        <a href="/wiki/%C5%87" title="Ň">
         ň
        </a>
       </i>
       used in
       <a href="/wiki/Czech_language" title="Czech language">
        Czech
       </a>
       and
       <a href="/wiki/Slovak_language" title="Slovak language">
        Slovak
       </a>
       are also equivalent to the Spanish letter
       <i>
        ñ
       </i>
       . The same sound is written
       <i>
        ny
       </i>
       in
       <a href="/wiki/Indonesian_language" title="Indonesian language">
        Indonesian
       </a>
       ,
       <a href="/wiki/Standard_Zhuang" title="Standard Zhuang">
        Zhuang
       </a>
       , and
       <a href="/wiki/Hungarian_language" title="Hungarian language">
        Hungarian
       </a>
       , and
       <i>
        nh
       </i>
       in
       <a href="/wiki/Vietnamese_language" title="Vietnamese language">
        Vietnamese
       </a>
       and
       <a href="/wiki/Portuguese_language" title="Portuguese language">
        Portuguese
       </a>
       .
      </p>
      <p>
       In
       <a href="/wiki/Tagalog_language" title="Tagalog language">
        Tagalog
       </a>
       ,
       <a href="/wiki/Visayan_languages" title="Visayan languages">
        Visayan
       </a>
       , and other
       <a href="/wiki/Languages_of_the_Philippines" title="Languages of the Philippines">
        Philippine Languages
       </a>
       , it is also written as
       <i>
        ny
       </i>
       for most terms. The conventional exceptions (with considerable variations) are proper names, which usually retain
       <i>
        ñ
       </i>
       and their original
       <a href="/wiki/Spanish_language_in_the_Philippines" title="Spanish language in the Philippines">
        Spanish
       </a>
       or Hispanicised spelling (
       <i>
        <a href="/wiki/Santo_Ni%C3%B1o_de_Ceb%C3%BA" title="Santo Niño de Cebú">
         Santo Niño
        </a>
       </i>
       ,
       <i>
        <a href="/wiki/Para%C3%B1aque" title="Parañaque">
         Parañaque
        </a>
       </i>
       ,
       <i>
        <a href="/wiki/Bamboo_Ma%C3%B1alac" title="Bamboo Mañalac">
         Mañalac
        </a>
       </i>
       ,
       <i>
        <a href="/wiki/Malaca%C3%B1ang_Palace" title="Malacañang Palace">
         Malacañan
        </a>
       </i>
       ). It is
       <a href="/wiki/Alphabetical_order" title="Alphabetical order">
        collated
       </a>
       as the 15th letter of the
       <a href="/wiki/Filipino_alphabet" title="Filipino alphabet">
        Filipino alphabet
       </a>
       . In
       <a href="/wiki/Filipino_orthography#Adoption_of_the_Latin_script" title="Filipino orthography">
        old Filipino orthography
       </a>
       , the letter was also used, along with
       <i>
        g
       </i>
       , to represent the velar nasal sound
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ŋ]
       </span>
       (except at the end of a word, when
       <i>
        ng
       </i>
       would be used) if appropriate instead of a tilde, which originally spanned a sequence of
       <i>
        n
       </i>
       and
       <i>
        g
       </i>
       (as in
       <i>
        n͠g
       </i>
       ), such as
       <i>
        pan͠galan
       </i>
       ("name"). That is because the old orthography was based on Spanish, and without the tilde,
       <i>
        pangalan
       </i>
       would have been pronounced with the sequence
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ŋɡ]
       </span>
       (therefore
       <i>
        pang-GAlan
       </i>
       ). The form
       <i>
        ñg
       </i>
       became a more common way to represent
       <i>
        n͠g
       </i>
       until the early 20th century, mainly because it was more readily available in typesets than the tilde spanning both letters.
      </p>
      <p>
       It is also used to represent the
       <a href="/wiki/Velar_nasal" title="Velar nasal">
        velar nasal
       </a>
       in
       <a href="/wiki/Crimean_Tatar_language" title="Crimean Tatar language">
        Crimean Tatar
       </a>
       . In
       <a class="mw-redirect" href="/wiki/Ja%EA%9E%91alif" title="Jaꞑalif">
        Latin-script writing
       </a>
       of the distantly-related
       <a href="/wiki/Tatar_language" title="Tatar language">
        Tatar language
       </a>
       ,
       <i>
        ñ
       </i>
       is sometimes used as a substitute for
       <a href="/wiki/N_with_descender" title="N with descender">
        n with descender
       </a>
       , which is not available on many computer systems.
      </p>
      <p>
       In the
       <a href="/wiki/Breton_language" title="Breton language">
        Breton language
       </a>
       , it
       <a href="/wiki/Nasal_vowel" title="Nasal vowel">
        nasalises
       </a>
       the preceding vowel, as in
       <i>
        Jañ
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        /ʒã/
       </span>
       , which corresponds to the French name
       <i>
        Jean
       </i>
       and has the same pronunciation.
      </p>
      <p>
       It is used in a number of
       <a class="mw-redirect" href="/wiki/English_(language)" title="English (language)">
        English
       </a>
       terms of Spanish origin, such as
       <i>
        <a href="/wiki/Jalape%C3%B1o" title="Jalapeño">
         jalapeño
        </a>
       </i>
       ,
       <i>
        <a href="/wiki/Pi%C3%B1a_colada" title="Piña colada">
         piña colada
        </a>
       </i>
       ,
       <i>
        <a href="/wiki/Pi%C3%B1ata" title="Piñata">
         piñata
        </a>
       </i>
       , and
       <i>
        <a class="mw-redirect" href="/wiki/El_Ni%C3%B1o-Southern_Oscillation" title="El Niño-Southern Oscillation">
         El Niño
        </a>
       </i>
       . The Spanish word
       <i>
        cañón
       </i>
       , however, became naturalized as
       <i>
        <a href="/wiki/Canyon" title="Canyon">
         canyon
        </a>
       </i>
       . Until the middle of the 20th century, adapting it as
       <i>
        nn
       </i>
       was more common in English, as in the phrase "
       <a href="/wiki/Battle_of_Corunna" title="Battle of Corunna">
        Battle of Corunna
       </a>
       "
       <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
        [
        <i>
         <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
          <span title="This claim needs references to reliable sources. (June 2015)">
           citation needed
          </span>
         </a>
        </i>
        ]
       </sup>
       . Now, it is almost always left unmodified.
      </p>
      <p>
       In the orthography for
       <a href="/wiki/Languages_of_Senegal" title="Languages of Senegal">
        languages of Senegal
       </a>
       ,
       <i>
        ñ
       </i>
       represents the
       <a href="/wiki/Palatal_nasal" title="Palatal nasal">
        palatal nasal
       </a>
       .
       <a href="/wiki/Senegal" title="Senegal">
        Senegal
       </a>
       is unique among countries of
       <a href="/wiki/West_Africa" title="West Africa">
        West Africa
       </a>
       in using this letter.
      </p>
      <p>
       The
       <a class="mw-redirect" href="/wiki/Sami_language" title="Sami language">
        Lule Sami language
       </a>
       uses the letter
       <i>
        <a href="/wiki/%C5%83" title="Ń">
         ń
        </a>
       </i>
       , which is often written
       <i>
        ñ
       </i>
       on computers, since
       <i>
        ñ
       </i>
       but not
       <i>
        ń
       </i>
       was available on keyboards and in the Latin-1 character set.
      </p>
      <h2>
       <span class="mw-headline" id="Cultural_significance">
        Cultural significance
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=%C3%91&amp;action=edit&amp;section=3" title="Edit section: Cultural significance">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <div class="thumb tright">
       <div class="thumbinner" style="width:302px;">
        <a class="image" href="/wiki/File:Potez_540-.JPG">
         <img alt="" class="thumbimage" data-file-height="518" data-file-width="963" height="161" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/30/Potez_540-.JPG/300px-Potez_540-.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/30/Potez_540-.JPG/450px-Potez_540-.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/30/Potez_540-.JPG/600px-Potez_540-.JPG 2x" width="300"/>
        </a>
        <div class="thumbcaption">
         <div class="magnify">
          <a class="internal" href="/wiki/File:Potez_540-.JPG" title="Enlarge">
          </a>
         </div>
         Serial letter 'Ñ'
         <a href="/wiki/Potez_540" title="Potez 540">
          Potez 540
         </a>
         plane of the Spanish Republican Air Force
        </div>
       </div>
      </div>
      <p>
       The letter
       <i>
        Ñ
       </i>
       has come to represent the identity of the Spanish language. Latino publisher Bill Teck labeled
       <a href="/wiki/Hispanic" title="Hispanic">
        Hispanic
       </a>
       culture and its influence on the United States "Generation Ñ" and later started a magazine with that name.
       <sup class="reference" id="cite_ref-7">
        <a href="#cite_note-7">
         [7]
        </a>
       </sup>
       Organizations such as the
       <a href="/wiki/Instituto_Cervantes" title="Instituto Cervantes">
        Instituto Cervantes
       </a>
       and the
       <a href="/wiki/National_Association_of_Hispanic_Journalists" title="National Association of Hispanic Journalists">
        National Association of Hispanic Journalists
       </a>
       have adopted the letter as their mark for Hispanic heritage.
      </p>
      <p>
       It was used in the
       <a href="/wiki/Spanish_Republican_Air_Force" title="Spanish Republican Air Force">
        Spanish Republican Air Force
       </a>
       for aircraft identification. The circumstances surrounding the crash of serial 'Ñ'
       <a href="/wiki/Potez_540" title="Potez 540">
        Potez 540
       </a>
       plane that was shot down over the
       <a href="/wiki/Sierra_de_G%C3%BAdar" title="Sierra de Gúdar">
        Sierra de Gúdar
       </a>
       range of the
       <a href="/wiki/Sistema_Ib%C3%A9rico" title="Sistema Ibérico">
        Sistema Ibérico
       </a>
       near
       <a href="/wiki/Valdelinares" title="Valdelinares">
        Valdelinares
       </a>
       inspired French writer
       <a href="/wiki/Andr%C3%A9_Malraux" title="André Malraux">
        André Malraux
       </a>
       to write the novel
       <i>
        L'Espoir
       </i>
       (1937), translated into English as
       <i>
        <a href="/wiki/Man%27s_Hope" title="Man's Hope">
         Man's Hope
        </a>
       </i>
       and made into the movie named
       <i>
        <a href="/wiki/Espoir:_Sierra_de_Teruel" title="Espoir: Sierra de Teruel">
         Espoir: Sierra de Teruel
        </a>
       </i>
       .
       <sup class="reference" id="cite_ref-8">
        <a href="#cite_note-8">
         [8]
        </a>
       </sup>
      </p>
      <p>
       In 1991, a
       <a class="mw-redirect" href="/wiki/European_Community" title="European Community">
        European Community
       </a>
       report recommended the repeal of a regulation preventing the sale in
       <a href="/wiki/Spain" title="Spain">
        Spain
       </a>
       of computer products not supporting "all the characteristics of the Spanish writing system," claiming that it was a
       <a href="/wiki/Protectionism" title="Protectionism">
        protectionist
       </a>
       measure against the principles of the free market. This would have allowed the distribution of keyboards without an "Ñ" key. The
       <a class="mw-redirect" href="/wiki/Real_Academia_Espa%C3%B1ola" title="Real Academia Española">
        Real Academia Española
       </a>
       stated that the matter was a serious attack against the language. Nobel Prize winner in literature
       <a href="/wiki/Gabriel_Garc%C3%ADa_M%C3%A1rquez" title="Gabriel García Márquez">
        Gabriel García Márquez
       </a>
       expressed his disdain over its elimination by saying: "The 'Ñ' is not an archaeological piece of junk, but just the opposite: a cultural leap of a Romance language that left the others behind in expressing with only one letter a sound that other languages continue to express with two."
       <sup class="reference" id="cite_ref-GGM_4-1">
        <a href="#cite_note-GGM-4">
         [4]
        </a>
       </sup>
      </p>
      <p>
       Among other forms of controversy are those pertaining to the anglicization of
       <a href="/wiki/Spanish_naming_customs" title="Spanish naming customs">
        Spanish surnames
       </a>
       . The replacement of
       <i>
        ñ
       </i>
       with another letter alters the pronunciation and meaning of a word or name, in the same manner that replacing any letter in a given word with another one would. For example,
       <i>
        <a href="/wiki/Pe%C3%B1a_(surname)" title="Peña (surname)">
         Peña
        </a>
       </i>
       is a common Spanish surname and a
       <a class="mw-redirect" href="/wiki/Common_noun" title="Common noun">
        common noun
       </a>
       that means "rocky hill"; it is often anglicized as
       <i>
        Pena
       </i>
       , changing the name to the Spanish word for "pity", often used in terms of sorrow.
      </p>
      <p>
       When
       <a href="/wiki/Federico_Pe%C3%B1a" title="Federico Peña">
        Federico Peña
       </a>
       was first running for
       <a class="mw-redirect" href="/wiki/Mayor_of_Denver" title="Mayor of Denver">
        mayor of Denver
       </a>
       in 1983, the
       <i>
        <a class="mw-redirect" href="/wiki/Denver_Post" title="Denver Post">
         Denver Post
        </a>
       </i>
       printed his name without the tilde as "Pena." After he won the election, they began printing his name with the tilde.
      </p>
      <p>
       Since 2011,
       <a href="/wiki/CNN_en_Espa%C3%B1ol" title="CNN en Español">
        CNN's Spanish-language news channel
       </a>
       incorporates a new logo wherein a tilde is placed over two
       <i>
        N
       </i>
       s.
      </p>
      <p>
       Another news channel,
       <a class="mw-redirect" href="/wiki/TLN_en_Espa%C3%B1ol" title="TLN en Español">
        TLN en Español
       </a>
       , has "tlñ", with an
       <i>
        ñ
       </i>
       taking the place of the expected
       <i>
        n
       </i>
       , as its logo.
      </p>

      <p>
       In
       <a href="/wiki/Unicode" title="Unicode">
        Unicode
       </a>
       <i>
        Ñ
       </i>
       has the code U+00D1 (decimal 209) while
       <i>
        ñ
       </i>
       has the code U+00F1 (decimal 241). Additionally, they can be generated using combining
       <a href="/wiki/Tilde" title="Tilde">
        tilde
       </a>
       , &amp;#x303, U+0303, decimal 771 followed by
       <i>
        N
       </i>
       or
       <i>
        n
       </i>
       .
      </p>
      <p>
       In
       <a class="mw-redirect" href="/wiki/HTML_character_entity_reference" title="HTML character entity reference">
        HTML character entity reference
       </a>
       , the codes for
       <i>
        Ñ
       </i>
       and
       <i>
        ñ
       </i>
       are
       <code>
        &amp;Ntilde;
       </code>
       and
       <code>
        &amp;ntilde;
       </code>
       or
       <code>
        &amp;#209;
       </code>
       and
       <code>
        &amp;#241;
       </code>
       .
      </p>
      <p>
       <i>
        Ñ
       </i>
       and
       <i>
        ñ
       </i>
       have their own key in the Spanish and Latin American keyboard layouts (see the corresponding sections at
       <a href="/wiki/Keyboard_layout" title="Keyboard layout">
        keyboard layout
       </a>
       and
       <a href="/wiki/Tilde#Role_of_mechanical_typewriters" title="Tilde">
        Tilde#Role of mechanical typewriters
       </a>
       ). The following instructions apply only to English-language keyboards.
      </p>
      <p>
       On
       <a href="/wiki/Android_(operating_system)" title="Android (operating system)">
        Android
       </a>
       devices, holding
       <i>
        n
       </i>
       or
       <i>
        N
       </i>
       down on the keyboard makes entry of
       <i>
        ñ
       </i>
       and
       <i>
        Ñ
       </i>
       possible.
      </p>
      <p>
       On
       <a class="mw-redirect" href="/wiki/Apple_Macintosh" title="Apple Macintosh">
        Apple Macintosh
       </a>
       operating systems (including
       <a class="mw-redirect" href="/wiki/Mac_OS_X" title="Mac OS X">
        Mac OS X
       </a>
       ), it can be typed by pressing and holding the Option key and then typing
       <i>
        N
       </i>
       , followed by typing either
       <i>
        N
       </i>
       or
       <i>
        n
       </i>
       .
      </p>
      <p>
       On the
       <a href="/wiki/IPhone" title="IPhone">
        iPhone
       </a>
       and
       <a href="/wiki/IPad" title="IPad">
        iPad
       </a>
       , which use the Apple
       <a href="/wiki/IOS" title="IOS">
        iOS
       </a>
       operating system, the
       <i>
        ñ
       </i>
       is accessed by holding down the "n" key, which opens a menu (on an English-language keyboard). Apple's
       <a href="/wiki/Mac_OS_X_Lion" title="Mac OS X Lion">
        Mac OS X 10.7 Lion
       </a>
       operating system also made the "ñ" available in the same way.
      </p>
      <p>
       To make a lowercase
       <i>
        ñ
       </i>
       in the
       <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">
        Microsoft Windows
       </a>
       operating system, hold down the Alt key and type the number 164 or 0241 on the numeric keypad (with Num Lock turned on).
       <sup class="reference" id="cite_ref-9">
        <a href="#cite_note-9">
         [9]
        </a>
       </sup>
       To make an uppercase
       <i>
        Ñ
       </i>
       , press Alt-165 or Alt-0209.
       <a href="/wiki/Character_Map_(Windows)" title="Character Map (Windows)">
        Character Map
       </a>
       in
       <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">
        Windows
       </a>
       identifies the letter as "Latin Small/Capital Letter N With Tilde". A soft (not physical) Spanish-language keyboard is easily installed in Windows.
      </p>
      <p>
       In
       <a href="/wiki/Microsoft_Word" title="Microsoft Word">
        Microsoft Word
       </a>
       ,
       <i>
        Ñ
       </i>
       can be typed by pressing Control-Shift-Tilde (~) and then an
       <i>
        N
       </i>
       .
      </p>
      <p>
       On
       <a href="/wiki/Linux" title="Linux">
        Linux
       </a>
       it can be created by pressing Ctrl+Shift+U and then typing '00D1' or '00F1', followed by space or Ctrl to end the character code input. This produces
       <i>
        Ñ
       </i>
       or
       <i>
        ñ
       </i>
       .
      </p>
      <p>
       Another option (for any operating system) is to configure the system to use the
       <a class="mw-redirect" href="/wiki/US-International" title="US-International">
        US-International
       </a>
       keyboard layout, with which
       <i>
        ñ
       </i>
       can be produced either by holding
       <a class="mw-redirect" href="/wiki/Alt_Gr" title="Alt Gr">
        Alt Gr
       </a>
       and then pressing
       <i>
        N
       </i>
       , or by typing the tilde (~) followed by
       <i>
        n
       </i>
       .
      </p>
      <p>
       Yet another option is to use a
       <a href="/wiki/Compose_key" title="Compose key">
        compose key
       </a>
       (hardware-based or software-emulated). Pressing the compose key, then
       <i>
        ~
       </i>
       , and then
       <i>
        n
       </i>
       results in
       <i>
        ñ
       </i>
       . A capital
       <i>
        N
       </i>
       can be substituted to produce
       <i>
        Ñ
       </i>
       , and in most cases the order of
       <i>
        ~
       </i>
       and
       <i>
        n
       </i>
       can be reversed.
      </p>
      <h3>
       <span class="mw-headline" id="Use_in_URLs">
        Use in URLs
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=%C3%91&amp;action=edit&amp;section=5" title="Edit section: Use in URLs">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       The letter
       <i>
        Ñ
       </i>
       may be used in
       <a href="/wiki/Internationalized_domain_name" title="Internationalized domain name">
        internationalized domain names
       </a>
       , but it will have to be converted from Unicode to
       <a href="/wiki/ASCII" title="ASCII">
        ASCII
       </a>
       during the registration process (i.e. from www.piñata.com to www.xn--piata-pta.com).
       <sup class="reference" id="cite_ref-10">
        <a href="#cite_note-10">
         [10]
        </a>
       </sup>
      </p>
      <p>
       In
       <a class="mw-redirect" href="/wiki/Uniform_Resource_Locator" title="Uniform Resource Locator">
        URLs
       </a>
       (except for the domain name),
       <i>
        Ñ
       </i>
       may be replaced by
       <code>
        %C3%91
       </code>
       , and
       <i>
        ñ
       </i>
       by
       <code>
        %C3%B1
       </code>
       . This is not needed for newer browsers. The hex digits represent the
       <a href="/wiki/UTF-8" title="UTF-8">
        UTF-8
       </a>
       encoding of the glyphs
       <i>
        Ñ
       </i>
       and
       <i>
        ñ
       </i>
       . This feature allows almost any Unicode character to be encoded, and it is considered important to support languages other than English.
      </p>
