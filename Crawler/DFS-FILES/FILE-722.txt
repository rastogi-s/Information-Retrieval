      <p>
       <i>
        <b>
         Ethnologue: Languages of the World
        </b>
       </i>
       is an annual reference publication in print and online that provides statistics and other information on the living
       <a href="/wiki/Language" title="Language">
        languages
       </a>
       of the world. It was first issued in 1951, and is now published annually by
       <a href="/wiki/SIL_International" title="SIL International">
        SIL International
       </a>
       , a U.S.-based, worldwide, Christian non-profit organization. SIL's main purpose is to study, develop and document languages to promote literacy and for religious purposes.
      </p>
      <p>
       As of 2017,
       <i>
        Ethnologue
       </i>
       contains web-based information on about 7,099 languages in its 20th edition,
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
       including the number of speakers, location, dialects, linguistic affiliations,
       <a href="/wiki/Exonym_and_endonym" title="Exonym and endonym">
        autonym
       </a>
       , availability of the Bible in each language and dialect described, a cursory description of revitalization efforts where reported, and an estimate of language viability using the
       <b>
        Expanded Graded Intergenerational Disruption Scale
       </b>
       (
       <i>
        <b>
         EGIDS
        </b>
       </i>
       ).
       <sup class="reference" id="cite_ref-2">
        <a href="#cite_note-2">
         [2]
        </a>
       </sup>
      </p>

      <h2>
       <span class="mw-headline" id="Overview">
        Overview
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Ethnologue&amp;action=edit&amp;section=1" title="Edit section: Overview">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       <i>
        Ethnologue
       </i>
       has been published by
       <a href="/wiki/SIL_International" title="SIL International">
        SIL International
       </a>
       (formerly known as the Summer Institute of Linguistics), a Christian
       <a href="/wiki/Linguistics" title="Linguistics">
        linguistic
       </a>
       service organization with an international office in
       <a href="/wiki/Dallas" title="Dallas">
        Dallas
       </a>
       , Texas. The organization studies numerous minority languages to facilitate language development, and to work with speakers of such language communities in translating portions of the Bible into their language.
       <sup class="reference" id="cite_ref-NYT_3-0">
        <a href="#cite_note-NYT-3">
         [3]
        </a>
       </sup>
      </p>
      <p>
       The determination of what characteristics define a single language depends upon
       <a href="/wiki/Sociolinguistics" title="Sociolinguistics">
        sociolinguistic
       </a>
       evaluation by various scholars; as the preface to
       <i>
        Ethnologue
       </i>
       states, "Not all scholars share the same set of criteria for what constitutes a 'language' and what features define a '
       <a href="/wiki/Dialect" title="Dialect">
        dialect
       </a>
       '."
       <i>
        Ethnologue
       </i>
       follows general linguistic criteria, which are based primarily on mutual intelligibility.
       <sup class="reference" id="cite_ref-4">
        <a href="#cite_note-4">
         [4]
        </a>
       </sup>
       Shared language intelligibility features are complex, and usually include etymological and grammatical evidence that is agreed upon by experts.
       <sup class="reference" id="cite_ref-5">
        <a href="#cite_note-5">
         [5]
        </a>
       </sup>
      </p>
      <p>
       In addition to choosing a primary name for a language,
       <i>
        Ethnologue
       </i>
       provides listings of the name(s) for the language and any dialects that are used by its speakers, government, foreigners and neighbors. Also included are any names that have been commonly referenced historically, regardless of whether a name is considered official, politically correct or offensive; this allows more complete historic research to be done. These lists of names are not necessarily complete.
      </p>
      <h2>
       <span class="mw-headline" id="History">
        History
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Ethnologue&amp;action=edit&amp;section=2" title="Edit section: History">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       In 1984,
       <i>
        Ethnologue
       </i>
       released a three-letter coding system, called an
       <a class="mw-redirect" href="/wiki/SIL_code" title="SIL code">
        SIL code
       </a>
       , to identify each language that it described. This set of codes significantly exceeded the scope of other standards, e.g.
       <a href="/wiki/ISO_639-1" title="ISO 639-1">
        ISO 639-1
       </a>
       and
       <a href="/wiki/ISO_639-2" title="ISO 639-2">
        ISO 639-2
       </a>
       .
       <sup class="reference" id="cite_ref-FOOTNOTEEveraert2009204_6-0">
        <a href="#cite_note-FOOTNOTEEveraert2009204-6">
         [6]
        </a>
       </sup>
       The 14th edition, published in 2000, included 7,148 language codes.
      </p>
      <p>
       In 2002,
       <i>
        Ethnologue
       </i>
       was asked to work with the
       <a href="/wiki/International_Organization_for_Standardization" title="International Organization for Standardization">
        International Organization for Standardization
       </a>
       (ISO) to integrate its codes into a draft international standard. The 15th edition of
       <i>
        Ethnologue
       </i>
       was the first edition to use this standard, called
       <a href="/wiki/ISO_639-3" title="ISO 639-3">
        ISO 639-3
       </a>
       . This standard is now administered separately from Ethnologue (though still by SIL according to rules established by ISO, and since then
       <i>
        Ethnologue
       </i>
       relies on the standard to determine what is listed as a language).
       <sup class="reference" id="cite_ref-ELL_7-0">
        <a href="#cite_note-ELL-7">
         [7]
        </a>
       </sup>
       In only one case,
       <i>
        Ethnologue
       </i>
       and the ISO standards treat languages slightly differently. ISO 639-3 considers
       <a href="/wiki/Akan_language" title="Akan language">
        Akan
       </a>
       to be a
       <a href="/wiki/ISO_639_macrolanguage" title="ISO 639 macrolanguage">
        macrolanguage
       </a>
       consisting of two distinct languages,
       <a href="/wiki/Twi" title="Twi">
        Twi
       </a>
       and
       <a href="/wiki/Fante_dialect" title="Fante dialect">
        Fante
       </a>
       , whereas
       <i>
        Ethnologue
       </i>
       considers Twi and Fante to be dialects of a single language (Akan), since they are mutually intelligible. This anomaly resulted because the ISO 639-2 standard has separate codes for Twi and Fante, which have separate literary traditions, and all 639-2 codes for individual languages are automatically part of 639–3, even though 639-3 would not normally assign them separate codes.
      </p>
      <p>
       In 2014, with the 17th edition,
       <i>
        Ethnologue
       </i>
       introduced a numerical code for language status using a framework called EGIDS (
       <a class="mw-redirect" href="/wiki/Expanded_Graded_Intergenerational_Disruption_Scale" title="Expanded Graded Intergenerational Disruption Scale">
        Expanded Graded Intergenerational Disruption Scale
       </a>
       ), an elaboration of
       <a href="/wiki/Joshua_Fishman" title="Joshua Fishman">
        Fishman's
       </a>
       GIDS (
       <a class="mw-redirect" href="/wiki/Graded_Intergenerational_Disruption_Scale" title="Graded Intergenerational Disruption Scale">
        Graded Intergenerational Disruption Scale
       </a>
       ). It ranks a language from 0 for an
       <a href="/wiki/World_language" title="World language">
        international language
       </a>
       to 10 for an
       <a href="/wiki/Extinct_language" title="Extinct language">
        extinct language
       </a>
       , i.e. a language with which no-one retains a sense of ethnic identity.
       <sup class="reference" id="cite_ref-8">
        <a href="#cite_note-8">
         [8]
        </a>
       </sup>
      </p>
      <p>
       In December 2015,
       <i>
        Ethnologue
       </i>
       launched a metered
       <a href="/wiki/Paywall" title="Paywall">
        paywall
       </a>
       ; users in
       <a href="/wiki/World_Bank_high-income_economy" title="World Bank high-income economy">
        high-income countries
       </a>
       who want to refer to more than seven pages of data per month must buy a
       <a class="mw-redirect" href="/wiki/Paid_subscription" title="Paid subscription">
        paid subscription
       </a>
       .
       <sup class="reference" id="cite_ref-9">
        <a href="#cite_note-9">
         [9]
        </a>
       </sup>
      </p>
      <p>
       As of 2017,
       <i>
        Ethnologue'
       </i>
       s 20th edition described 237
       <a class="mw-redirect" href="/wiki/Language_families" title="Language families">
        language families
       </a>
       including 86
       <a class="mw-redirect" href="/wiki/Language_isolates" title="Language isolates">
        language isolates
       </a>
       and six typological categories, namely
       <a href="/wiki/Sign_language" title="Sign language">
        sign languages
       </a>
       ,
       <a href="/wiki/Creole_language" title="Creole language">
        creoles
       </a>
       ,
       <a href="/wiki/Pidgin" title="Pidgin">
        pidgins
       </a>
       ,
       <a href="/wiki/Mixed_language" title="Mixed language">
        mixed languages
       </a>
       ,
       <a href="/wiki/Constructed_language" title="Constructed language">
        constructed languages
       </a>
       , and as yet
       <a class="mw-redirect" href="/wiki/Unclassified_languages" title="Unclassified languages">
        unclassified languages
       </a>
       .
       <sup class="reference" id="cite_ref-EthnologueList_10-0">
        <a href="#cite_note-EthnologueList-10">
         [10]
        </a>
       </sup>
      </p>
      <h2>
       <span class="mw-headline" id="Reputation">
        Reputation
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Ethnologue&amp;action=edit&amp;section=3" title="Edit section: Reputation">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       In 1986,
       <a href="/wiki/William_Bright" title="William Bright">
        William Bright
       </a>
       , then editor of the journal
       <i>
        <a href="/wiki/Language_(journal)" title="Language (journal)">
         Language
        </a>
       </i>
       , wrote of
       <i>
        Ethnologue
       </i>
       that it "is indispensable for any reference shelf on the languages of the world".
       <sup class="reference" id="cite_ref-11">
        <a href="#cite_note-11">
         [11]
        </a>
       </sup>
       In 2008 in the same journal, Lyle Campbell and Verónica Grondona said: "
       <i>
        Ethnologue
       </i>
       ...has become the standard reference, and its usefulness is hard to overestimate."
       <sup class="reference" id="cite_ref-12">
        <a href="#cite_note-12">
         [12]
        </a>
       </sup>
      </p>
      <p>
       In 2015, Harald Hammarström, an editor of
       <i>
        <a href="/wiki/Glottolog" title="Glottolog">
         Glottolog
        </a>
       </i>
       , criticized the publication for frequently lacking citations and failing to articulate clear principles of
       <a class="mw-redirect" href="/wiki/Language_classification" title="Language classification">
        language classification
       </a>
       and identification. However, he concluded that, on balance, "
       <i>
        Ethnologue
       </i>
       is an impressively comprehensive catalogue of world languages, and it is far superior to anything else produced prior to 2009."
       <sup class="reference" id="cite_ref-13">
        <a href="#cite_note-13">
         [13]
        </a>
       </sup>
      </p>
