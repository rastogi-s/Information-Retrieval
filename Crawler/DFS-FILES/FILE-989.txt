      <p>
       <b>
        Central Italian
       </b>
       (
       <i>
        italiano centrale
       </i>
       or
       <i>
        mediano
       </i>
       ) is a group of
       <a class="mw-redirect" href="/wiki/Italo-Dalmatian" title="Italo-Dalmatian">
        Italo-Dalmatian
       </a>
       <a href="/wiki/Romance_languages" title="Romance languages">
        Romance
       </a>
       dialects spoken in central
       <a href="/wiki/Italy" title="Italy">
        Italy
       </a>
       in
       <a href="/wiki/Lazio" title="Lazio">
        Lazio
       </a>
       ,
       <a href="/wiki/Umbria" title="Umbria">
        Umbria
       </a>
       , central
       <a href="/wiki/Marche" title="Marche">
        Marche
       </a>
       , the far south of
       <a href="/wiki/Tuscany" title="Tuscany">
        Tuscany
       </a>
       , and a small part of
       <a href="/wiki/Abruzzo" title="Abruzzo">
        Abruzzo
       </a>
       . The differences between these dialects are slight. They are closely related to
       <a href="/wiki/Tuscan_dialect" title="Tuscan dialect">
        Tuscan
       </a>
       and all are mutually intelligible with each other as well as with
       <a href="/wiki/Italian_language" title="Italian language">
        Standard Italian
       </a>
       . They are the
       <a href="/wiki/Regional_Italian" title="Regional Italian">
        Italian dialects
       </a>
       not affected by
       <a class="mw-redirect" href="/wiki/Substratum" title="Substratum">
        substrata
       </a>
       .
       <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
        [
        <i>
         <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
          <span title="This claim needs references to reliable sources. (July 2009)">
           citation needed
          </span>
         </a>
        </i>
        ]
       </sup>
      </p>
      <p>
       The following dialects are part of Central Italian:
      </p>
      <ul>
       <li>
        <a class="mw-redirect" href="/wiki/Marchigiano" title="Marchigiano">
         Marchigiano
        </a>
        (central part of
        <a href="/wiki/Marche" title="Marche">
         Marche
        </a>
        )
       </li>
       <li>
        Umbrian dialects (
        <a href="/wiki/Umbria" title="Umbria">
         Umbria
        </a>
        )
       </li>
       <li>
        <a href="/wiki/Sabino_dialect" title="Sabino dialect">
         Sabino
        </a>
        (
        <a href="/wiki/L%27Aquila" title="L'Aquila">
         L'Aquila
        </a>
        and
        <a href="/wiki/Province_of_Rieti" title="Province of Rieti">
         Province of Rieti
        </a>
        )
       </li>
       <li>
        Tuscia dialect (
        <a href="/wiki/Tuscia" title="Tuscia">
         Tuscia
        </a>
        , northern part of
        <a href="/wiki/Latium" title="Latium">
         Latium
        </a>
        )
       </li>
       <li>
        <a href="/wiki/Romanesco_dialect" title="Romanesco dialect">
         Romanesco
        </a>
        (
        <a class="mw-redirect" href="/wiki/Metropolitan_City_of_Rome" title="Metropolitan City of Rome">
         Metropolitan City of Rome
        </a>
        )
       </li>
       <li>
        Castelli Romani dialect (
        <a class="mw-redirect" href="/wiki/Castelli_Romani" title="Castelli Romani">
         Castelli Romani
        </a>
        )
       </li>
       <li>
        Ciociaro (
        <a href="/wiki/Ciociaria" title="Ciociaria">
         Ciociaria
        </a>
        , southern part of
        <a href="/wiki/Latium" title="Latium">
         Latium
        </a>
        )
       </li>
      </ul>
