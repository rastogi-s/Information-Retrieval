    Coordinate system
   </h1>

      <div class="thumb tright">
       <div class="thumbinner" style="width:302px;">
        <a class="image" href="/wiki/File:3D_Spherical.svg">
         <img alt="" class="thumbimage" data-file-height="516" data-file-width="558" height="277" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/4f/3D_Spherical.svg/300px-3D_Spherical.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/4f/3D_Spherical.svg/450px-3D_Spherical.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/4f/3D_Spherical.svg/600px-3D_Spherical.svg.png 2x" width="300"/>
        </a>
        <div class="thumbcaption">
         <div class="magnify">
          <a class="internal" href="/wiki/File:3D_Spherical.svg" title="Enlarge">
          </a>
         </div>
         The
         <a href="/wiki/Spherical_coordinate_system" title="Spherical coordinate system">
          spherical coordinate system
         </a>
         is commonly used in
         <i>
          physics
         </i>
         . It assigns three numbers (known as coordinates) to every point in
         <a href="/wiki/Euclidean_space" title="Euclidean space">
          Euclidean space
         </a>
         : radial distance
         <i>
          r
         </i>
         , polar angle
         <i>
          θ
         </i>
         (
         <a href="/wiki/Theta" title="Theta">
          theta
         </a>
         ), and azimuthal angle
         <i>
          φ
         </i>
         (
         <a href="/wiki/Phi" title="Phi">
          phi
         </a>
         ). The symbol
         <i>
          ρ
         </i>
         (
         <a href="/wiki/Rho" title="Rho">
          rho
         </a>
         ) is often used instead of
         <i>
          r
         </i>
         .
        </div>
       </div>
      </div>
      <p>
       In
       <a href="/wiki/Geometry" title="Geometry">
        geometry
       </a>
       , a
       <b>
        coordinate system
       </b>
       is a system which uses one or more
       <a href="/wiki/Number" title="Number">
        numbers
       </a>
       , or
       <b>
        coordinates
       </b>
       , to uniquely determine the position of the
       <a href="/wiki/Point_(geometry)" title="Point (geometry)">
        points
       </a>
       or other geometric elements on a
       <a href="/wiki/Manifold" title="Manifold">
        manifold
       </a>
       such as
       <a href="/wiki/Euclidean_space" title="Euclidean space">
        Euclidean space
       </a>
       .
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-2">
        <a href="#cite_note-2">
         [2]
        </a>
       </sup>
       The order of the coordinates is significant, and they are sometimes identified by their position in an ordered
       <a href="/wiki/Tuple" title="Tuple">
        tuple
       </a>
       and sometimes by a letter, as in "the
       <i>
        x
       </i>
       -coordinate". The coordinates are taken to be
       <a href="/wiki/Real_number" title="Real number">
        real numbers
       </a>
       in
       <a href="/wiki/Elementary_mathematics" title="Elementary mathematics">
        elementary mathematics
       </a>
       , but may be
       <a href="/wiki/Complex_number" title="Complex number">
        complex numbers
       </a>
       or elements of a more abstract system such as a
       <a href="/wiki/Commutative_ring" title="Commutative ring">
        commutative ring
       </a>
       . The use of a coordinate system allows problems in geometry to be translated into problems about numbers and
       <i>
        vice versa
       </i>
       ; this is the basis of
       <a href="/wiki/Analytic_geometry" title="Analytic geometry">
        analytic geometry
       </a>
       .
       <sup class="reference" id="cite_ref-3">
        <a href="#cite_note-3">
         [3]
        </a>
       </sup>
      </p>
      <p>
      </p>

      <h2>
       <span class="mw-headline" id="Common_coordinate_systems">
        Common coordinate systems
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Coordinate_system&amp;action=edit&amp;section=1" title="Edit section: Common coordinate systems">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <h3>
       <span class="mw-headline" id="Number_line">
        Number line
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Coordinate_system&amp;action=edit&amp;section=2" title="Edit section: Number line">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <div class="hatnote navigation-not-searchable" role="note">
       Main article:
       <a href="/wiki/Number_line" title="Number line">
        Number line
       </a>
      </div>
      <p>
       <span id="Number_line">
       </span>
       The simplest example of a coordinate system is the identification of points on a line with real numbers using the
       <i>
        number line
       </i>
       . In this system, an arbitrary point
       <i>
        O
       </i>
       (the
       <i>
        origin
       </i>
       ) is chosen on a given line. The coordinate of a point
       <i>
        P
       </i>
       is defined as the signed distance from
       <i>
        O
       </i>
       to
       <i>
        P
       </i>
       , where the signed distance is the distance taken as positive or negative depending on which side of the line
       <i>
        P
       </i>
       lies. Each point is given a unique coordinate and each real number is the coordinate of a unique point.
       <sup class="reference" id="cite_ref-4">
        <a href="#cite_note-4">
         [4]
        </a>
       </sup>
      </p>
      <div class="center">
       <div class="floatnone">
        <a class="image" href="/wiki/File:Number-line.svg" title="The number line">
         <img alt="The number line" data-file-height="50" data-file-width="750" height="50" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/93/Number-line.svg/750px-Number-line.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/93/Number-line.svg/1125px-Number-line.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/93/Number-line.svg/1500px-Number-line.svg.png 2x" width="750"/>
        </a>
       </div>
      </div>
      <h3>
       <span class="mw-headline" id="Cartesian_coordinate_system">
        Cartesian coordinate system
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Coordinate_system&amp;action=edit&amp;section=3" title="Edit section: Cartesian coordinate system">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <div class="hatnote navigation-not-searchable" role="note">
       Main article:
       <a href="/wiki/Cartesian_coordinate_system" title="Cartesian coordinate system">
        Cartesian coordinate system
       </a>
      </div>
      <div class="thumb tright">
       <div class="thumbinner" style="width:252px;">
        <a class="image" href="/wiki/File:Cartesian-coordinate-system.svg">
         <img alt="" class="thumbimage" data-file-height="354" data-file-width="354" height="250" src="//upload.wikimedia.org/wikipedia/commons/thumb/0/0e/Cartesian-coordinate-system.svg/250px-Cartesian-coordinate-system.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/0/0e/Cartesian-coordinate-system.svg/375px-Cartesian-coordinate-system.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/0/0e/Cartesian-coordinate-system.svg/500px-Cartesian-coordinate-system.svg.png 2x" width="250"/>
        </a>
        <div class="thumbcaption">
         <div class="magnify">
          <a class="internal" href="/wiki/File:Cartesian-coordinate-system.svg" title="Enlarge">
          </a>
         </div>
         The
         <a href="/wiki/Cartesian_coordinate_system" title="Cartesian coordinate system">
          Cartesian coordinate system
         </a>
         in the plane.
        </div>
       </div>
      </div>
      <p>
       The prototypical example of a coordinate system is the Cartesian coordinate system. In the
       <a href="/wiki/Plane_(geometry)" title="Plane (geometry)">
        plane
       </a>
       , two
       <a href="/wiki/Perpendicular" title="Perpendicular">
        perpendicular
       </a>
       lines are chosen and the coordinates of a point are taken to be the signed distances to the lines.
      </p>
      <div class="thumb tleft">
       <div class="thumbinner" style="width:252px;">
        <a class="image" href="/wiki/File:Rectangular_coordinates.svg">
         <img alt="Rectangular coordinates.svg" class="thumbimage" data-file-height="425" data-file-width="429" height="248" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/fd/Rectangular_coordinates.svg/250px-Rectangular_coordinates.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/fd/Rectangular_coordinates.svg/375px-Rectangular_coordinates.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/f/fd/Rectangular_coordinates.svg/500px-Rectangular_coordinates.svg.png 2x" width="250"/>
        </a>
        <div class="thumbcaption">
         <div class="magnify">
          <a class="internal" href="/wiki/File:Rectangular_coordinates.svg" title="Enlarge">
          </a>
         </div>
        </div>
       </div>
      </div>
      <p>
       In three dimensions, three perpendicular planes are chosen and the three coordinates of a point are the signed distances to each of the planes.
       <sup class="reference" id="cite_ref-5">
        <a href="#cite_note-5">
         [5]
        </a>
       </sup>
       This can be generalized to create
       <i>
        n
       </i>
       coordinates for any point in
       <i>
        n
       </i>
       -dimensional
       <a href="/wiki/Euclidean_space" title="Euclidean space">
        Euclidean space
       </a>
       .
      </p>
      <p>
       Depending on the direction and order of the coordinate axis the system may be a
       <a href="/wiki/Right-hand_rule" title="Right-hand rule">
        right-hand
       </a>
       or a left-hand system. This is one of many coordinate systems.
      </p>
      <div style="clear:both;">
      </div>
      <h3>
       <span class="mw-headline" id="Polar_coordinate_system">
        Polar coordinate system
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Coordinate_system&amp;action=edit&amp;section=4" title="Edit section: Polar coordinate system">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <div class="hatnote navigation-not-searchable" role="note">
       Main article:
       <a href="/wiki/Polar_coordinate_system" title="Polar coordinate system">
        Polar coordinate system
       </a>
      </div>
      <p>
       Another common coordinate system for the plane is the
       <i>
        polar coordinate system
       </i>
       .
       <sup class="reference" id="cite_ref-6">
        <a href="#cite_note-6">
         [6]
        </a>
       </sup>
       A point is chosen as the
       <i>
        pole
       </i>
       and a ray from this point is taken as the
       <i>
        polar axis
       </i>
       . For a given angle θ, there is a single line through the pole whose angle with the polar axis is θ (measured counterclockwise from the axis to the line). Then there is a unique point on this line whose signed distance from the origin is
       <i>
        r
       </i>
       for given number
       <i>
        r
       </i>
       . For a given pair of coordinates (
       <i>
        r
       </i>
       , θ) there is a single point, but any point is represented by many pairs of coordinates. For example, (
       <i>
        r
       </i>
       , θ), (
       <i>
        r
       </i>
       , θ+2π) and (−
       <i>
        r
       </i>
       , θ+π) are all polar coordinates for the same point. The pole is represented by (0, θ) for any value of θ.
      </p>
      <div style="clear:both;">
      </div>
      <h3>
       <span class="mw-headline" id="Cylindrical_and_spherical_coordinate_systems">
        Cylindrical and spherical coordinate systems
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Coordinate_system&amp;action=edit&amp;section=5" title="Edit section: Cylindrical and spherical coordinate systems">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <div class="hatnote navigation-not-searchable" role="note">
       Main articles:
       <a href="/wiki/Cylindrical_coordinate_system" title="Cylindrical coordinate system">
        Cylindrical coordinate system
       </a>
       and
       <a href="/wiki/Spherical_coordinate_system" title="Spherical coordinate system">
        Spherical coordinate system
       </a>
      </div>
      <p>
       There are two common methods for extending the polar coordinate system to three dimensions. In the
       <b>
        cylindrical coordinate system
       </b>
       , a
       <i>
        z
       </i>
       -coordinate with the same meaning as in Cartesian coordinates is added to the
       <i>
        r
       </i>
       and
       <i>
        θ
       </i>
       polar coordinates giving a triple (
       <i>
        r
       </i>
       ,
       <i>
        θ
       </i>
       ,
       <i>
        z
       </i>
       ).
       <sup class="reference" id="cite_ref-7">
        <a href="#cite_note-7">
         [7]
        </a>
       </sup>
       Spherical coordinates take this a step further by converting the pair of cylindrical coordinates (
       <i>
        r
       </i>
       ,
       <i>
        z
       </i>
       ) to polar coordinates (
       <i>
        ρ
       </i>
       ,
       <i>
        φ
       </i>
       ) giving a triple (
       <i>
        ρ
       </i>
       ,
       <i>
        θ
       </i>
       ,
       <i>
        φ
       </i>
       ).
       <sup class="reference" id="cite_ref-8">
        <a href="#cite_note-8">
         [8]
        </a>
       </sup>
      </p>
      <h3>
       <span class="mw-headline" id="Homogeneous_coordinate_system">
        Homogeneous coordinate system
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Coordinate_system&amp;action=edit&amp;section=6" title="Edit section: Homogeneous coordinate system">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <div class="hatnote navigation-not-searchable" role="note">
       Main article:
       <a href="/wiki/Homogeneous_coordinates" title="Homogeneous coordinates">
        Homogeneous coordinates
       </a>
      </div>
      <p>
       A point in the plane may be represented in
       <i>
        homogeneous coordinates
       </i>
       by a triple (
       <i>
        x
       </i>
       ,
       <i>
        y
       </i>
       ,
       <i>
        z
       </i>
       ) where
       <i>
        x
       </i>
       /
       <i>
        z
       </i>
       and
       <i>
        y
       </i>
       /
       <i>
        z
       </i>
       are the Cartesian coordinates of the point.
       <sup class="reference" id="cite_ref-9">
        <a href="#cite_note-9">
         [9]
        </a>
       </sup>
       This introduces an "extra" coordinate since only two are needed to specify a point on the plane, but this system is useful in that it represents any point on the
       <a href="/wiki/Projective_plane" title="Projective plane">
        projective plane
       </a>
       without the use of
       <a href="/wiki/Infinity" title="Infinity">
        infinity
       </a>
       . In general, a homogeneous coordinate system is one where only the ratios of the coordinates are significant and not the actual values.
      </p>
      <h3>
       <span class="mw-headline" id="Other_commonly_used_systems">
        Other commonly used systems
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Coordinate_system&amp;action=edit&amp;section=7" title="Edit section: Other commonly used systems">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       Some other common coordinate systems are the following:
      </p>
      <ul>
       <li>
        <a href="/wiki/Curvilinear_coordinates" title="Curvilinear coordinates">
         Curvilinear coordinates
        </a>
        are a generalization of coordinate systems generally; the system is based on the intersection of curves.
        <ul>
         <li>
          <a href="/wiki/Orthogonal_coordinates" title="Orthogonal coordinates">
           Orthogonal coordinates
          </a>
          :
          <a class="mw-redirect" href="/wiki/Coordinate_surfaces" title="Coordinate surfaces">
           coordinate surfaces
          </a>
          meet at right angles
         </li>
         <li>
          <a href="/wiki/Skew_coordinates" title="Skew coordinates">
           Skew coordinates
          </a>
          :
          <a class="mw-redirect" href="/wiki/Coordinate_surfaces" title="Coordinate surfaces">
           coordinate surfaces
          </a>
          are not orthogonal
         </li>
        </ul>
       </li>
       <li>
        The
        <a href="/wiki/Log-polar_coordinates" title="Log-polar coordinates">
         log-polar coordinate system
        </a>
        represents a point in the plane by the logarithm of the distance from the origin and an angle measured from a reference line intersecting the origin.
       </li>
       <li>
        <a href="/wiki/Pl%C3%BCcker_coordinates" title="Plücker coordinates">
         Plücker coordinates
        </a>
        are a way of representing lines in 3D Euclidean space using a six-tuple of numbers as
        <a href="/wiki/Homogeneous_coordinates" title="Homogeneous coordinates">
         homogeneous coordinates
        </a>
        .
       </li>
       <li>
        <a href="/wiki/Generalized_coordinates" title="Generalized coordinates">
         Generalized coordinates
        </a>
        are used in the
        <a href="/wiki/Lagrangian_mechanics" title="Lagrangian mechanics">
         Lagrangian
        </a>
        treatment of mechanics.
       </li>
       <li>
        <a href="/wiki/Canonical_coordinates" title="Canonical coordinates">
         Canonical coordinates
        </a>
        are used in the
        <a href="/wiki/Hamiltonian_mechanics" title="Hamiltonian mechanics">
         Hamiltonian
        </a>
        treatment of mechanics.
       </li>
       <li>
        <a class="mw-redirect" href="/wiki/Barycentric_coordinates_(mathematics)" title="Barycentric coordinates (mathematics)">
         Barycentric coordinates
        </a>
        as used for
        <a href="/wiki/Ternary_plot" title="Ternary plot">
         ternary plots
        </a>
        and more generally in the analysis of
        <a href="/wiki/Triangle" title="Triangle">
         triangles
        </a>
        .
       </li>
       <li>
        <a href="/wiki/Trilinear_coordinates" title="Trilinear coordinates">
         Trilinear coordinates
        </a>
        are used in the context of triangles.
       </li>
      </ul>
      <p>
       There are ways of describing curves without coordinates, using
       <a href="/wiki/Intrinsic_equation" title="Intrinsic equation">
        intrinsic equations
       </a>
       that use invariant quantities such as
       <a href="/wiki/Curvature" title="Curvature">
        curvature
       </a>
       and
       <a href="/wiki/Arc_length" title="Arc length">
        arc length
       </a>
       . These include:
      </p>
      <ul>
       <li>
        The
        <a href="/wiki/Whewell_equation" title="Whewell equation">
         Whewell equation
        </a>
        relates arc length and the
        <a href="/wiki/Tangential_angle" title="Tangential angle">
         tangential angle
        </a>
        .
       </li>
       <li>
        The
        <a href="/wiki/Ces%C3%A0ro_equation" title="Cesàro equation">
         Cesàro equation
        </a>
        relates arc length and curvature.
       </li>
      </ul>
      <h2>
       <span class="mw-headline" id="Coordinates_of_geometric_objects">
        Coordinates of geometric objects
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Coordinate_system&amp;action=edit&amp;section=8" title="Edit section: Coordinates of geometric objects">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       Coordinates systems are often used to specify the position of a point, but they may also be used to specify the position of more complex figures such as lines, planes, circles or spheres. For example,
       <a href="/wiki/Pl%C3%BCcker_coordinates" title="Plücker coordinates">
        Plücker coordinates
       </a>
       are used to determine the position of a line in space.
       <sup class="reference" id="cite_ref-10">
        <a href="#cite_note-10">
         [10]
        </a>
       </sup>
       When there is a need, the type of figure being described is used to distinguish the type of coordinate system, for example the term
       <i>
        <a href="/wiki/Line_coordinates" title="Line coordinates">
         line coordinates
        </a>
       </i>
       is used for any coordinate system that specifies the position of a line.
      </p>
      <p>
       It may occur that systems of coordinates for two different sets of geometric figures are equivalent in terms of their analysis. An example of this is the systems of homogeneous coordinates for points and lines in the projective plane. The two systems in a case like this are said to be
       <i>
        dualistic
       </i>
       . Dualistic systems have the property that results from one system can be carried over to the other since these results are only different interpretations of the same analytical result; this is known as the
       <i>
        principle of
        <a href="/wiki/Duality_(mathematics)" title="Duality (mathematics)">
         duality
        </a>
       </i>
       .
       <sup class="reference" id="cite_ref-11">
        <a href="#cite_note-11">
         [11]
        </a>
       </sup>
      </p>
      <h2>
       <span class="mw-headline" id="Transformations">
        Transformations
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Coordinate_system&amp;action=edit&amp;section=9" title="Edit section: Transformations">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <div class="hatnote navigation-not-searchable" role="note">
       See also:
       <a href="/wiki/Active_and_passive_transformation" title="Active and passive transformation">
        Active and passive transformation
       </a>
      </div>
      <div class="hatnote navigation-not-searchable" role="note">
       Main article:
       <a href="/wiki/List_of_common_coordinate_transformations" title="List of common coordinate transformations">
        List of common coordinate transformations
       </a>
      </div>
      <p>
       Because there are often many different possible coordinate systems for describing geometrical figures, it is important to understand how they are related. Such relations are described by
       <i>
        coordinate transformations
       </i>
       which give formulas for the coordinates in one system in terms of the coordinates in another system. For example, in the plane, if Cartesian coordinates (
       <i>
        x
       </i>
       ,
       <i>
        y
       </i>
       ) and polar coordinates (
       <i>
        r
       </i>
       ,
       <i>
        θ
       </i>
       ) have the same origin, and the polar axis is the positive
       <i>
        x
       </i>
       axis, then the coordinate transformation from polar to Cartesian coordinates is given by
       <i>
        x
       </i>
       =
       <i>
        r
       </i>
       cos
       <i>
        θ
       </i>
       and
       <i>
        y
       </i>
       =
       <i>
        r
       </i>
       sin
       <i>
        θ
       </i>
       .
      </p>
      <p>
       With every
       <a href="/wiki/Bijection" title="Bijection">
        bijection
       </a>
       from the space to itself two coordinate transformations can be associated:
      </p>
      <ul>
       <li>
        such that the new coordinates of the image of each point are the same as the old coordinates of the original point (the formulas for the mapping are the inverse of those for the coordinate transformation)
       </li>
       <li>
        such that the old coordinates of the image of each point are the same as the new coordinates of the original point (the formulas for the mapping are the same as those for the coordinate transformation)
       </li>
      </ul>
      <p>
       For example, in
       <a href="/wiki/Dimension" title="Dimension">
        1D
       </a>
       , if the mapping is a translation of 3 to the right, the first moves the origin from 0 to 3, so that the coordinate of each point becomes 3 less, while the second moves the origin from 0 to −3, so that the coordinate of each point becomes 3 more.
      </p>
      <h2>
       <span id="Coordinate_lines.2Fcurves_and_planes.2Fsurfaces">
       </span>
       <span class="mw-headline" id="Coordinate_lines/curves_and_planes/surfaces">
        <span id="Coordinate_line">
        </span>
        <span id="Coordinate_curve">
        </span>
        <span id="Coordinate_plane">
        </span>
        <span id="Coordinate_surface">
        </span>
        Coordinate lines/curves and planes/surfaces
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Coordinate_system&amp;action=edit&amp;section=10" title="Edit section: Coordinate lines/curves and planes/surfaces">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <div class="hatnote navigation-not-searchable" role="note">
       "Coordinate line" redirects here. It is not to be confused with
       <a href="/wiki/Line_coordinates" title="Line coordinates">
        Line coordinates
       </a>
       .
      </div>
      <div class="hatnote navigation-not-searchable" role="note">
       "Coordinate plane" redirects here. It is not to be confused with
       <a class="mw-redirect" href="/wiki/Plane_coordinates" title="Plane coordinates">
        Plane coordinates
       </a>
       .
      </div>
      <p>
       In two dimensions, if one of the coordinates in a point coordinate system is held constant and the other coordinate is allowed to vary, then the resulting curve is called a
       <b>
        coordinate curve
       </b>
       . In the Cartesian coordinate system the coordinate curves are, in fact,
       <a class="mw-redirect" href="/wiki/Straight_line" title="Straight line">
        straight lines
       </a>
       , thus
       <b>
        coordinate lines
       </b>
       . Specifically, they are the lines parallel to one of the coordinate axes. For other coordinate systems the coordinates curves may be general curves. For example, the coordinate curves in polar coordinates obtained by holding
       <i>
        r
       </i>
       constant are the circles with center at the origin. Coordinates systems for Euclidean space other than the Cartesian coordinate system are called
       <a href="/wiki/Curvilinear_coordinates" title="Curvilinear coordinates">
        curvilinear coordinate systems
       </a>
       .
       <sup class="reference" id="cite_ref-12">
        <a href="#cite_note-12">
         [12]
        </a>
       </sup>
       This procedure does not always make sense, for example there are no coordinate curves in a
       <a class="mw-redirect" href="/wiki/Homogeneous_coordinate_system" title="Homogeneous coordinate system">
        homogeneous coordinate system
       </a>
       .
      </p>
      <div class="thumb tleft">
       <div class="thumbinner" style="width:192px;">
        <a class="image" href="/wiki/File:Parabolic_coordinates_3D.png">
         <img alt="" class="thumbimage" data-file-height="598" data-file-width="560" height="203" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/e1/Parabolic_coordinates_3D.png/190px-Parabolic_coordinates_3D.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/e1/Parabolic_coordinates_3D.png/285px-Parabolic_coordinates_3D.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/e1/Parabolic_coordinates_3D.png/380px-Parabolic_coordinates_3D.png 2x" width="190"/>
        </a>
        <div class="thumbcaption">
         <div class="magnify">
          <a class="internal" href="/wiki/File:Parabolic_coordinates_3D.png" title="Enlarge">
          </a>
         </div>
         Coordinate surfaces of the three-dimensional paraboloidal coordinates.
        </div>
       </div>
      </div>
      <p>
       In three-dimensional space, if one coordinate is held constant and the other two are allowed to vary, then the resulting surface is called a
       <b>
        coordinate surface
       </b>
       . For example, the coordinate surfaces obtained by holding ρ constant in the
       <a href="/wiki/Spherical_coordinate_system" title="Spherical coordinate system">
        spherical coordinate system
       </a>
       are the spheres with center at the origin. In three-dimensional space the intersection of two coordinate surfaces is a coordinate curve. In the Cartesian coordinate system we may speak of
       <b>
        coordinate planes
       </b>
       .
      </p>
      <p>
       Similarly,
       <b>
        coordinate hypersurfaces
       </b>
       are the
       <span class="nowrap">
        (
        <i>
         n
        </i>
        − 1)
       </span>
       -dimensional spaces resulting from fixing a single coordinate of an
       <i>
        n
       </i>
       -dimensional coordinate system.
       <sup class="reference" id="cite_ref-13">
        <a href="#cite_note-13">
         [13]
        </a>
       </sup>
      </p>
      <h2>
       <span class="mw-headline" id="Coordinate_maps">
        Coordinate maps
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Coordinate_system&amp;action=edit&amp;section=11" title="Edit section: Coordinate maps">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <div class="hatnote navigation-not-searchable" role="note">
       Main article:
       <a class="mw-redirect" href="/wiki/Coordinate_map" title="Coordinate map">
        Coordinate map
       </a>
      </div>
      <div class="hatnote navigation-not-searchable" role="note">
       Further information:
       <a href="/wiki/Manifold" title="Manifold">
        Manifold
       </a>
      </div>
      <p>
       The concept of a
       <i>
        coordinate map
       </i>
       , or
       <i>
        coordinate chart
       </i>
       is central to the theory of manifolds. A coordinate map is essentially a coordinate system for a subset of a given space with the property that each point has exactly one set of coordinates. More precisely, a coordinate map is a
       <a href="/wiki/Homeomorphism" title="Homeomorphism">
        homeomorphism
       </a>
       from an open subset of a space
       <i>
        X
       </i>
       to an open subset of
       <b>
        R
       </b>
       <sup>
        <i>
         n
        </i>
       </sup>
       .
       <sup class="reference" id="cite_ref-14">
        <a href="#cite_note-14">
         [14]
        </a>
       </sup>
       It is often not possible to provide one consistent coordinate system for an entire space. In this case, a collection of coordinate maps are put together to form an
       <a href="/wiki/Atlas_(topology)" title="Atlas (topology)">
        atlas
       </a>
       covering the space. A space equipped with such an atlas is called a
       <i>
        manifold
       </i>
       and additional structure can be defined on a manifold if the structure is consistent where the coordinate maps overlap. For example, a
       <a href="/wiki/Differentiable_manifold" title="Differentiable manifold">
        differentiable manifold
       </a>
       is a manifold where the change of coordinates from one coordinate map to another is always a differentiable function.
      </p>
      <h2>
       <span class="mw-headline" id="Orientation-based_coordinates">
        Orientation-based coordinates
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Coordinate_system&amp;action=edit&amp;section=12" title="Edit section: Orientation-based coordinates">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       In
       <a href="/wiki/Geometry" title="Geometry">
        geometry
       </a>
       and
       <a href="/wiki/Kinematics" title="Kinematics">
        kinematics
       </a>
       , coordinate systems are used not only to describe the (linear) position of points, but also to describe the
       <a href="/wiki/Orientation_(geometry)" title="Orientation (geometry)">
        angular position
       </a>
       of axes, planes, and
       <a class="mw-redirect" href="/wiki/Rigid_bodies" title="Rigid bodies">
        rigid bodies
       </a>
       .
       <sup class="reference" id="cite_ref-15">
        <a href="#cite_note-15">
         [15]
        </a>
       </sup>
       In the latter case, the orientation of a second (typically referred to as "local") coordinate system, fixed to the node, is defined based on the first (typically referred to as "global" or "world" coordinate system). For instance, the orientation of a rigid body can be represented by an orientation
       <a href="/wiki/Matrix_(mathematics)" title="Matrix (mathematics)">
        matrix
       </a>
       , which includes, in its three columns, the
       <a class="mw-redirect" href="/wiki/Cartesian_coordinates" title="Cartesian coordinates">
        Cartesian coordinates
       </a>
       of three points. These points are used to define the orientation of the axes of the local system; they are the tips of three
       <a class="mw-redirect" href="/wiki/Unit_vectors" title="Unit vectors">
        unit vectors
       </a>
       aligned with those axes.
      </p>
