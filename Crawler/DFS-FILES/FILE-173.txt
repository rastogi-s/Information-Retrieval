      <p>
       Wikidata is a
       <a href="/wiki/Document-oriented_database" title="Document-oriented database">
        document-oriented database
       </a>
       , focused on items. Each item represents a topic (or an
       <a href="/wiki/Wikipedia:Administration#Data_structure_and_development" title="Wikipedia:Administration">
        administrative page
       </a>
       used to maintain Wikipedia) and is identified by a unique number, prefixed with the letter Q—for example, the item for the topic
       <a href="/wiki/Politics" title="Politics">
        Politics
       </a>
       is
       <a class="extiw" href="https://www.wikidata.org/wiki/Q7163" title="d:Q7163">
        Q7163
       </a>
       . This enables the basic information required to identify the topic the item covers to be translated without favouring any language.
      </p>
      <p>
       Information is added to items by creating statements. Statements take the form of
       <a class="mw-redirect" href="/wiki/Key-value_pair" title="Key-value pair">
        key-value pairs
       </a>
       , with each statement consisting of a property (the key) and a value linked to the property.
      </p>

      <h2>
       <span class="mw-headline" id="Development_history">
        Development history
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Wikidata&amp;action=edit&amp;section=2" title="Edit section: Development history">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       The creation of the project was funded by donations from the
       <a href="/wiki/Allen_Institute_for_Artificial_Intelligence" title="Allen Institute for Artificial Intelligence">
        Allen Institute for Artificial Intelligence
       </a>
       , the
       <a href="/wiki/Gordon_and_Betty_Moore_Foundation" title="Gordon and Betty Moore Foundation">
        Gordon and Betty Moore Foundation
       </a>
       , and
       <a class="mw-redirect" href="/wiki/Google,_Inc." title="Google, Inc.">
        Google, Inc.
       </a>
       , totaling
       <a href="/wiki/Euro" title="Euro">
        €
       </a>
       1.3 million.
       <sup class="reference" id="cite_ref-5">
        <a href="#cite_note-5">
         [5]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-6">
        <a href="#cite_note-6">
         [6]
        </a>
       </sup>
       Initial development of the project is being overseen by
       <a class="mw-redirect" href="/wiki/Wikimedia_Deutschland" title="Wikimedia Deutschland">
        Wikimedia Deutschland
       </a>
       and has been split into three phases:
       <sup class="reference" id="cite_ref-wikidata_on_meta_7-0">
        <a href="#cite_note-wikidata_on_meta-7">
         [7]
        </a>
       </sup>
      </p>
      <ol>
       <li>
        Centralising interlanguage links – links between Wikipedia articles about the same topic in different languages
       </li>
       <li>
        Providing a central place for infobox data for all Wikipedias
       </li>
       <li>
        Creating and updating list articles based on data in Wikidata
       </li>
      </ol>
      <h3>
       <span class="mw-headline" id="Phase_1">
        Phase 1
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Wikidata&amp;action=edit&amp;section=3" title="Edit section: Phase 1">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       Wikidata was launched on 30 October 2012 and was the first new project of the Wikimedia Foundation since 2006.
       <sup class="reference" id="cite_ref-Wikidata_Main_Page_archive_2-1">
        <a href="#cite_note-Wikidata_Main_Page_archive-2">
         [2]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-8">
        <a href="#cite_note-8">
         [8]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-9">
        <a href="#cite_note-9">
         [9]
        </a>
       </sup>
       At this time, only the first phase was available. This enabled items to be created and filled with basic information: a label – a name or title, aliases – alternative terms for the label, a description, and links to articles about the topic in all the various language editions of Wikipedia.
      </p>
      <p>
       Historically, a Wikipedia article would include a list of interlanguage links, being links to articles on the same topic in other editions of Wikipedia, if present. Initially, Wikidata was a self-contained repository of interlanguage links. No Wikipedia language editions were able to access Wikidata, so they needed to continue to maintain their own lists of interlanguage links. On 14 January 2013, the
       <a href="/wiki/Hungarian_Wikipedia" title="Hungarian Wikipedia">
        Hungarian Wikipedia
       </a>
       became the first to enable the provision of interlanguage links via Wikidata.
       <sup class="reference" id="cite_ref-10">
        <a href="#cite_note-10">
         [10]
        </a>
       </sup>
       This functionality was extended to the
       <a href="/wiki/Hebrew_Wikipedia" title="Hebrew Wikipedia">
        Hebrew
       </a>
       and
       <a href="/wiki/Italian_Wikipedia" title="Italian Wikipedia">
        Italian
       </a>
       Wikipedias on 30 January, to the
       <a href="/wiki/English_Wikipedia" title="English Wikipedia">
        English Wikipedia
       </a>
       on 13 February and to all other Wikipedias on 6 March.
       <sup class="reference" id="cite_ref-wmde_30_jan_11-0">
        <a href="#cite_note-wmde_30_jan-11">
         [11]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-12">
        <a href="#cite_note-12">
         [12]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-13">
        <a href="#cite_note-13">
         [13]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-14">
        <a href="#cite_note-14">
         [14]
        </a>
       </sup>
       After no consensus was reached over a proposal to restrict the removal of language links from the English Wikipedia,
       <sup class="reference" id="cite_ref-15">
        <a href="#cite_note-15">
         [15]
        </a>
       </sup>
       the power to delete them from the English Wikipedia was granted to automatic editors (
       <a href="/wiki/Internet_bot" title="Internet bot">
        bots
       </a>
       ). On 23 September 2013, phase 1 went live on
       <a href="/wiki/Wikimedia_Commons" title="Wikimedia Commons">
        Wikimedia Commons
       </a>
       .
       <sup class="reference" id="cite_ref-16">
        <a href="#cite_note-16">
         [16]
        </a>
       </sup>
      </p>
      <h3>
       <span class="mw-headline" id="Phase_2">
        Phase 2
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Wikidata&amp;action=edit&amp;section=4" title="Edit section: Phase 2">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       The initial features of the second phase were deployed on 4 February 2013, introducing statements to Wikidata entries. The values were initially limited to two data types (items and images on Wikimedia Commons), with more data types (such as coordinates and dates) to follow later. The first new type, string, was deployed on 6 March.
       <sup class="reference" id="cite_ref-17">
        <a href="#cite_note-17">
         [17]
        </a>
       </sup>
      </p>
      <p>
       The ability for the various language editions of Wikipedia to access data from Wikidata was rolled out progressively between 27 March and 25 April 2013.
       <sup class="reference" id="cite_ref-18">
        <a href="#cite_note-18">
         [18]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-19">
        <a href="#cite_note-19">
         [19]
        </a>
       </sup>
      </p>
      <p>
       On 16 September 2015, Wikidata began allowing so-called
       <i>
        arbitrary access
       </i>
       , or access from a given Wikidata item to the properties of items not directly connected to it. For example, it became possible to read data about Germany from the Berlin article, which was not feasible before.
       <sup class="reference" id="cite_ref-20">
        <a href="#cite_note-20">
         [20]
        </a>
       </sup>
       On 27 April 2016
       <i>
        arbitrary access
       </i>
       was activated on
       <a href="/wiki/Wikimedia_Commons" title="Wikimedia Commons">
        Wikimedia Commons
       </a>
       .
       <sup class="reference" id="cite_ref-21">
        <a href="#cite_note-21">
         [21]
        </a>
       </sup>
      </p>
      <h3>
       <span class="mw-headline" id="Phase_3">
        Phase 3
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Wikidata&amp;action=edit&amp;section=5" title="Edit section: Phase 3">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       Phase 3 will involve database querying and the creation of lists based on data stored on Wikidata.
       <sup class="reference" id="cite_ref-wikidata_on_meta_7-1">
        <a href="#cite_note-wikidata_on_meta-7">
         [7]
        </a>
       </sup>
       As of October 2016 two tools for querying Wikidata (
       <a class="extiw" href="https://www.wikidata.org/wiki/Wikidata:List_of_queries" title="wikidata:Wikidata:List of queries">
        Wikidata:List of queries
       </a>
       ) were available: AutoList
       <sup class="reference" id="cite_ref-22">
        <a href="#cite_note-22">
         [22]
        </a>
       </sup>
       and
       <a href="/wiki/Wikipedia:PetScan" title="Wikipedia:PetScan">
        PetScan
       </a>
       ,
       <sup class="reference" id="cite_ref-23">
        <a href="#cite_note-23">
         [23]
        </a>
       </sup>
       additionally to a public
       <a href="/wiki/SPARQL" title="SPARQL">
        SPARQL
       </a>
       endpoint.
       <sup class="reference" id="cite_ref-24">
        <a href="#cite_note-24">
         [24]
        </a>
       </sup>
      </p>
      <h2>
       <span class="mw-headline" id="Reception">
        Reception
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Wikidata&amp;action=edit&amp;section=6" title="Edit section: Reception">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       There is concern that the project is being influenced by lobbying companies, PR professionals and
       <a href="/wiki/Search_engine_optimization" title="Search engine optimization">
        search engine optimizers
       </a>
       .
       <sup class="reference" id="cite_ref-Kolbe2015_25-0">
        <a href="#cite_note-Kolbe2015-25">
         [25]
        </a>
       </sup>
      </p>
      <p>
       As of December 2015
       <sup class="plainlinks noexcerpt noprint asof-tag update" style="display:none;">
        <a class="external text" href="//en.wikipedia.org/w/index.php?title=Wikidata&amp;action=edit">
         [update]
        </a>
       </sup>
       , according to Wikimedia statistics, half of the information in Wikidata is unsourced.
       <sup class="reference" id="cite_ref-Kolbe2015_25-1">
        <a href="#cite_note-Kolbe2015-25">
         [25]
        </a>
       </sup>
       Another 30% is labeled as having come from Wikipedia, but with no indication as to which article.
       <sup class="reference" id="cite_ref-Kolbe2015_25-2">
        <a href="#cite_note-Kolbe2015-25">
         [25]
        </a>
       </sup>
      </p>
