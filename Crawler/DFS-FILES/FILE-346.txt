      <p>
       <b>
        ISO 639-2:1998
       </b>
       ,
       <i>
        Codes for the representation of names of languages — Part 2: Alpha-3 code
       </i>
       , is the second part of the
       <a href="/wiki/ISO_639" title="ISO 639">
        ISO 639
       </a>
       <a href="/wiki/International_standard" title="International standard">
        standard
       </a>
       , which lists
       <a href="/wiki/Language_code" title="Language code">
        codes
       </a>
       for the representation of the names of
       <a class="mw-redirect" href="/wiki/Languages" title="Languages">
        languages
       </a>
       . The three-letter codes given for each language in this part of the standard are referred to as "Alpha-3" codes. There are 464 entries in the
       <a href="/wiki/List_of_ISO_639-2_codes" title="List of ISO 639-2 codes">
        list of ISO 639-2 codes
       </a>
       .
      </p>
      <p>
       The
       <a href="/wiki/United_States" title="United States">
        US
       </a>
       <a href="/wiki/Library_of_Congress" title="Library of Congress">
        Library of Congress
       </a>
       is the registration authority for ISO 639-2 (referred to as ISO 639-2/RA). As registration authority, the LOC receives and reviews proposed changes; they also have representation on the ISO 639-RA Joint Advisory Committee responsible for maintaining the ISO 639 code tables.
      </p>
      <p>
      </p>

      <h2>
       <span class="mw-headline" id="History_and_relationship_to_other_ISO_639_standards">
        History and relationship to other ISO 639 standards
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=ISO_639-2&amp;action=edit&amp;section=1" title="Edit section: History and relationship to other ISO 639 standards">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       Work was begun on the ISO 639-2 standard in 1989, because the
       <a href="/wiki/ISO_639-1" title="ISO 639-1">
        ISO 639-1
       </a>
       standard, which uses only two-letter codes for languages, is not able to accommodate a sufficient number of languages. The ISO 639-2 standard was first released in 1998.
      </p>
      <p>
       In practice, ISO 639-2 has largely been superseded by
       <a href="/wiki/ISO_639-3" title="ISO 639-3">
        ISO 639-3
       </a>
       (2007), which includes codes for all the individual languages in ISO 639-2 plus many more. It also includes the special and reserved codes, and is designed not to conflict with ISO 639-2. ISO 639-3, however, does not include any of the collective languages in ISO 639-2; most of these are included in
       <a href="/wiki/ISO_639-5" title="ISO 639-5">
        ISO 639-5
       </a>
       .
      </p>
      <h2>
       <span class="mw-headline" id="B_and_T_codes">
        B and T codes
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=ISO_639-2&amp;action=edit&amp;section=2" title="Edit section: B and T codes">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       While most languages are given one code by the standard, twenty of the languages described have two three-letter codes, a "bibliographic" code (ISO 639-2/B), which is derived from the English name for the language and was a necessary
       <a class="mw-redirect" href="/wiki/Legacy_systems" title="Legacy systems">
        legacy
       </a>
       feature, and a "terminological" code (ISO 639-2/T), which is derived from the native name for the language and resembles the language's two-letter code in ISO 639-1. There were originally 22 B codes;
       <b>
        scc
       </b>
       and
       <b>
        scr
       </b>
       are now deprecated.
      </p>
      <p>
       In general the T codes are favored;
       <a href="/wiki/ISO_639-3" title="ISO 639-3">
        ISO 639-3
       </a>
       uses ISO 639-2/T. However,
       <a href="/wiki/ISO_15924" title="ISO 15924">
        ISO 15924
       </a>
       derives its codes from ISO 639-2/B when possible.
      </p>
      <h2>
       <span class="mw-headline" id="Scopes_and_types">
        Scopes and types
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=ISO_639-2&amp;action=edit&amp;section=3" title="Edit section: Scopes and types">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       The codes in ISO 639-2 have a variety of "scopes of denotation", or types of meaning and use, some of which are described in more detail below.
      </p>
      <ul>
       <li>
        Individual languages
       </li>
       <li>
        Macrolanguages (see
        <a href="/wiki/ISO_639_macrolanguage" title="ISO 639 macrolanguage">
         ISO 639 macrolanguage
        </a>
        )
       </li>
       <li>
        <a href="#Collections_of_languages">
         Collections of languages
        </a>
       </li>
       <li>
        Dialects
       </li>
       <li>
        <a href="#Reserved_for_local_use">
         Reserved for local use
        </a>
       </li>
       <li>
        <a href="#Special_situations">
         Special situations
        </a>
       </li>
      </ul>
      <p>
       Individual languages are further classified as to type:
      </p>
      <ul>
       <li>
        Living languages
       </li>
       <li>
        Extinct languages
       </li>
       <li>
        Ancient languages
       </li>
       <li>
        Historic languages
       </li>
       <li>
        Constructed languages
       </li>
      </ul>
      <p>
       <span id="Collective_language_codes">
       </span>
      </p>
      <h3>
       <span class="mw-headline" id="Collections_of_languages">
        Collections of languages
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=ISO_639-2&amp;action=edit&amp;section=4" title="Edit section: Collections of languages">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       Some ISO 639-2 codes that are commonly used for languages do not precisely represent a particular language or some related languages (as the above macrolanguages). They are regarded as collective language codes and are excluded from
       <a href="/wiki/ISO_639-3" title="ISO 639-3">
        ISO 639-3
       </a>
       . For a definition of macrolanguages and collective languages see
       <a class="external autonumber" href="http://www.sil.org/iso639-3/scope.asp" rel="nofollow">
        [1]
       </a>
       .
      </p>
      <p>
       The collective language codes in ISO 639-2 are listed below.
      </p>

       The following two codes are identified as collective codes in ISO 639-2 but are (at present) missing from ISO 639-5:

      <h3>
       <span class="mw-headline" id="Reserved_for_local_use">
        Reserved for local use
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=ISO_639-2&amp;action=edit&amp;section=5" title="Edit section: Reserved for local use">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       The interval from
       <b>
        qaa
       </b>
       to
       <b>
        qtz
       </b>
       is 'reserved for local use' and is not used in ISO 639-2 nor in
       <a href="/wiki/ISO_639-3#Special_codes" title="ISO 639-3">
        ISO 639-3
       </a>
       . These codes are typically used privately for languages not (yet) in either standard.
      </p>
      <h3>
       <span class="mw-headline" id="Special_situations">
        Special situations
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=ISO_639-2&amp;action=edit&amp;section=6" title="Edit section: Special situations">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       There are four generic codes for special situations:
      </p>
      <ul>
       <li>
        <b>
         mis
        </b>
        is listed as "uncoded languages" (originally an abbreviation for "miscellaneous")
       </li>
       <li>
        <b>
         mul
        </b>
        (for multiple languages) is applied when several languages are used and it is not practical to specify all the appropriate language codes
       </li>
       <li>
        <b>
         und
        </b>
        (for undetermined) is used in situations in which a language or languages must be indicated but the language cannot be identified.
       </li>
       <li>
        <b>
         zxx
        </b>
        is listed in the code list as "no linguistic content", e.g. animal sounds (added 2006-01-11)
       </li>
      </ul>
      <p>
       These four codes are also used in
       <a href="/wiki/ISO_639-3#Special_codes" title="ISO 639-3">
        ISO 639-3
       </a>
       .
      </p>
