      <p>
       The
       <b>
        diaeresis
       </b>
       (
       <span class="nowrap">
        <span class="IPA nopopups noexcerpt">
         <a href="/wiki/Help:IPA/English" title="Help:IPA/English">
          /
          <span style="border-bottom:1px dotted">
           <span title="'d' in 'dye'">
            d
           </span>
           <span title="/aɪ/: 'i' in 'tide'">
            aɪ
           </span>
           <span title="/ˈ/: primary stress follows">
            ˈ
           </span>
           <span title="/ɛr/: 'err' in 'merry'">
            ɛr
           </span>
           <span title="/ə/: 'a' in 'about'">
            ə
           </span>
           <span title="'s' in 'sigh'">
            s
           </span>
           <span title="/ɪ/: 'i' in 'kit'">
            ɪ
           </span>
           <span title="'s' in 'sigh'">
            s
           </span>
          </span>
          /
         </a>
        </span>
       </span>
       <a href="/wiki/Help:Pronunciation_respelling_key" title="Help:Pronunciation respelling key">
        <i title="English pronunciation respelling">
         dy-
         <span style="font-size:90%">
          ERR
         </span>
         -ə-sis
        </i>
       </a>
       ; plural:
       <b>
        diaereses
       </b>
       ), also spelled
       <b>
        diæresis
       </b>
       or
       <b>
        dieresis
       </b>
       and also known as the
       <b>
        tréma
       </b>
       (also:
       <b>
        trema
       </b>
       ) or the
       <b>
        umlaut
       </b>
       , is a
       <a class="mw-redirect" href="/wiki/Diacritical_mark" title="Diacritical mark">
        diacritical mark
       </a>
       that consists of two dots
       <span class="nowrap">
        (
        <b>
         ¨
        </b>
        )
       </span>
       placed over a letter, usually a
       <a href="/wiki/Vowel" title="Vowel">
        vowel
       </a>
       . When that letter is an
       <i>
        i
       </i>
       or a
       <i>
        j
       </i>
       , the diacritic replaces the
       <a href="/wiki/Tittle" title="Tittle">
        tittle
       </a>
       :
       <i>
        ï
       </i>
       .
       <sup class="reference" id="cite_ref-Unicode5.0_1-0">
        <a href="#cite_note-Unicode5.0-1">
         [1]
        </a>
       </sup>
      </p>
      <p>
       The
       <i>
        diaeresis
       </i>
       and the
       <i>
        umlaut
       </i>
       are diacritics marking two distinct
       <a href="/wiki/Phonology" title="Phonology">
        phonological
       </a>
       phenomena. The diaeresis represents the phenomenon also known as
       <i>
        diaeresis
       </i>
       or
       <i>
        <a href="/wiki/Hiatus_(linguistics)" title="Hiatus (linguistics)">
         hiatus
        </a>
       </i>
       in which a vowel letter is pronounced separately from an adjacent vowel and not as part of a
       <a href="/wiki/Digraph_(orthography)" title="Digraph (orthography)">
        digraph
       </a>
       or
       <a href="/wiki/Diphthong" title="Diphthong">
        diphthong
       </a>
       . The
       <i>
        <a href="/wiki/Umlaut_(linguistics)" title="Umlaut (linguistics)">
         umlaut
        </a>
       </i>
       (
       <span class="nowrap">
        <span class="IPA nopopups noexcerpt">
         <a href="/wiki/Help:IPA/English" title="Help:IPA/English">
          /
          <span style="border-bottom:1px dotted">
           <span title="/ˈ/: primary stress follows">
            ˈ
           </span>
           <span title="/ʊ/: 'oo' in 'foot'">
            ʊ
           </span>
           <span title="'m' in 'my'">
            m
           </span>
           <span title="'l' in 'lie'">
            l
           </span>
           <span title="/aʊ/: 'ou' in 'mouth'">
            aʊ
           </span>
           <span title="'t' in 'tie'">
            t
           </span>
          </span>
          /
         </a>
        </span>
       </span>
       <a href="/wiki/Help:Pronunciation_respelling_key" title="Help:Pronunciation respelling key">
        <i title="English pronunciation respelling">
         <span style="font-size:90%">
          UUM
         </span>
         -lowt
        </i>
       </a>
       ), in contrast, indicates a
       <a class="mw-redirect" href="/wiki/Sound_shift" title="Sound shift">
        sound shift
       </a>
       . These two diacritics originated separately; the diaeresis is considerably older. Nevertheless, in modern computer systems using
       <a href="/wiki/Unicode" title="Unicode">
        Unicode
       </a>
       , the umlaut and diaeresis diacritics are identical, e.g.
       <span class="nowrap">
        <span class="monospaced" style="font-family: monospace, monospace;">
         U+00E4
        </span>
       </span>
       <span style="font-size:125%">
        ä
       </span>
       <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">
        LATIN SMALL LETTER A WITH DIAERESIS
       </span>
       (HTML
       <code>
        &amp;#228;
       </code>
       <b>
        ·
       </b>
       <code>
        &amp;auml;
       </code>
       ) represents both
       <i>
        a-umlaut
       </i>
       and
       <i>
        a-diaeresis
       </i>
       .
      </p>
      <p>
       The same symbol is also used as a diacritic in other cases, distinct from both diaeresis and umlaut. For example, in
       <a href="/wiki/Albanian_language" title="Albanian language">
        Albanian
       </a>
       and
       <a href="/wiki/Tagalog_language" title="Tagalog language">
        Tagalog
       </a>
       <i>
        ë
       </i>
       represents a
       <a href="/wiki/Schwa" title="Schwa">
        schwa
       </a>
       .
      </p>

      <h2>
       <span class="mw-headline" id="Names">
        Names
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=1" title="Edit section: Names">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       The word
       <i>
        diaeresis
       </i>
       is from
       <a href="/wiki/Ancient_Greek" title="Ancient Greek">
        Greek
       </a>
       <i>
        <span lang="grc-Latn" xml:lang="grc-Latn">
         diaíresis
        </span>
       </i>
       (
       <span lang="grc" xml:lang="grc">
        διαίρεσις
       </span>
       ), meaning "division", "separation", or "distinction".
       <sup class="reference" id="cite_ref-2">
        <a href="#cite_note-2">
         [2]
        </a>
       </sup>
      </p>
      <p>
       The word
       <i>
        trema
       </i>
       (plural:
       <i>
        tremas
       </i>
       or
       <i>
        tremata
       </i>
       ), used in French linguistics and also
       <a href="/wiki/Classics" title="Classics">
        classical scholarship
       </a>
       , is from the Greek
       <i>
        <span lang="grc-Latn" xml:lang="grc-Latn">
         trē̂ma
        </span>
       </i>
       (
       <span lang="grc" xml:lang="grc">
        τρῆμα
       </span>
       ) and means a "perforation", "orifice", or "pip" (as on
       <a href="/wiki/Dice" title="Dice">
        dice
       </a>
       ),
       <sup class="reference" id="cite_ref-3">
        <a href="#cite_note-3">
         [3]
        </a>
       </sup>
       thus describing the form of the diacritic rather than its function.
      </p>
      <p>
       <i>
        <a class="extiw" href="https://en.wiktionary.org/wiki/Umlaut" title="wikt:Umlaut">
         Umlaut
        </a>
       </i>
       is the German name of both the
       <a href="/wiki/Germanic_umlaut" title="Germanic umlaut">
        Germanic umlaut
       </a>
       , a sound-law also known as
       <i>
        i-mutation
       </i>
       , and the corresponding diacritic.
      </p>
      <h2>
       <span class="mw-headline" id="Diaeresis">
        Diaeresis
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=2" title="Edit section: Diaeresis">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       The diaeresis indicates that two adjoining letters that would normally form a
       <a href="/wiki/Digraph_(orthography)" title="Digraph (orthography)">
        digraph
       </a>
       and be pronounced as one are instead to be read as separate vowels in two syllables. The diaeresis indicates that a vowel should be pronounced apart from the letter that precedes it. For example, in the spelling
       <i>
        coöperate
       </i>
       , the diaeresis reminds the reader that the word has four syllables
       <i>
        co-op-er-ate
       </i>
       , not three,
       <i>
        *coop-er-ate
       </i>
       . In British English this usage has been considered obsolete for many years, and in US English, although it persisted for longer, it is now considered archaic as well.
       <sup class="reference" id="cite_ref-4">
        <a href="#cite_note-4">
         [4]
        </a>
       </sup>
       Nevertheless, it is still used by the US magazine
       <i>
        <a href="/wiki/The_New_Yorker" title="The New Yorker">
         The New Yorker
        </a>
       </i>
       .
       <sup class="reference" id="cite_ref-curse_5-0">
        <a href="#cite_note-curse-5">
         [5]
        </a>
       </sup>
       In English language texts it is perhaps most familiar in the spellings
       <i>
        <a class="mw-redirect" href="/wiki/Na%C3%AFve" title="Naïve">
         naïve
        </a>
       </i>
       ,
       <i>
        <a href="/wiki/Noel_(given_name)" title="Noel (given name)">
         Noël
        </a>
       </i>
       , and
       <i>
        <a class="mw-redirect" href="/wiki/Chlo%C3%AB" title="Chloë">
         Chloë
        </a>
       </i>
       , and is also used officially in the name of the island
       <a href="/wiki/Te%C3%A4n" title="Teän">
        Teän
       </a>
       . Languages such as
       <a href="/wiki/Dutch_language" title="Dutch language">
        Dutch
       </a>
       ,
       <a href="/wiki/Catalan_language" title="Catalan language">
        Catalan
       </a>
       ,
       <a href="/wiki/French_language" title="French language">
        French
       </a>
       ,
       <a href="/wiki/Galician_language" title="Galician language">
        Galician
       </a>
       and
       <a href="/wiki/Spanish_language" title="Spanish language">
        Spanish
       </a>
       make regular use of the diaeresis.
      </p>
      <h3>
       <span class="mw-headline" id="History">
        History
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=3" title="Edit section: History">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <h4>
       <span class="mw-headline" id="Greek_alphabet">
        Greek alphabet
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=4" title="Edit section: Greek alphabet">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <p>
       Two dots, called a
       <i>
        trema
       </i>
       , were used in the
       <a href="/wiki/Hellenistic_period" title="Hellenistic period">
        Hellenistic period
       </a>
       on the letters ι and υ, most often at the beginning of a word, as in ϊδων, ϋιος, and ϋβριν, to separate them from a preceding vowel, as writing was
       <i>
        <a href="/wiki/Scriptio_continua" title="Scriptio continua">
         scriptio continua
        </a>
       </i>
       , where spacing was not yet used as a
       <a href="/wiki/Word_divider" title="Word divider">
        word divider
       </a>
       . (See
       <a href="/wiki/Coptic_alphabet" title="Coptic alphabet">
        Coptic alphabet
       </a>
       , for example.) However, it was also used to indicate that a vowel formed its own syllable (in
       <a href="/wiki/Hiatus_(linguistics)" title="Hiatus (linguistics)">
        phonological hiatus
       </a>
       ), as in ηϋ and Αϊδι.
       <sup class="reference" id="cite_ref-6">
        <a href="#cite_note-6">
         [6]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-7">
        <a href="#cite_note-7">
         [7]
        </a>
       </sup>
      </p>
      <p>
       In
       <a href="/wiki/Modern_Greek" title="Modern Greek">
        Modern Greek
       </a>
       , αϊ and οϊ represent the diphthongs
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        /ai̯/
       </span>
       and
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        /oi̯/
       </span>
       , and εϊ the disyllabic sequence
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        /e.i/
       </span>
       , whereas αι, οι, and ει transcribe the simple vowels
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        /e/
       </span>
       ,
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        /i/
       </span>
       , and
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        /i/
       </span>
       . The diacritic can be the only one on a vowel, as in ακαδημαϊκός
       <i>
        akadēmaïkos
       </i>
       "academic", or in combination with an
       <a href="/wiki/Acute_accent" title="Acute accent">
        acute accent
       </a>
       , as in πρωτεΐνη
       <i>
        prōteïnē
       </i>
       "protein".
      </p>
      <h4>
       <span class="mw-headline" id="Latin_alphabet">
        Latin alphabet
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=5" title="Edit section: Latin alphabet">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <p>
       The diaeresis was borrowed
       <sup class="noprint Inline-Template" style="white-space:nowrap;">
        [
        <i>
         <a href="/wiki/Wikipedia:Manual_of_Style/Dates_and_numbers" title="Wikipedia:Manual of Style/Dates and numbers">
          <span title="Need the year this event took place (May 2015)">
           year needed
          </span>
         </a>
        </i>
        ]
       </sup>
       for this purpose in several languages of western and southern Europe, among them
       <a href="/wiki/Occitan_language" title="Occitan language">
        Occitan
       </a>
       ,
       <a href="/wiki/Catalan_language" title="Catalan language">
        Catalan
       </a>
       ,
       <a href="/wiki/French_language" title="French language">
        French
       </a>
       ,
       <a href="/wiki/Dutch_language" title="Dutch language">
        Dutch
       </a>
       ,
       <a href="/wiki/Welsh_language" title="Welsh language">
        Welsh
       </a>
       , and (rarely)
       <a href="/wiki/English_language" title="English language">
        English
       </a>
       .
      </p>
      <p>
       When a vowel in Greek was stressed, it did not assimilate to a preceding vowel but remained as a separate syllable. Such vowels were marked with an accent such as the
       <a href="/wiki/Acute_accent" title="Acute accent">
        acute
       </a>
       , a tradition that has also been adopted by other languages, such as
       <a href="/wiki/Spanish_language" title="Spanish language">
        Spanish
       </a>
       and
       <a href="/wiki/Portuguese_language" title="Portuguese language">
        Portuguese
       </a>
       . For example, the Portuguese words
       <i>
        saia
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ˈsajɐ]
       </span>
       "skirt" and the imperfect
       <i>
        saía
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [saˈi.ɐ]
       </span>
       "I used to leave" differ in that the sequence
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        /ai/
       </span>
       forms a diphthong in the former (
       <a href="/wiki/Synaeresis" title="Synaeresis">
        synaeresis
       </a>
       ), but is a
       <a href="/wiki/Hiatus_(linguistics)" title="Hiatus (linguistics)">
        hiatus
       </a>
       in the latter (diaeresis).
      </p>
      <h3>
       <span class="mw-headline" id="Hiatus">
        Hiatus
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=6" title="Edit section: Hiatus">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       In
       <a href="/wiki/Catalan_language" title="Catalan language">
        Catalan
       </a>
       , the digraphs
       <i>
        ai
       </i>
       ,
       <i>
        ei
       </i>
       ,
       <i>
        oi
       </i>
       ,
       <i>
        au
       </i>
       ,
       <i>
        eu
       </i>
       , and
       <i>
        iu
       </i>
       are normally read as
       <a href="/wiki/Diphthong" title="Diphthong">
        diphthongs
       </a>
       . To indicate exceptions to this rule (
       <a href="/wiki/Hiatus_(linguistics)" title="Hiatus (linguistics)">
        hiatus
       </a>
       ), a diaeresis mark is placed on the second vowel: without this the words
       <i>
        raïm
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [rəˈim]
       </span>
       ("grape") and
       <i>
        diürn
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [diˈurn]
       </span>
       ("diurnal") would be read *
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ˈrajm]
       </span>
       and *
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ˈdiwrn]
       </span>
       , respectively. The
       <a href="/wiki/Occitan_language" title="Occitan language">
        Occitan
       </a>
       use of diaeresis is very similar to that of Catalan:
       <i>
        ai, ei, oi, au, eu, ou
       </i>
       are
       <a href="/wiki/Diphthong" title="Diphthong">
        diphthongs
       </a>
       consisting of one syllable but
       <i>
        aï, eï, oï, aü, eü, oü
       </i>
       are groups consisting of two distinct syllables.
      </p>
      <p>
       In
       <a href="/wiki/Welsh_language" title="Welsh language">
        Welsh
       </a>
       , where the diaeresis appears, it is usually on the stressed vowel, and this is most often on the first of the two adjacent vowels; a typical example is
       <i>
        copïo
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [kɔ.ˈpi.ɔ]
       </span>
       (to copy), cf.
       <i>
        mopio
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ˈmɔ.pjɔ]
       </span>
       (to mop). It is also used on the first of two vowels that would otherwise form a diphthong (crëir
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ˈkreː.ɪr]
       </span>
       rather than creir
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ˈkrəi̯r]
       </span>
       ) and on the first of three vowels to separate it from a following diphthong: crëwyd is pronounced
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ˈkreː.ʊi̯d]
       </span>
       rather than
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ˈkrɛu̯.ɨd]
       </span>
       .
      </p>
      <p>
       In Dutch, spellings such as
       <i>
        coëfficiënt
       </i>
       are necessary because the digraphs
       <i>
        oe
       </i>
       and
       <i>
        ie
       </i>
       normally represent the simple vowels
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [u]
       </span>
       and
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [i]
       </span>
       , respectively. However, hyphenation is now preferred for compound words so that
       <i>
        zeeëend
       </i>
       (sea duck) is now spelled
       <i>
        zee-eend
       </i>
       .
       <sup class="reference" id="cite_ref-8">
        <a href="#cite_note-8">
         [8]
        </a>
       </sup>
      </p>
      <p>
       In German, diaeresis occurs in a few proper names, such as
       <a href="/wiki/Ferdinand_Pi%C3%ABch" title="Ferdinand Piëch">
        Ferdinand Piëch
       </a>
       and
       <a href="/wiki/Bernhard_Ho%C3%ABcker" title="Bernhard Hoëcker">
        Bernhard Hoëcker
       </a>
       .
      </p>
      <p>
       In
       <a href="/wiki/Galician_language" title="Galician language">
        Galician
       </a>
       , diaeresis is employed to indicate hiatus in the first and second persons of the plural of the
       <a class="mw-redirect" href="/wiki/Imperfect_tense" title="Imperfect tense">
        imperfect tense
       </a>
       of verbs ended in
       <i>
        -aer
       </i>
       ,
       <i>
        -oer
       </i>
       ,
       <i>
        -aír
       </i>
       and
       <i>
        -oír
       </i>
       (
       <i>
        saïamos
       </i>
       ,
       <i>
        caïades
       </i>
       ). This stems from the fact that an unstressed
       <i>
        -i-
       </i>
       is left between vowels, but constituting its own syllable, ending with a form identical in writing but different in pronunciation with those of the Present
       <a class="mw-redirect" href="/wiki/Subjunctive" title="Subjunctive">
        subjunctive
       </a>
       , as those have said
       <i>
        i
       </i>
       forming a diphthong with the following
       <i>
        a
       </i>
       .
      </p>
      <h3>
       <span class="mw-headline" id="Non-silent_vowels">
        Non-silent vowels
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=7" title="Edit section: Non-silent vowels">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       As a further extension, some languages began to use a diaeresis whenever a vowel letter was to be pronounced separately. This included vowels that would otherwise form digraphs with consonants or simply be silent.
      </p>
      <p>
       In the orthographies of
       <a href="/wiki/Spanish_language" title="Spanish language">
        Spanish
       </a>
       ,
       <a href="/wiki/Catalan_language" title="Catalan language">
        Catalan
       </a>
       ,
       <a href="/wiki/French_language" title="French language">
        French
       </a>
       ,
       <a href="/wiki/Galician_language" title="Galician language">
        Galician
       </a>
       , and
       <a href="/wiki/Occitan_language" title="Occitan language">
        Occitan
       </a>
       , the graphemes
       <i>
        gu
       </i>
       and
       <i>
        qu
       </i>
       normally represent a single sound,
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ɡ]
       </span>
       or
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [k]
       </span>
       , before the front vowels
       <i>
        e
       </i>
       and
       <i>
        i
       </i>
       (or before nearly all vowels in Occitan). In the few exceptions where the
       <i>
        u
       </i>
       is pronounced, a diaeresis is added to it. Before the
       <a href="/wiki/Portuguese_Language_Orthographic_Agreement_of_1990" title="Portuguese Language Orthographic Agreement of 1990">
        1990 Orthographic Agreement
       </a>
       , a diaeresis ("
       <i>
        trema
       </i>
       ") was also used in (mainly Brazilian)
       <a href="/wiki/Portuguese_language" title="Portuguese language">
        Portuguese
       </a>
       in this manner, in words like
       <i>
        sangüíneo
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        <a href="/wiki/Help:IPA/Portuguese" title="Help:IPA/Portuguese">
         [sɐ̃ˈɡwiniu]
        </a>
       </span>
       “sanguineous”; after the implementation of the Orthographic Agreement, it was abolished altogether from all Portuguese words. In French, in the aforementioned cases the diaeresis is usually written over the following vowel.
      </p>
      <p>
       Examples:
      </p>
      <ul>
       <li>
        Spanish
        <i>
         pingüino
        </i>
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         <a href="/wiki/Help:IPA/Spanish" title="Help:IPA/Spanish">
          [piŋˈɡwino]
         </a>
        </span>
        "penguin"
       </li>
       <li>
        Catalan
        <i>
         aigües
        </i>
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         <a href="/wiki/Help:IPA/Catalan" title="Help:IPA/Catalan">
          [ˈajɣwəs]
         </a>
        </span>
        "waters",
        <i>
         qüestió
        </i>
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         <a href="/wiki/Help:IPA/Catalan" title="Help:IPA/Catalan">
          [kwəstiˈo]
         </a>
        </span>
        "matter, question"
       </li>
       <li>
        Occitan
        <i>
         lingüista
        </i>
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         <a href="/wiki/Help:IPA/Occitan" title="Help:IPA/Occitan">
          [liŋˈɡwistɔ]
         </a>
        </span>
        "linguist",
        <i>
         aqüatic
        </i>
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         <a href="/wiki/Help:IPA/Occitan" title="Help:IPA/Occitan">
          [aˈkwatik]
         </a>
        </span>
        "aquatic"
       </li>
       <li>
        French
        <i>
         aiguë
        </i>
        or
        <i>
         aigüe
        </i>
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         <a href="/wiki/Help:IPA/French" title="Help:IPA/French">
          [eɡy]
         </a>
        </span>
        "acute (fem.)" (note that the
        <i>
         e
        </i>
        is silent; without the diacritic, both it and the
        <i>
         u
        </i>
        would be silent)
       </li>
       <li>
        Galician
        <i>
         mingüei
        </i>
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         <a href="/wiki/Help:IPA/Galician" title="Help:IPA/Galician">
          [miŋˈɡwei]
         </a>
        </span>
        "I shrank"
       </li>
       <li>
        <a class="mw-redirect" href="/wiki/Luxembourgish_language" title="Luxembourgish language">
         Luxembourgish
        </a>
        <i>
         chance
        </i>
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         <a href="/wiki/Help:IPA" title="Help:IPA">
          [ˈʃãːs]
         </a>
        </span>
        "opportunity",
        <i>
         chancë
        </i>
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         <a href="/wiki/Help:IPA" title="Help:IPA">
          [ˈʃãːsə]
         </a>
        </span>
        (before a consonant) "opportunities"
       </li>
       <li>
        English
        <i>
         Brontë
        </i>
        <span class="nowrap">
         <span class="IPA nopopups noexcerpt">
          <a href="/wiki/Help:IPA/English" title="Help:IPA/English">
           /
           <span style="border-bottom:1px dotted">
            <span title="/ˈ/: primary stress follows">
             ˈ
            </span>
            <span title="'b' in 'buy'">
             b
            </span>
            <span title="'r' in 'rye'">
             r
            </span>
            <span title="/ɒ/: 'o' in 'body'">
             ɒ
            </span>
            <span title="'n' in 'nigh'">
             n
            </span>
            <span title="'t' in 'tie'">
             t
            </span>
            <span title="/i/: 'y' in 'happy'">
             i
            </span>
           </span>
           /
          </a>
         </span>
        </span>
        (see
        <a href="/wiki/Bront%C3%AB_family" title="Brontë family">
         Brontë family
        </a>
        )
       </li>
      </ul>
      <p>
       This has been extended to
       <a href="/wiki/Luganda" title="Luganda">
        Ganda
       </a>
       , where a diaeresis separates
       <i>
        y
       </i>
       from
       <i>
        n
       </i>
       :
       <i>
        anya
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        <a href="/wiki/Help:IPA" title="Help:IPA">
         [aɲa]
        </a>
       </span>
       ,
       <i>
        anÿa
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        <a href="/wiki/Help:IPA" title="Help:IPA">
         [aɲja]
        </a>
       </span>
       .
      </p>
      <h3>
       <span class="mw-headline" id="French">
        French
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=8" title="Edit section: French">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       <span id="In_French">
       </span>
       In
       <a href="/wiki/French_language" title="French language">
        French
       </a>
       , some diphthongs that were written with pairs of vowel letters were later reduced to
       <a href="/wiki/Monophthong" title="Monophthong">
        monophthongs
       </a>
       , which led to an extension of the value of this diacritic. It often now indicates that the second vowel letter is to be pronounced separately from the first, rather than merge with it into a single sound. For example, the French words
       <i>
        maïs
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ma.is]
       </span>
       and
       <i>
        naïve
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [na.iv]
       </span>
       would be pronounced
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        *[mɛ]
       </span>
       and
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        *[nɛv]
       </span>
       , respectively, without the diaeresis mark, since the
       <a href="/wiki/Digraph_(orthography)" title="Digraph (orthography)">
        digraph
       </a>
       <i>
        ai
       </i>
       is pronounced
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ɛ]
       </span>
       . The English spelling of
       <i>
        Noël
       </i>
       "Christmas" (French
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [nɔ.ɛl]
       </span>
       ) comes from this use.
       <i>
        Ÿ
       </i>
       occurs in French as a variant of
       <i>
        ï
       </i>
       in a few proper nouns, as in the name of the
       <a href="/wiki/Paris" title="Paris">
        Parisian
       </a>
       suburb of
       <a href="/wiki/L%27Ha%C3%BF-les-Roses" title="L'Haÿ-les-Roses">
        L'Haÿ-les-Roses
       </a>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [la.i le ʁoz]
       </span>
       .
      </p>
      <p>
       The diaeresis is also used when a silent
       <i>
        e
       </i>
       is added to the sequence
       <i>
        gu
       </i>
       , to show that it is to be pronounced
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ɡy]
       </span>
       rather than as a digraph for
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ɡ]
       </span>
       . For example, when the feminine
       <i>
        -e
       </i>
       is added to
       <i>
        aigu
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [eɡy]
       </span>
       "sharp", the pronunciation does not change:
       <i>
        aiguë
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [eɡy]
       </span>
       . Similar is the feminine noun
       <i>
        ciguë
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [siɡy]
       </span>
       "hemlock"; compare
       <i>
        figue
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [fiɡ]
       </span>
       "fig". In the ongoing
       <a href="/wiki/Reforms_of_French_orthography" title="Reforms of French orthography">
        French spelling reform
       </a>
       of 1990, this was moved to the
       <i>
        u
       </i>
       (
       <i>
        aigüe, cigüe
       </i>
       ), though the earlier orthography continues to be widely used
       <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
        [
        <i>
         <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
          <span title="doubt over claim (May 2016)">
           citation needed
          </span>
         </a>
        </i>
        ]
       </sup>
       . (In
       <i>
        canoë
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [kanɔ.e]
       </span>
       the
       <i>
        e
       </i>
       is not silent, and so is not affected by the spelling reform.)
      </p>
      <p>
       In some names, a diaeresis is used to show what
       <i>
        used to be
       </i>
       two vowels in hiatus, although the second vowel has since fallen silent, as in
       <a href="/wiki/Camille_Saint-Sa%C3%ABns" title="Camille Saint-Saëns">
        Saint-Saëns
       </a>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [sɛ̃sɑ̃s]
       </span>
       and
       <a class="mw-redirect" href="/wiki/Anne_Louise_Germaine_de_Sta%C3%ABl" title="Anne Louise Germaine de Staël">
        de Staël
       </a>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [də stal]
       </span>
       .
      </p>
      <h3>
       <span class="mw-headline" id="English">
        English
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=9" title="Edit section: English">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       <span id="In_English">
       </span>
       The
       <a href="/wiki/Grave_accent" title="Grave accent">
        grave accent
       </a>
       and the diaeresis are the only
       <a href="/wiki/English_terms_with_diacritical_marks" title="English terms with diacritical marks">
        diacritics native to Modern English
       </a>
       (apart from diacritics used in
       <a href="/wiki/Loanword" title="Loanword">
        loanwords
       </a>
       , such as the
       <a href="/wiki/Acute_accent" title="Acute accent">
        acute accent
       </a>
       , the
       <a href="/wiki/Cedilla" title="Cedilla">
        cedilla
       </a>
       , or the
       <a href="/wiki/Tilde" title="Tilde">
        tilde
       </a>
       ). The use of both, however, is considered to be largely
       <a href="/wiki/Archaism" title="Archaism">
        archaic
       </a>
       .
       <sup class="reference" id="cite_ref-Fowler_9-0">
        <a href="#cite_note-Fowler-9">
         [9]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-10">
        <a href="#cite_note-10">
         [10]
        </a>
       </sup>
      </p>
      <p>
       The diaeresis mark is sometimes used in English personal first and last names to indicate that two adjacent vowels should be pronounced separately, rather than as a
       <a href="/wiki/Diphthong" title="Diphthong">
        diphthong
       </a>
       . Examples include the given names
       <i>
        Chloë
       </i>
       and
       <i>
        Zoë
       </i>
       , which otherwise might be pronounced with a silent
       <i>
        e
       </i>
       . To discourage a similar mispronunciation, the mark is also used in the surname
       <i>
        <a class="mw-redirect" href="/wiki/Bront%C3%AB" title="Brontë">
         Brontë
        </a>
       </i>
       . It may be used optionally for words that do not have a morphological break at the diaeresis point, such as
       <i>
        <a class="mw-redirect" href="/wiki/Na%C3%AFve" title="Naïve">
         naïve
        </a>
       </i>
       ,
       <i>
        <a href="/wiki/Bo%C3%B6tes" title="Boötes">
         Boötes
        </a>
       </i>
       , and
       <i>
        Noël
       </i>
       . However, it is far less commonly used in words such as
       <i>
        coöperate
       </i>
       and
       <i>
        reënter
       </i>
       except in a very few publications—notably
       <i>
        <a href="/wiki/The_New_Yorker" title="The New Yorker">
         The New Yorker
        </a>
       </i>
       .
       <sup class="reference" id="cite_ref-11">
        <a href="#cite_note-11">
         [11]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-12">
        <a href="#cite_note-12">
         [12]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-curse_5-1">
        <a href="#cite_note-curse-5">
         [5]
        </a>
       </sup>
      </p>
      <p>
       <i>
        Ÿ
       </i>
       is sometimes used in transcribed
       <a href="/wiki/Greek_language" title="Greek language">
        Greek
       </a>
       , where it represents the Greek letter
       <a class="mw-redirect" href="/wiki/%CE%A5" title="Υ">
        υ
       </a>
       (upsilon) in
       <a href="/wiki/Hiatus_(linguistics)" title="Hiatus (linguistics)">
        hiatus
       </a>
       with
       <a class="mw-redirect" href="/wiki/%CE%91" title="Α">
        α
       </a>
       . For example, it can be seen in the transcription
       <i>
        Artaÿctes
       </i>
       of the Persian name
       <span lang="grc" xml:lang="grc">
        Ἀρταΰκτης (
        <i>
         Artaüktēs
        </i>
        )
       </span>
       at the very end of
       <a href="/wiki/Herodotus" title="Herodotus">
        Herodotus
       </a>
       , or the name of
       <a href="/wiki/Taygetus" title="Taygetus">
        Mount Taÿgetus
       </a>
       on the southern Peloponnesus peninsula, which in modern Greek is spelled
       <span lang="el" xml:lang="el">
        Ταΰγετος
       </span>
       .
      </p>
      <h2>
       <span class="mw-headline" id="Umlaut">
        Umlaut
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=10" title="Edit section: Umlaut">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       <a href="/wiki/Germanic_umlaut" title="Germanic umlaut">
        Germanic umlaut
       </a>
       is a specific historical phenomenon of vowel-fronting in
       <a href="/wiki/German_language" title="German language">
        German
       </a>
       and other languages.
       <sup class="reference" id="cite_ref-13">
        <a href="#cite_note-13">
         [13]
        </a>
       </sup>
       In German it causes back vowels
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        /a/
       </span>
       ,
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        /o/
       </span>
       and
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        /u/
       </span>
       to shift forward in the mouth to
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        /e/
       </span>
       ,
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        /ø/
       </span>
       and
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        /y/
       </span>
       , respectively. In modern German orthography, the affected graphemes ⟨a⟩, ⟨o⟩ and ⟨u⟩ are written as ⟨
       <a href="/wiki/%C3%84" title="Ä">
        ä
       </a>
       ⟩, ⟨
       <a href="/wiki/%C3%96" title="Ö">
        ö
       </a>
       ⟩ and ⟨
       <a href="/wiki/%C3%9C" title="Ü">
        ü
       </a>
       ⟩, i.e. they are written with diacritical marks identical to the
       <i>
        diaeresis
       </i>
       mark. Therefore, in German (and in some languages influenced by German orthography), the diacritical symbol itself is sometimes called
       <i>
        umlaut
       </i>
       .
      </p>
      <h3>
       <span class="mw-headline" id="History_2">
        History
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=11" title="Edit section: History">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       <span id="Umlaut_history">
       </span>
      </p>
      <div class="hatnote navigation-not-searchable" role="note">
       Further information:
       <a href="/wiki/Germanic_umlaut" title="Germanic umlaut">
        Germanic umlaut
       </a>
      </div>

       <div class="thumbinner" style="width:102px;">

        </div>
       </div>
      </div>
      <div class="thumb tright">
       <div class="thumbinner" style="width:222px;">

        </a>

         <i>
          schoen

         <i>

         ('beautiful'). The Sütterlin script used here is a later development, however.
        </div>

      <p>
       German
       <a href="/wiki/Germanic_umlaut" title="Germanic umlaut">
        phonological umlaut
       </a>
       was present in the
       <a href="/wiki/Old_High_German" title="Old High German">
        Old High German
       </a>
       period and continued to develop in
       <a href="/wiki/Middle_High_German" title="Middle High German">
        Middle High German
       </a>
       . From the Middle High German period, it was sometimes denoted in written German by adding an
       <i>
        e
       </i>
       to the affected vowel, either after the vowel or, in small form, above it. This can still be seen in some names, e.g.
       <a class="mw-redirect" href="/wiki/Goethe" title="Goethe">
        Goethe
       </a>
       ,
       <a class="mw-redirect" href="/wiki/Goebbels" title="Goebbels">
        Goebbels
       </a>
       ,
       <a href="/wiki/Staedtler" title="Staedtler">
        Staedtler
       </a>
       .
       <sup class="reference" id="cite_ref-14">
        <a href="#cite_note-14">
         [14]
        </a>
       </sup>
      </p>
      <p>
       In
       <a href="/wiki/Blackletter" title="Blackletter">
        blackletter
       </a>
       handwriting as used in German manuscripts of the later Middle Ages, and also in many printed texts of the early modern period, the superscript ⟨e⟩ still had a form that would be recognisable to us as an ⟨e⟩, but in manuscript writing, umlauted vowels could be indicated by two dots since the late medieval period.
      </p>
      <p>
       In the
       <a href="/wiki/Kurrent" title="Kurrent">
        forms of handwriting
       </a>
       that emerged in the
       <a href="/wiki/Early_modern_period" title="Early modern period">
        early modern period
       </a>
       (of which
       <a href="/wiki/S%C3%BCtterlin" title="Sütterlin">
        Sütterlin
       </a>
       is the latest and best-known example) the letter ⟨e⟩ was composed of two short vertical lines very close together, and the superscript ⟨e⟩ looked like two tiny strokes. Even from the 16th century, the handwritten convention of indicating umlaut by two dots placed above the affected vowel is also found in printed texts.
      </p>
      <p>
       In modern handwriting, the umlaut sometimes resembles a
       <a href="/wiki/Tilde" title="Tilde">
        tilde
       </a>
       ,
       <a href="/wiki/Quotation_mark" title="Quotation mark">
        quotation mark
       </a>
       ,
       <a href="/wiki/Dash" title="Dash">
        dash
       </a>
       , miniature u or other small mark.
       <sup class="noprint Inline-Template" style="margin-left:0.1em; white-space:nowrap;">
        [
        <i>
         <a href="/wiki/Wikipedia:Please_clarify" title="Wikipedia:Please clarify">
          <span title="The text near this tag may need clarification or removal of jargon. (May 2015)">
           clarification needed
          </span>
         </a>
        </i>
        ]
       </sup>
      </p>
      <p>
       Unusual umlaut designs are sometimes also created for graphic design purposes, such as to fit an umlaut into tightly-spaced lines of text.
       <sup class="reference" id="cite_ref-Unusual_Umlauts_(German)_15-0">
        <a href="#cite_note-Unusual_Umlauts_(German)-15">
         [15]
        </a>
       </sup>
       This may include umlauts placed vertically or inside the body of the letter.
       <sup class="reference" id="cite_ref-16">
        <a href="#cite_note-16">
         [16]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-Flickr_collection:_vertical_umlauts_17-0">
        <a href="#cite_note-Flickr_collection:_vertical_umlauts-17">
         [17]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-Compact_umlaut_18-0">
        <a href="#cite_note-Compact_umlaut-18">
         [18]
        </a>
       </sup>
      </p>
      <h3>
       <span class="mw-headline" id="Printing_conventions_in_German">
        Printing conventions in German
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=12" title="Edit section: Printing conventions in German">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       When typing German, if umlaut letters are not available, it is usual to replace them with the underlying vowel followed by an ⟨e⟩. So, for example, "Schröder" becomes "Schroeder". As the pronunciation differs greatly between the normal letter and the umlaut, simply omitting the dots is incorrect. The result might often be a different word, as in
       <i>
        schon
       </i>
       'already',
       <i>
        schön
       </i>
       'beautiful'; or a different grammatic form, e.g.
       <i>
        Mutter
       </i>
       'mother',
       <i>
        Mütter
       </i>
       'mothers'.
      </p>
      <p>
       Despite this, the umlauted letters are not considered as separate letters of the alphabet proper in German, in contrast to other Germanic languages.
      </p>
      <p>
       When
       <a href="/wiki/Collation" title="Collation">
        alphabetically sorting
       </a>
       German words, the umlaut is usually not distinguished from the underlying vowel, although if two words differ only by an umlaut, the umlauted one comes second, for example:
      </p>
      <ol>
       <li>
        Schon
       </li>
       <li>
        Schön
       </li>
       <li>
        Schonen
       </li>
      </ol>
      <p>
       There is a second system in limited use, mostly for sorting names (colloquially called "telephone directory sorting")
       <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
        [
        <i>
         <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
          <span title='""mostly",'>
           citation needed
          </span>
         </a>
        </i>
        ]
       </sup>
       , which treats ü like ue, and so on.
      </p>
      <ol>
       <li>
        Schön
       </li>
       <li>
        Schon
       </li>
       <li>
        Schonen
       </li>
      </ol>
      <p>
       <a href="/wiki/Austria" title="Austria">
        Austrian
       </a>
       telephone directories insert ö after oz.
      </p>
      <ol>
       <li>
        Schon
       </li>
       <li>
        Schonen
       </li>
       <li>
        Schön
       </li>
      </ol>
      <p>
       In
       <a href="/wiki/Switzerland" title="Switzerland">
        Switzerland
       </a>
       , capital umlauts are sometimes printed as
       <a href="/wiki/Digraph_(orthography)" title="Digraph (orthography)">
        digraphs
       </a>
       , in other words, ⟨Ae⟩, ⟨Oe⟩, ⟨Ue⟩, instead of ⟨Ä⟩, ⟨Ö⟩, ⟨Ü⟩ (see
       <i>
        <a class="mw-redirect" href="/wiki/German_alphabet" title="German alphabet">
         German alphabet
        </a>
       </i>
       for an elaboration.) This is because the Swiss typewriter keyboard contains the French accents on the same keys as the umlauts (selected by Shift). To write capital umlauts the ¨-key is pressed followed by the capital letter to which the umlaut should apply.
      </p>
      <h3>
       <span class="mw-headline" id="Borrowing_of_German_umlaut_notation">
        Borrowing of German umlaut notation
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=13" title="Edit section: Borrowing of German umlaut notation">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       Some languages have borrowed some of the forms of the German letters
       <a href="/wiki/%C3%84" title="Ä">
        Ä
       </a>
       ,
       <a href="/wiki/%C3%96" title="Ö">
        Ö
       </a>
       , or
       <a href="/wiki/%C3%9C" title="Ü">
        Ü
       </a>
       , including
       <a href="/wiki/Azerbaijani_language" title="Azerbaijani language">
        Azerbaijani
       </a>
       ,
       <a href="/wiki/Estonian_language" title="Estonian language">
        Estonian
       </a>
       ,
       <a href="/wiki/Finnish_language" title="Finnish language">
        Finnish
       </a>
       ,
       <a href="/wiki/Hungarian_language" title="Hungarian language">
        Hungarian
       </a>
       ,
       <a href="/wiki/Karelian_language" title="Karelian language">
        Karelian
       </a>
       , some of the
       <a href="/wiki/Sami_languages" title="Sami languages">
        Sami languages
       </a>
       ,
       <a href="/wiki/Slovak_language" title="Slovak language">
        Slovak
       </a>
       ,
       <a href="/wiki/Swedish_language" title="Swedish language">
        Swedish
       </a>
       , and
       <a href="/wiki/Turkish_language" title="Turkish language">
        Turkish
       </a>
       . This indicates sounds similar to the corresponding umlauted letters in German. In spoken Scandinavian languages the grammatical umlaut change is used (singular to plural, derivations etc.) but the character used differs between languages. In Finnish, a/ä and o/ö change systematically in suffixes according to the rules of
       <a href="/wiki/Vowel_harmony" title="Vowel harmony">
        vowel harmony
       </a>
       . In Hungarian, where long vowels are indicated with an acute accent, the umlaut notation has been expanded with a version of the umlaut which looks like
       <a href="/wiki/Double_acute_accent" title="Double acute accent">
        double acute accents
       </a>
       , indicating a blend of umlaut and acute. Contrast: short ö; long ő. The Estonian alphabet has borrowed ⟨ä⟩, ⟨ö⟩, and ⟨ü⟩ from German; Swedish and Finnish have ⟨ä⟩ and ⟨ö⟩; and Slovak has ⟨ä⟩. In Estonian, Swedish, Finnish, and Sami ⟨ä⟩ and ⟨ö⟩ denote
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [æ]
       </span>
       and
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ø]
       </span>
       , respectively. Hungarian has ⟨ö⟩ and ⟨ü⟩. The Slovak language uses the letter ⟨ä⟩ to denote
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ɛ]
       </span>
       (or a bit archaic but still correct
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [æ]
       </span>
       ). The sign is called
       <i>
        dve bodky
       </i>
       ("two dots"), and the full name of the letter
       <i>
        ä
       </i>
       is
       <i>
        a s dvomi bodkami
       </i>
       ("a with two dots"). The similar word
       <i>
        dvojbodka
       </i>
       ("double dot") however refers to the colon. In these languages, with the exception of Hungarian, the replacement rule for situations where the umlaut character is not available, is to simply use the underlying unaccented character instead. Hungarian follows the German rules and replaces ⟨ö⟩ and ⟨ü⟩ with ⟨oe⟩ and ⟨ue⟩ respectively
       <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
        [
        <i>
         <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
          <span title="This claim needs references to reliable sources. (June 2017)">
           citation needed
          </span>
         </a>
        </i>
        ]
       </sup>
       – at least for telegrams and telex messages. The same rule is followed for the near-lookalikes ⟨ő⟩ and ⟨ű⟩.
      </p>
      <p>
       In
       <a class="mw-redirect" href="/wiki/Luxembourgish_language" title="Luxembourgish language">
        Luxembourgish
       </a>
       (
       <i>
        Lëtzebuergesch
       </i>
       ), the umlaut diacritic in ⟨ä⟩ and ⟨ë⟩ represents a stressed
       <a href="/wiki/Schwa" title="Schwa">
        schwa
       </a>
       . The letters ⟨ü⟩ and ⟨ö⟩ do not occur in native Luxembourgish words, but at least the former is common in words borrowed from standard German.
      </p>
      <p>
       When Turkish switched from the Arabic to the Latin alphabet in 1928, it adopted a number of diacritics borrowed from various languages, including ⟨ü⟩ and ⟨ö⟩ from German (probably reinforced by their use in languages like Swedish, Hungarian, etc.). These Turkish graphemes represent sounds similar to their respective values in German (see
       <i>
        <a href="/wiki/Turkish_alphabet" title="Turkish alphabet">
         Turkish alphabet
        </a>
       </i>
       ).
      </p>
      <p>
       As the borrowed diacritic has lost its relationship to Germanic i-mutation, they are in some languages considered independent graphemes, and cannot be replaced with ⟨ae⟩, ⟨oe⟩, or ⟨ue⟩ as in German. In Estonian and Finnish, for example, these latter diphthongs have independent meanings. Even some Germanic languages, such as Swedish (which
       <i>
        does
       </i>
       have a transformation analogous to the German umlaut, called
       <i>
        omljud
       </i>
       ), treat them always as independent letters. In
       <a href="/wiki/Collation" title="Collation">
        collation
       </a>
       , this means they have their own positions in the alphabet, for example at the end ("A–Ö" or "A–Ü", not "A–Z") as in Swedish, Estonian and Finnish, which means that the dictionary order is different from German. The transformations
       <i>
        ä
       </i>
       →
       <i>
        ae
       </i>
       and
       <i>
        ö
       </i>
       →
       <i>
        oe
       </i>
       can, therefore, be considered less appropriate for these languages, although Swedish and Finnish passports use the transformation to render ö and ä (and
       <i>
        å
       </i>
       as
       <i>
        aa
       </i>
       ) in the
       <a href="/wiki/Machine-readable_passport" title="Machine-readable passport">
        machine-readable zone
       </a>
       . In contexts of technological limitation, e.g. in English based systems, Swedes can either be forced to omit the diacritics or use the two letter system.
      </p>
      <p>
       When typing in
       <a href="/wiki/Norwegian_language" title="Norwegian language">
        Norwegian
       </a>
       , the letters
       <a href="/wiki/%C3%86" title="Æ">
        Æ
       </a>
       and
       <a href="/wiki/%C3%98" title="Ø">
        Ø
       </a>
       might be replaced with
       <a href="/wiki/%C3%84" title="Ä">
        Ä
       </a>
       and
       <a href="/wiki/%C3%96" title="Ö">
        Ö
       </a>
       respectively if the former are not available. If
       <i>
        ä
       </i>
       is not available either, it is appropriate to use
       <i>
        ae
       </i>
       . The same goes for
       <i>
        ö
       </i>
       and
       <i>
        oe
       </i>
       . While
       <i>
        ae
       </i>
       has a great resemblance to the letter
       <i>
        æ
       </i>
       and, therefore, does not impede legibility, the digraph
       <i>
        oe
       </i>
       is likely to reduce the legibility of a Norwegian text. This especially applies to the digraph
       <i>
        <a class="mw-redirect" href="/wiki/%C3%98y" title="Øy">
         øy
        </a>
       </i>
       , which would be rendered in the more cryptic form
       <i>
        oey
       </i>
       . Also in
       <a href="/wiki/Danish_language" title="Danish language">
        Danish
       </a>
       , Ö has been used in place of Ø in some older texts and to distinguish between open and closed ö-sounds and when confusion with other symbols could occur, e.g. on maps. The Danish/Norwegian Ø is like the German Ö a development of OE, to be compared with the French
       <a href="/wiki/%C5%92" title="Œ">
        Œ
       </a>
       .
      </p>
      <p>
       Early
       <a href="/wiki/Volap%C3%BCk" title="Volapük">
        Volapük
       </a>
       used
       <a class="mw-redirect" href="/wiki/Fraktur_(typeface_sub-classification)" title="Fraktur (typeface sub-classification)">
        Fraktur
       </a>
       <i>
        a
       </i>
       ,
       <i>
        o
       </i>
       and
       <i>
        u
       </i>
       as different from
       <a href="/wiki/Antiqua_(typeface_class)" title="Antiqua (typeface class)">
        Antiqua
       </a>
       ones. Later, the Fraktur forms were replaced with umlauted vowels.
      </p>
      <p>
       The usage of umlaut-like diacritic vowels, particularly
       <i>
        ü
       </i>
       , occurs in the transcription of languages that do not use the Roman alphabet, such as
       <a href="/wiki/Chinese_language" title="Chinese language">
        Chinese
       </a>
       . For example, 女 (
       <i>
        female
       </i>
       ) is transcribed as
       <i>
        nǚ
       </i>
       in proper
       <a class="mw-redirect" href="/wiki/Standard_Mandarin" title="Standard Mandarin">
        Mandarin Chinese
       </a>
       <a href="/wiki/Pinyin" title="Pinyin">
        pinyin
       </a>
       , while
       <i>
        nv
       </i>
       is sometimes used as a replacement for convenience since the letter v is not used in pinyin.
       <a href="/wiki/Tibetan_pinyin" title="Tibetan pinyin">
        Tibetan pinyin
       </a>
       uses
       <i>
        ä, ö, ü
       </i>
       with approximately their German values.
      </p>
      <p>
       The
       <a href="/wiki/Cyrillic_script" title="Cyrillic script">
        Cyrillic
       </a>
       letters
       <a class="mw-redirect" href="/wiki/%D3%92" title="Ӓ">
        ӓ
       </a>
       ,
       <a class="mw-redirect" href="/wiki/%D3%A6" title="Ӧ">
        ӧ
       </a>
       ,
       <a class="mw-redirect" href="/wiki/%D3%B0" title="Ӱ">
        ӱ
       </a>
       are used in
       <a href="/wiki/Mari_language" title="Mari language">
        Mari
       </a>
       ,
       <a href="/wiki/Khanty_language" title="Khanty language">
        Khanty
       </a>
       , and other languages for approximately
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [æ]
       </span>
       ,
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ø]
       </span>
       , and
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [y]
       </span>
       . These directly parallel the German umlaut
       <i>
        ä, ö, ü.
       </i>
       Other vowels using a double dot to modify their values in various minority languages of Russia are
       <a class="mw-redirect" href="/wiki/%D3%9A" title="Ӛ">
        ӛ
       </a>
       ,
       <a class="mw-redirect" href="/wiki/%D3%AA" title="Ӫ">
        ӫ
       </a>
       , and
       <a class="mw-redirect" href="/wiki/%D3%B8" title="Ӹ">
        ӹ
       </a>
       .
      </p>

       <span class="mw-editsection">
        <span class="mw-editsection-bracket">

        </span>

      <h2>
       <span class="mw-headline" id="Other_uses">
        Other uses
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=15" title="Edit section: Other uses">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       A double dot is also used as a diacritic in cases where it functions as neither a diaeresis nor an umlaut. In the
       <a href="/wiki/International_Phonetic_Alphabet" title="International Phonetic Alphabet">
        International Phonetic Alphabet
       </a>
       (IPA), a double dot is used for a
       <a class="mw-redirect" href="/wiki/Centralization_(phonetics)" title="Centralization (phonetics)">
        centralized vowel
       </a>
       , a situation more similar to umlaut than to diaeresis. In other languages it is used for vowel length, nasalization, tone, and various other uses where diaeresis or umlaut was available typographically. The IPA uses a double dot below letters to indicate a breathy-voice or
       <a href="/wiki/Murmured_voice" title="Murmured voice">
        murmur
       </a>
       .
      </p>

        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">

      <h3>
       <span class="mw-headline" id="Consonants">
        Consonants
       </span>
       <span class="mw-editsection">

         ]
        </span>
       </span>
      </h3>
      <p>
       <a class="mw-redirect" href="/wiki/Jacaltec" title="Jacaltec">
        Jacaltec
       </a>
       (a
       <a href="/wiki/Mayan_languages" title="Mayan languages">
        Mayan
       </a>
       language) and
       <a href="/wiki/Malagasy_language" title="Malagasy language">
        Malagasy
       </a>
       are among the very few languages with a diaeresis on the letter "n"; in both,
       <a class="mw-redirect" href="/wiki/N%CC%88" title="N̈">
        n̈
       </a>
       is the
       <a href="/wiki/Velar_nasal" title="Velar nasal">
        velar nasal
       </a>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ŋ]
       </span>
       .
      </p>
      <p>
       In
       <a href="/wiki/Udmurt_language" title="Udmurt language">
        Udmurt
       </a>
       , a double dot is also used with the consonant letters
       <a class="mw-redirect" href="/wiki/%D3%9C" title="Ӝ">
        ӝ
       </a>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [dʒ]
       </span>
       (from ж
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ʒ]
       </span>
       ),
       <a class="mw-redirect" href="/wiki/%D3%9E" title="Ӟ">
        ӟ
       </a>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [dʑ]
       </span>
       (from з
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [z] ~ [ʑ]
       </span>
       ) and
       <a class="mw-redirect" href="/wiki/%D3%B4" title="Ӵ">
        ӵ
       </a>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [tʃ]
       </span>

       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [tɕ]
       </span>
       ).
      </p>
      <p>
       <a class="mw-redirect" href="/wiki/%E1%B8%A6" title="Ḧ">
        Ḧ
       </a>
       and
       <a class="mw-redirect" href="/wiki/%E1%BA%8C" title="Ẍ">
        ẍ
       </a>

       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">

        Kurdish alphabet
       </a>
       . These are foreign sounds borrowed from Arabic.
      </p>
      <p>
       <a class="mw-redirect" href="/wiki/%E1%BA%84" title="Ẅ">
        Ẅ
       </a>
       and
       <a class="mw-redirect" href="/wiki/%C5%B8" title="Ÿ">
        ÿ
       </a>
       :
       <i>
        Ÿ
       </i>
       is generally a vowel, but it is used as the (semi-vowel) consonant
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ɰ]
       </span>
       (a
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [w]
       </span>
       without the use of the lips) in
       <a href="/wiki/Tlingit_language" title="Tlingit language">
        Tlingit
       </a>
       . This sound is also found in
       <a class="mw-redirect" href="/wiki/Coast_Tsimshian" title="Coast Tsimshian">
        Coast Tsimshian
       </a>
       , where it is written
       <a class="mw-redirect" href="/wiki/%E1%BA%84" title="Ẅ">
        ẅ
       </a>
       .
      </p>
      <p>
       A number of languages in
       <a href="/wiki/Vanuatu" title="Vanuatu">
        Vanuatu
       </a>
       use double dots on consonants, to represent
       <a href="/wiki/Linguolabial_consonant" title="Linguolabial consonant">
        linguolabial
       </a>
       (or
       <i>
        apicolabial
       </i>
       ) phonemes in their orthography. Thus
       <a href="/wiki/Araki_language#Consonants" title="Araki language">
        Araki
       </a>
       contrasts bilabial
       <i>
        p
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [p]
       </span>
       with linguolabial
       <i>
        p̈
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [t̼]
       </span>
       ; bilabial
       <i>
        m
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [m]
       </span>
       with linguolabial
       <i>
        m̈
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [n̼]
       </span>
       ; and bilabial
       <i>
        v
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [β]
       </span>
       with linguolabial
       <i>
        v̈
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ð̼]
       </span>
       .
      </p>
      <p>
       Seneca uses ⟨s̈⟩ for
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        [ʃ]
       </span>
       .
      </p>
      <p>
       The letter
       <a class="mw-redirect" href="/wiki/%E1%BA%97" title="ẗ">
        ẗ
       </a>
       is used in the
       <a href="/wiki/ISO_233" title="ISO 233">
        ISO 233
       </a>
       transliteration of Arabic for
       <a class="mw-redirect" href="/wiki/T%C4%81%CA%BE_marb%C5%AB%E1%B9%ADa" title="Tāʾ marbūṭa">
        tāʾ marbūṭa
       </a>
       ة.
      </p>
      <p>
       <a href="/wiki/Syriac_language" title="Syriac language">
        Syriac
       </a>
       uses a two dots above a letter, called
       <a class="new" href="/w/index.php?title=Siyame&amp;action=edit&amp;redlink=1" title="Siyame (page does not exist)">
        Siyame
       </a>
       , to indicate that the word should be understood as plural. For instance, ܒܝܬܐ (
       <i>
        bayta
       </i>
       ) means 'house', while ܒܝ̈ܬܐ (
       <i>
        bayte
       </i>
       ) means 'houses'. The sign is used especially when no vowel marks are present, which could differentiate between the two forms. Although the origin of the
       <a class="new" href="/w/index.php?title=Siyame&amp;action=edit&amp;redlink=1" title="Siyame (page does not exist)">
        Siyame
       </a>
       is different from that of the Diaeresis sign, in modern computer systems both are represented by the same Unicode character. This, however, often leads to wrong rendering of the Syriac text.
      </p>

      <p>
       If letters with double dots are not present on the keyboard (or if they are not recognized by the operating system), there are a number of ways to input them into a computer system.
      </p>
      <p>
       Using
       <a href="/wiki/Microsoft_Word" title="Microsoft Word">
        Microsoft Word
       </a>
       for
       <a class="mw-redirect" href="/wiki/Windows" title="Windows">
        Windows
       </a>
       , a letter with double dots can be produced by pressing
       <i>
        Ctrl-Shift-:
       </i>
       , then the letter.
      </p>
      <p>
       Using
       <a href="/wiki/Linux" title="Linux">
        Linux
       </a>
       , a letter with double dots can be produced by pressing
       <i>
        AltGr-Shift-:
       </i>
       , then the letter
      </p>
      <p>
       Using
       <a class="mw-redirect" href="/wiki/Mac_OS" title="Mac OS">
        Mac OS
       </a>
       or
       <a class="mw-redirect" href="/wiki/OS_X" title="OS X">
        OS X
       </a>
       , a letter with double dots can be produced by pressing
       <kbd class="keyboard-key nowrap" style="border: 1px solid #aaa; -moz-border-radius: 0.2em; -webkit-border-radius: 0.2em; border-radius: 0.2em; -moz-box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); -webkit-box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); background-color: #f9f9f9; background-image: -moz-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -o-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -webkit-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: linear-gradient(to bottom, #eee, #f9f9f9, #eee); padding: 0.1em 0.3em; font-family: inherit; font-size: 0.85em;">
        ⌥ Option
       </kbd>
       +
       <kbd class="keyboard-key nowrap" style="border: 1px solid #aaa; -moz-border-radius: 0.2em; -webkit-border-radius: 0.2em; border-radius: 0.2em; -moz-box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); -webkit-box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); background-color: #f9f9f9; background-image: -moz-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -o-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -webkit-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: linear-gradient(to bottom, #eee, #f9f9f9, #eee); padding: 0.1em 0.3em; font-family: inherit; font-size: 0.85em;">
        U
       </kbd>
       , then the letter. This works on English and other keyboards and is documented further in the supplied manuals.
      </p>

        X-based systems
       </a>
       with a
       <a href="/wiki/Compose_key" title="Compose key">
        Compose key
       </a>
       set in the system can usually insert characters with double dots by typing
       <kbd class="keyboard-key nowrap" style="border: 1px solid #aaa; -moz-border-radius: 0.2em; -webkit-border-radius: 0.2em; border-radius: 0.2em; -moz-box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); -webkit-box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); background-color: #f9f9f9; background-image: -moz-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -o-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -webkit-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: linear-gradient(to bottom, #eee, #f9f9f9, #eee); padding: 0.1em 0.3em; font-family: inherit; font-size: 0.85em;">
        Compose
       </kbd>
       ,
       <kbd class="keyboard-key nowrap" style="border: 1px solid #aaa; -moz-border-radius: 0.2em; -webkit-border-radius: 0.2em; border-radius: 0.2em; -moz-box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); -webkit-box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); box-shadow: 0.1em 0.1em 0.2em rgba(0,0,0,0.1); background-color: #f9f9f9; background-image: -moz-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -o-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: -webkit-linear-gradient(top, #eee, #f9f9f9, #eee); background-image: linear-gradient(to bottom, #eee, #f9f9f9, #eee); padding: 0.1em 0.3em; font-family: inherit; font-size: 0.85em;">

       </kbd>
       (i.e.
       <tt>
        "
       </tt>
       ) followed by the letter.

      </p>
      <p>
       <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">

      </p>
      <p>

      <h3>
       <span class="mw-headline" id="On-screen_keyboards">
        On-screen keyboards
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=20" title="Edit section: On-screen keyboards">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       The early 21st century has seen noticeable growth in stylus- and touch-operated interfaces, making the use of on-screen keyboards operated by pointing devices (mouse, stylus, or finger) more important. These "soft" keyboards may replicate the modifier keys found on hardware keyboards, but they may also employ other means of selecting options from a base key, such as right-click or press-and-hold. Soft keyboards may also have multiple contexts, such as letter, numeric, and symbol.
      </p>
      <h4>
       <span class="mw-headline" id="Windows">
        Windows
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=21" title="Edit section: Windows">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <p>
       In Windows 8, the standard (but not extended) touch US keyboard allows entry of
       <i>
        ä ë ï ö ü ẅ ÿ
       </i>
       by holding the corresponding un-accented keys, and of spacing ¨ by holding ~ or ^. Upper-case equivalents are accessed with the Shift key, as would be expected.
      </p>
      <h4>
       <span class="mw-headline" id="iOS">
        iOS
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=22" title="Edit section: iOS">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <p>
       iOS provides accented letters through press-and-hold on most European Latin-script keyboards, including English. Some keyboard layouts feature combining-accent keys that can add accents to any appropriate letter.
      </p>
      <h4>
       <span class="mw-headline" id="Dedicated_keys">
        Dedicated keys
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=23" title="Edit section: Dedicated keys">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <p>
       The German keyboard has dedicated keys for
       <i>
        ü ö ä
       </i>
       . Scandinavian and Turkish keyboards have dedicated keys for their respective language-specific letters, including
       <i>
        ö
       </i>
       for Swedish, Finnish, and Icelandic, and both
       <i>
        ö
       </i>
       and
       <i>
        ü
       </i>
       for Turkish.
      </p>
      <h4>
       <span class="mw-headline" id="Other_scripts">
        Other scripts
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=24" title="Edit section: Other scripts">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <p>
       For non-Latin scripts, Greek and Russian use press-and-hold for double-dot diacritics on only a few characters. The Greek keyboard has dialytica and dialytica–tonos variants for upsilon and iota (ϋ ΰ ϊ ΐ), but not for ε ο α η ω, following modern monotonic usage. Russian keyboards feature separate keys for е and ё.
      </p>
      <h3>
       <span class="mw-headline" id="Character_encodings">
        Character encodings
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=25" title="Edit section: Character encodings">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       The
       <a class="mw-redirect" href="/wiki/ISO_8859-1" title="ISO 8859-1">
        ISO 8859-1
       </a>
       character encoding includes the letters
       <i>
        ä
       </i>
       ,
       <i>
        ë
       </i>
       ,
       <i>
        ï
       </i>
       ,
       <i>
        ö
       </i>
       ,
       <i>
        ü
       </i>
       , and their respective
       <a class="mw-redirect" href="/wiki/Majuscule" title="Majuscule">
        capital
       </a>
       forms, as well as
       <i>
        ÿ
       </i>
       in
       <a class="mw-redirect" href="/wiki/Lower_case" title="Lower case">
        lower case
       </a>
       only, with
       <i>
        Ÿ
       </i>
       added in the revised edition
       <a class="mw-redirect" href="/wiki/ISO_8859-15" title="ISO 8859-15">
        ISO 8859-15
       </a>
       and
       <a href="/wiki/Windows-1252" title="Windows-1252">
        Windows-1252
       </a>
       .
      </p>
      <p>
       <a href="/wiki/Unicode" title="Unicode">
        Unicode
       </a>
       includes all of ISO-8859 and also provides the double dot as
       <span class="nowrap">
        <span class="monospaced" style="font-family: monospace, monospace;">
         U+00A8
        </span>
       </span>
       <span style="font-size:125%">
        ¨
       </span>
       <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">
        Diaeresis
       </span>
       and as
       <span class="nowrap">
        <span class="monospaced" style="font-family: monospace, monospace;">
         U+0308
        </span>
       </span>
       <span style="font-size:125%">
        ◌̈
       </span>
       <a href="/wiki/Combining_character" title="Combining character">
        <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">
         Combining diaeresis
        </span>
       </a>
       . Mainly for compatibility with older character encodings, dozens of codepoints with letters with double dots are available.
      </p>
      <p>
       Both the combining character U+0308 and the precombined codepoints can be used as umlaut or diaeresis.
      </p>
      <p>
       Sometimes, there's a need to distinguish between the umlaut sign and the diaeresis sign. ISO/IEC JTC 1/SC 2/WG 2 recommends the following for these cases:
      </p>
      <ul>
       <li>
        To represent the
        <b>
         umlaut
        </b>
        use Combining Diaeresis (U+0308)
       </li>
       <li>
        To represent the
        <b>
         diaeresis
        </b>
        use Combining Grapheme Joiner (CGJ, U+034F) + Combining Diaeresis (U+0308)
       </li>
      </ul>
      <p>
       As of version 3.2.0, Unicode also provides
       <span class="nowrap">
        <span class="monospaced" style="font-family: monospace, monospace;">
         U+0364
        </span>
       </span>
       <span style="font-size:125%">
        ◌ͤ
       </span>
       <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">
        COMBINING LATIN SMALL LETTER E
       </span>
       which can produce the older umlaut typography.
      </p>
      <p>
       Unicode provides a combining double dot below as
       <span class="nowrap">
        <span class="monospaced" style="font-family: monospace, monospace;">
         U+0324
        </span>
       </span>
       <span style="font-size:125%">
        ◌̤
       </span>
       <span class="smallcaps" style="font-variant: small-caps; font-size: smaller;">
        Combining diaeresis below
       </span>
       .
      </p>
      <h3>
       <span class="mw-headline" id="HTML">
        HTML
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=26" title="Edit section: HTML">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       In
       <a href="/wiki/HTML" title="HTML">
        HTML
       </a>
       , vowels with double dots can be entered with an entity reference of the form
       <code>
        &amp;?uml;
       </code>
       , where
       <code>
        ?
       </code>
       can be any of
       <code>
        a
       </code>
       ,
       <code>
        e
       </code>
       ,
       <code>
        i
       </code>
       ,
       <code>
        o
       </code>
       ,
       <code>
        u
       </code>
       ,
       <code>
        y
       </code>
       or their
       <a class="mw-redirect" href="/wiki/Majuscule" title="Majuscule">
        majuscule
       </a>
       counterparts. With the exception of the uppercase
       <i>
        Ÿ
       </i>
       , these characters are also available in all of the
       <a class="mw-redirect" href="/wiki/ISO_8859" title="ISO 8859">
        ISO 8859
       </a>
       character sets and thus have the same codepoints in
       <a class="mw-redirect" href="/wiki/ISO-8859-1" title="ISO-8859-1">
        ISO-8859-1
       </a>
       (
       <a class="mw-redirect" href="/wiki/ISO_8859-2" title="ISO 8859-2">
        -2
       </a>
       ,
       <a class="mw-redirect" href="/wiki/ISO_8859-3" title="ISO 8859-3">
        -3
       </a>
       ,
       <a class="mw-redirect" href="/wiki/ISO_8859-4" title="ISO 8859-4">

       </a>
       ,
       <a class="mw-redirect" href="/wiki/ISO_8859-9" title="ISO 8859-9">

       </a>

       <a class="mw-redirect" href="/wiki/ISO_8859-10" title="ISO 8859-10">

       <a class="mw-redirect" href="/wiki/ISO_8859-13" title="ISO 8859-13">

       <a class="mw-redirect" href="/wiki/ISO_8859-14" title="ISO 8859-14">

       <a href="/wiki/Unicode" title="Unicode">

      <h3>
       <span class="mw-headline" id="TeX_and_LaTeX">
        TeX and LaTeX
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Diaeresis_(diacritic)&amp;action=edit&amp;section=27" title="Edit section: TeX and LaTeX">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       <a href="/wiki/TeX" title="TeX">
        TeX
       </a>
       (and its derivatives, most notably
       <a href="/wiki/LaTeX" title="LaTeX">
        LaTeX
       </a>
       ) also allows double dots to be placed over letters. The standard way is to use the control sequence
       <tt>
        \"
       </tt>
       followed by the relevant letter, e.g.
       <tt>
        \"u
       </tt>
       . It is good practice to set the sequence off with curly braces:
       <tt>
        {\"u}
       </tt>
       or
       <tt>
        \"{u}
       </tt>
       .
      </p>
      <p>
       <a href="/wiki/TeX" title="TeX">
        TeX
       </a>
       's "German" package can be used: it adds the
       <tt>
        "
       </tt>
       control sequence (without the backslash) to produce the Umlaut. However, this can cause conflicts if the main language of the document is not German. Since the integration of Unicode through the development of
       <a href="/wiki/XeTeX" title="XeTeX">
        XeTeX
       </a>
       and
       <a class="mw-redirect" href="/wiki/XeLaTeX" title="XeLaTeX">
        XeLaTeX
       </a>
       , it is also possible to input the Unicode character directly into the document, using one of the recognized methods such as
       <a href="/wiki/Compose_key" title="Compose key">
        Compose key
       </a>
       or direct
       <a href="/wiki/Unicode_input" title="Unicode input">
        Unicode input
       </a>
       .
      </p>
      <p>
       <a href="/wiki/TeX" title="TeX">
        TeX
       </a>
       's traditional control sequences can still be used and will produce the same output (in very early versions of
       <a href="/wiki/TeX" title="TeX">
        TeX
       </a>
       these sequences would produce double dots that were too far above the letter's body).
      </p>
      <p>
       All these methods can be used with all available font variations (italic, bold etc.).
      </p>
