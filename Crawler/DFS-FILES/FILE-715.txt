      <p>
       <i>
        <b>
         Leísmo
        </b>
       </i>
       ("using
       <i>
        le
       </i>
       ") is a dialectal variation in the
       <a href="/wiki/Spanish_language" title="Spanish language">
        Spanish language
       </a>
       that occurs largely in Spain. It involves using the
       <a href="/wiki/Object_(grammar)" title="Object (grammar)">
        indirect object
       </a>
       <a href="/wiki/Pronoun" title="Pronoun">
        pronoun
       </a>
       <i>
        le
       </i>
       in place of the (standard) masculine
       <a href="/wiki/Object_(grammar)" title="Object (grammar)">
        direct object
       </a>
       <a href="/wiki/Pronoun" title="Pronoun">
        pronoun
       </a>
       <i>
        lo
       </i>
       , especially when the direct object refers to a male person.
      </p>
      <p>
       Leísmo with animate objects is both common and
       <a href="/wiki/Linguistic_prescription" title="Linguistic prescription">
        prescriptively
       </a>
       accepted in many dialects spoken in
       <a href="/wiki/Spain" title="Spain">
        Spain
       </a>
       , but uncommon in most others. Leísmo is always rejected in
       <a href="/wiki/Linguistic_prescription" title="Linguistic prescription">
        linguistic prescription
       </a>
       when the direct object to which it refers is not an animate object.
      </p>
      <p>
       Examples:
      </p>
      <dl>
       <dd>
        <i>
         Veo al chico
        </i>
        ("I see the boy") →
        <i>
         Lo veo
        </i>
        (standard Spanish, with
        <i>
         lo
        </i>
        )
       </dd>
       <dd>
        <i>
         Veo al chico
        </i>
        ("I see the boy") →
        <i>
         Le veo
        </i>
        (leísmo, common in Spain; other regions prefer
        <i>
         lo veo
        </i>
        )
       </dd>
       <dd>
        <i>
         Veo el árbol
        </i>
        ("I see the tree") →
        <i>
         Le veo
        </i>
        (not accepted in linguistic prescription — the tree is not a person)
       </dd>
      </dl>
      <p>
       The use of
       <i>
        le
       </i>
       in dialects where leísmo is common typically correlates with the use of the preposition
       <i>
        a
       </i>
       for animate direct objects. That is, if a dialect features leísmo,
       <i>
        le
       </i>
       replaces masculine direct objects that would have been preceded by
       <i>
        a
       </i>
       if expressed in full. (For this "personal a", see
       <a href="/wiki/Spanish_prepositions#a" title="Spanish prepositions">
        Spanish prepositions
       </a>
       .)
      </p>
      <p>
       <i>
        Le
       </i>
       is properly speaking the
       <a href="/wiki/Epicene" title="Epicene">
        epicene
       </a>
       indirect object pronoun, used for both masculine and feminine referents, whether animate or inanimate. In certain dialects it can be replaced by
       <i>
        lo
       </i>
       or
       <i>
        la
       </i>
       , i.e. —
       <i>
        dila que la quiero
       </i>
       – but this usage is restricted to informal speech.
      </p>
      <dl>
       <dd>
        <i>
         Le voy a dar un regalo
        </i>
        ("I am giving him/her a present")
       </dd>
       <dd>
        <i>
         La voy a dar un regalo
        </i>
        ("I am giving her a present", in some dialects; see
        <a class="mw-redirect" href="/wiki/La%C3%ADsmo" title="Laísmo">
         laísmo
        </a>
        )
       </dd>
      </dl>
