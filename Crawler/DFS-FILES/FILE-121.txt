      <p>
       The
       <b>
        Irish grid reference system
       </b>
       is a system of geographic
       <a href="/wiki/Grid_reference" title="Grid reference">
        grid references
       </a>
       commonly
       <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
        [
        <i>
         <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
          <span title="This claim needs references to reliable sources. (October 2015)">
           citation needed
          </span>
         </a>
        </i>
        ]
       </sup>
       used in
       <a href="/wiki/Ireland" title="Ireland">
        Ireland
       </a>
       (both
       <a href="/wiki/Northern_Ireland" title="Northern Ireland">
        Northern Ireland
       </a>
       and the
       <a href="/wiki/Republic_of_Ireland" title="Republic of Ireland">
        Republic of Ireland
       </a>
       ). The Irish grid partially overlaps the
       <a class="mw-redirect" href="/wiki/British_national_grid_reference_system" title="British national grid reference system">
        British grid
       </a>
       , and uses a similar co-ordinate system but with a meridian more suited to its westerly location.
      </p>

       <span class="mw-headline" id="Usage">
        Usage
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Irish_grid_reference_system&amp;action=edit&amp;section=1" title="Edit section: Usage">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       In general, neither
       <a href="/wiki/Ireland" title="Ireland">
        Ireland
       </a>
       nor
       <a href="/wiki/Great_Britain" title="Great Britain">
        Great Britain
       </a>
       uses
       <a href="/wiki/Latitude" title="Latitude">
        latitude
       </a>
       or
       <a href="/wiki/Longitude" title="Longitude">
        longitude
       </a>
       in describing internal geographic locations. Instead
       <a href="/wiki/Grid_reference" title="Grid reference">
        grid reference
       </a>
       systems are in common usage.
      </p>
      <p>
       The national grid referencing system was devised by the
       <a href="/wiki/Ordnance_Survey" title="Ordnance Survey">
        Ordnance Survey
       </a>
       , and is heavily used in their survey data, and in maps (whether published by the
       <a class="mw-redirect" href="/wiki/Ordnance_Survey_of_Ireland" title="Ordnance Survey of Ireland">
        Ordnance Survey of Ireland
       </a>
       , the
       <a href="/wiki/Ordnance_Survey_of_Northern_Ireland" title="Ordnance Survey of Northern Ireland">
        Ordnance Survey of Northern Ireland
       </a>
       or commercial map producers) based on those surveys. Additionally grid references are commonly quoted in other publications and data sources, such as guide books or government planning documents.
      </p>
      <p>
       The Ordnance Survey of Ireland has implemented
       <sup class="noprint Inline-Template" style="white-space:nowrap;">
        [
        <i>
         <a href="/wiki/Wikipedia:Manual_of_Style/Dates_and_numbers#Chronological_items" title="Wikipedia:Manual of Style/Dates and numbers">
          <span title="The time period mentioned near this tag is ambiguous. (February 2016)">
           when?
          </span>
         </a>
        </i>
        ]
       </sup>
       a new coordinate system for Ireland called
       <a href="/wiki/Irish_Transverse_Mercator" title="Irish Transverse Mercator">
        Irish Transverse Mercator
       </a>
       , or ITM, which will initially run in parallel with the existing Irish grid system. In both systems, the true origin is at 53° 30' N,
       <a href="/wiki/8th_meridian_west" title="8th meridian west">
        8° W
       </a>
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
       — a point in
       <a href="/wiki/Lough_Ree" title="Lough Ree">
        Lough Ree
       </a>
       , close to the western (
       <a href="/wiki/County_Roscommon" title="County Roscommon">
        Co. Roscommon
       </a>
       ) shore, whose grid reference is
       <a class="external text" href="http://getamap.ordnancesurvey.co.uk/getamap/frames.htm?mapAction=gaz&amp;gazName=g&amp;gazString=N000500" rel="nofollow">
        N 000 500
       </a>
       .
      </p>
      <h2>
       <span class="mw-headline" id="Grid_letters">
        Grid letters
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Irish_grid_reference_system&amp;action=edit&amp;section=2" title="Edit section: Grid letters">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <div class="thumb tright">
       <div class="thumbinner" style="width:222px;">
        <a class="image" href="/wiki/File:Irish_Grid.svg">
         <img alt="" class="thumbimage" data-file-height="485" data-file-width="485" height="220" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/c3/Irish_Grid.svg/220px-Irish_Grid.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/c3/Irish_Grid.svg/330px-Irish_Grid.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/c3/Irish_Grid.svg/440px-Irish_Grid.svg.png 2x" width="220"/>
        </a>
        <div class="thumbcaption">
         <div class="magnify">
          <a class="internal" href="/wiki/File:Irish_Grid.svg" title="Enlarge">
          </a>
         </div>
         Irish grid
        </div>
       </div>
      </div>
      <p>
       The area of Ireland is divided into 25 squares, measuring 100 by 100 km (62 by 62 mi), each identified by a single letter. The squares are numbered A to Z with I being omitted. Seven of the squares do not actually cover any land in Ireland: A, E, K, P, U, Y and Z.
      </p>
      <h2>
       <span class="mw-headline" id="Eastings_and_northings">
        Eastings and northings
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Irish_grid_reference_system&amp;action=edit&amp;section=3" title="Edit section: Eastings and northings">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       Within each square, eastings and northings from the origin (south west corner) of the square are given numerically. For example, G0305 means 'square G, 3 km (1.9 mi) east, 5 km (3.1 mi) north'. A location can be indicated to varying resolutions numerically, usually from two digits in each coordinate (for a 1 km (0.62 mi) square) through to five (for a 1 m (3 ft 3 in)) square; the most common usage is the
       <i>
        six figure grid reference
       </i>
       , employing three digits in each coordinate to determine a 100 m (330 ft) square.
      </p>
      <p>
       Coordinates may also be given relative to the origin of the entire 500 by 500 km (310 by 310 mi) grid (in the format easting, northing). For example, the location of the
       <a href="/wiki/Spire_of_Dublin" title="Spire of Dublin">
        Spire of Dublin
       </a>
       on
       <a href="/wiki/O%27Connell_Street" title="O'Connell Street">
        O'Connell Street
       </a>
       may be given as 315904, 234671 as well as O1590434671. Coordinates in this format must never be truncated, because, for example, 31590, 23467 is also a valid location.
      </p>
      <h2>
       <span class="mw-headline" id="Summary_parameters_of_the_Irish_Grid_coordinate_system">
        Summary parameters of the Irish Grid coordinate system
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Irish_grid_reference_system&amp;action=edit&amp;section=4" title="Edit section: Summary parameters of the Irish Grid coordinate system">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <dl>
       <dd>
        Spheroid: Airy Modified,
       </dd>
       <dd>
        Datum: 1965,
       </dd>
       <dd>
        Map projection:
        <a class="mw-redirect" href="/wiki/Transverse_Mercator" title="Transverse Mercator">
         Transverse Mercator
        </a>
       </dd>
       <dd>
        Latitude of Origin: 53°30'00 N
       </dd>
       <dd>
        Longitude of Origin:
        <a href="/wiki/8th_meridian_west" title="8th meridian west">
         8°00'00 W
        </a>
       </dd>
       <dd>
        Scale Factor: 1.000 035
       </dd>
       <dd>
        False Easting: 200000 m
       </dd>
       <dd>
        False Northing: 250000 m
       </dd>
      </dl>
