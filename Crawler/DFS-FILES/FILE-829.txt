      <p>
       <i>
        <b>
         Nationalencyklopedin
        </b>
       </i>
       (
       <small>
        Swedish:
       </small>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        <a href="/wiki/Help:IPA/Swedish" title="Help:IPA/Swedish">
         [natɧʊ²nɑːlɛnsʏklʊpɛˌdiːn]
        </a>
       </span>
       ), abbreviated
       <b>
        NE
       </b>
       , is a comprehensive contemporary
       <a href="/wiki/Swedish_language" title="Swedish language">
        Swedish-language
       </a>
       <a href="/wiki/Encyclopedia" title="Encyclopedia">
        encyclopedia
       </a>
       , initiated by a favourable loan from the
       <a href="/wiki/Government_of_Sweden" title="Government of Sweden">
        Government of Sweden
       </a>
       of 17 million
       <a href="/wiki/Swedish_krona" title="Swedish krona">
        Swedish kronor
       </a>
       in 1980, which was repaid by December 1990.
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
       The printed version consists of 20 volumes with 172,000 articles; the
       <a href="/wiki/Internet" title="Internet">
        Internet
       </a>
       version comprises 260,000 articles (as of June 2005).
      </p>

      </p>

      <h2>
       <span class="mw-headline" id="History">
        History
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Nationalencyklopedin&amp;action=edit&amp;section=1" title="Edit section: History">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       The project was born in 1980, when a government committee suggested that negotiations be initiated with various
       <a class="mw-redirect" href="/wiki/Publisher" title="Publisher">
        publishers
       </a>
       . This stage was finished in August 1985, when
       <a class="new" href="/w/index.php?title=Bra_B%C3%B6cker&amp;action=edit&amp;redlink=1" title="Bra Böcker (page does not exist)">
        Bra Böcker
       </a>
       <span class="noprint" style="font-size:85%; font-style: normal;">
        (
        <a class="extiw" href="https://sv.wikipedia.org/wiki/Bra_B%C3%B6cker" title="sv:Bra Böcker">
         sv
        </a>
        )
       </span>
       in
       <a href="/wiki/H%C3%B6gan%C3%A4s" title="Höganäs">
        Höganäs
       </a>
       became the publisher responsible for the project. The project specifications were for a modern reference work based on a
       <a href="/wiki/Science" title="Science">
        scientific
       </a>
       <a href="/wiki/Paradigm" title="Paradigm">
        paradigm
       </a>
       incorporating
       <a href="/wiki/Gender" title="Gender">
        gender
       </a>
       and
       <a href="/wiki/Natural_environment" title="Natural environment">
        environmental
       </a>
       issues.
      </p>
      <p>
       Pre-orders for the work were unprecedented; before the first volume was published in December 1989, 54,000 customers had ordered the encyclopedia. The last volume came out in 1996, with three supplemental volumes in 2000.
      </p>
      <p>
       Associated with the
       <i>
        Nationalencyklopedin
       </i>
       project are also:
      </p>
      <ul>
       <li>
        <i>
         NE:s Ordbok
        </i>
        , a
        <a href="/wiki/Dictionary" title="Dictionary">
         dictionary
        </a>
        in three volumes (1995–1996)
       </li>
       <li>
        <i>
         NE:s Årsband
        </i>
        , complementary volumes concerning current events and fast changing information distributed annually since 1997
       </li>
       <li>
        <i>
         NE:s Sverigeatlas
        </i>
        , an atlas of
        <a href="/wiki/Sweden" title="Sweden">
         Sweden
        </a>
        (1998)
       </li>
       <li>
        <i>
         NE:s Världsatlas
        </i>
        , a world atlas (1998)
       </li>
       <li>
        <i>
         NE-spelet
        </i>
        , a quiz game with 8,000 questions (1999)
       </li>
      </ul>
      <p>
       In 1997, the first
       <a href="/wiki/Digital_data" title="Digital data">
        digital
       </a>
       form of the encyclopedia was released on 6
       <a href="/wiki/CD-ROM" title="CD-ROM">
        CD-ROMs
       </a>
       (later on
       <a href="/wiki/DVD" title="DVD">
        DVD
       </a>
       as well), and in 2000 as an
       <a href="/wiki/Internet" title="Internet">
        Internet
       </a>
       <a class="mw-redirect" href="/wiki/Subscription" title="Subscription">
        subscription
       </a>
       service. The online version contains the dictionary as well as an updated version of the original encyclopedia. It has 356,000 entries, 183,000 of which are encyclopedic articles. The service has been completed with several features not available in the printed version, such as a Swedish–English dictionary.
      </p>
