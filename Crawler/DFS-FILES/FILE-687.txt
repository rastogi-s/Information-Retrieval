   <h1 class="firstHeading" id="firstHeading" lang="en">
    Object pronoun
   </h1>
   <div class="mw-body-content" id="bodyContent">
    <div class="noprint" id="siteSub">

    </div>
    <div id="contentSub">
    </div>

    <div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en">
     <div class="mw-parser-output">
      <p>
       In
       <a href="/wiki/Linguistics" title="Linguistics">
        linguistics
       </a>
       , an
       <b>
        object pronoun
       </b>
       is a
       <a href="/wiki/Personal_pronoun" title="Personal pronoun">
        personal pronoun
       </a>
       that is used typically as a
       <a class="mw-redirect" href="/wiki/Grammatical_object" title="Grammatical object">
        grammatical object
       </a>
       : the direct or indirect object of a verb, or the object of a preposition. Object pronouns contrast with
       <a href="/wiki/Subject_pronoun" title="Subject pronoun">
        subject pronouns
       </a>
       . Object pronouns in English take the
       <i>
        <a href="/wiki/Oblique_case" title="Oblique case">
         objective case
        </a>
       </i>
       , sometimes called the
       <i>
        oblique case
       </i>
       or
       <i>
        object case
       </i>
       .
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
       For example, the English object pronoun
       <i>
        me
       </i>
       is found in "They see
       <b>
        me
       </b>
       " (direct object), "He's giving
       <b>
        me
       </b>
       my book" (indirect object), and "Sit with
       <b>
        me
       </b>
       " (object of a preposition); this contrasts with the subject pronoun in "
       <b>
        I
       </b>
       see them," "
       <b>
        I
       </b>
       am getting my book," and "
       <b>
        I
       </b>
       am sitting here."
      </p>
      <p>
      </p>

      <h2>
       <span class="mw-headline" id="Archaic_second_person_forms">
        Archaic second person forms
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Object_pronoun&amp;action=edit&amp;section=2" title="Edit section: Archaic second person forms">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       Historically,
       <a href="/wiki/Middle_English" title="Middle English">
        Middle English
       </a>
       and
       <a href="/wiki/Early_Modern_English" title="Early Modern English">
        Early Modern English
       </a>
       retained the
       <a href="/wiki/T%E2%80%93V_distinction" title="T–V distinction">
        T–V distinction
       </a>
       ; the
       <a href="/wiki/Grammatical_person" title="Grammatical person">
        second person pronoun
       </a>
       had separate singular/familiar and plural/formal forms with subject and object forms of both. In standard modern forms of English, all second person forms have been reduced to simply "you". These forms are still retained (sometimes partially) in some dialects of
       <a href="/wiki/English_language_in_Northern_England" title="English language in Northern England">
        Northern English
       </a>
       ,
       <a href="/wiki/Scottish_English" title="Scottish English">
        Scottish English
       </a>
       , and in the
       <a href="/wiki/Scots_language" title="Scots language">
        Scots language
       </a>
       , a Germanic language closely related to English which diverged from it during the Early Modern period.
      </p>

      <h2>
       <span class="mw-headline" id="Other_languages">
        Other languages
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Object_pronoun&amp;action=edit&amp;section=3" title="Edit section: Other languages">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       In some languages the
       <i>
        direct
       </i>
       object pronoun and the
       <i>
        indirect
       </i>
       object pronoun have separate forms. For example, in the
       <a class="mw-redirect" href="/wiki/Spanish_object_pronoun" title="Spanish object pronoun">
        Spanish object pronoun
       </a>
       system,
       <i>
        direct object
       </i>
       :
       <b>
        Lo
       </b>
       mandaron a la escuela (They sent
       <b>
        him
       </b>
       to school) and
       <i>
        indirect object
       </i>
       :
       <b>
        Le
       </b>
       mandaron una carta (They sent
       <b>
        him
       </b>
       a letter). Other languages divide object pronouns into a larger variety of classes. On the other hand, many languages, for example Persian, do not have distinct object pronouns:
       <i>
        <b>
         Man
        </b>
        Farsi balad-am
       </i>
       (
       <b>
        I
       </b>
       can speak
       <a href="/wiki/Persian_language" title="Persian language">
        Persian
       </a>
       ).
       <i>
        <b>
         Man
        </b>
        ra mishenasad.
       </i>
       (He knows
       <b>
        me
       </b>
       ).
      </p>
      <h2>
       <span class="mw-headline" id="History">
        History
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Object_pronoun&amp;action=edit&amp;section=4" title="Edit section: History">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       Object pronouns, in languages where they are distinguished from subject pronouns, are typically a vestige of an older
       <a href="/wiki/Grammatical_case" title="Grammatical case">
        case
       </a>
       system. English, for example, once had an extensive
       <a href="/wiki/Declension" title="Declension">
        declension
       </a>
       system that specified distinct
       <a href="/wiki/Accusative_case" title="Accusative case">
        accusative
       </a>
       and
       <a href="/wiki/Dative_case" title="Dative case">
        dative
       </a>
       case
       <i>
        forms
       </i>
       for both nouns and pronouns. And after a preposition, a noun or pronoun could be in either of these cases, or in the
       <a href="/wiki/Genitive_case" title="Genitive case">
        genitive
       </a>
       or
       <a href="/wiki/Instrumental_case" title="Instrumental case">
        instrumental
       </a>
       case. With the exception of the genitive (the
       <a href="/wiki/English_possessive" title="English possessive">
        "apostrophe-s" form
       </a>
       ), in nouns this system disappeared entirely, while in personal pronouns it collapsed into a single case, covering the roles of both accusative and dative, as well as all instances after a preposition. That is, the new oblique (object) case came to be used for the object of either a verb or a preposition, contrasting with the genitive, which links two nouns.
      </p>
      <p>
       For a discussion of the use of historically object pronouns in subject position in English (e.g. "Jay and
       <b>
        me
       </b>
       will arrive later"), see the article on
       <a href="/wiki/English_personal_pronouns#Case_usage" title="English personal pronouns">
        English personal pronouns
       </a>
       .
      </p>
