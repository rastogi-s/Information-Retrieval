      <p>
       <i>
        <b>
         Argentina
        </b>
       </i>
       (
       <b>
        silverweeds
       </b>
       ) is a
       <a href="/wiki/Genus" title="Genus">
        genus
       </a>
       of
       <a href="/wiki/Plant" title="Plant">
        plants
       </a>
       in the
       <a href="/wiki/Rose" title="Rose">
        rose
       </a>
       <a href="/wiki/Family_(biology)" title="Family (biology)">
        family
       </a>
       (
       <a href="/wiki/Rosaceae" title="Rosaceae">
        Rosaceae
       </a>
       ) which is accepted by some authors,
       <sup class="reference" id="cite_ref-Sojak_1-1">
        <a href="#cite_note-Sojak-1">
         [1]
        </a>
       </sup>
       as containing 64 species, but classified in
       <i>
        <a href="/wiki/Potentilla" title="Potentilla">
         Potentilla
        </a>
       </i>
       <sup class="reference" id="cite_ref-2">
        <a href="#cite_note-2">
         [2]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-3">
        <a href="#cite_note-3">
         [3]
        </a>
       </sup>
       sect.
       <i>
        Leptostylae
       </i>
       by others.
      </p>
