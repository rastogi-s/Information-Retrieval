      <p>
       The
       <b>
        International Terrestrial Reference System
       </b>
       (
       <b>
        ITRS
       </b>
       ) describes procedures for creating
       <a href="/wiki/Frame_of_reference" title="Frame of reference">
        reference frames
       </a>
       suitable for use with measurements on or near the Earth's surface. This is done in much the same way that a
       <a href="/wiki/Standard_(metrology)" title="Standard (metrology)">
        physical standard
       </a>
       might be described as a set of procedures for creating a
       <i>
        realization
       </i>
       of that standard. The ITRS defines a
       <a class="mw-redirect" href="/wiki/Geocentric" title="Geocentric">
        geocentric
       </a>
       system of coordinates using the
       <a class="mw-redirect" href="/wiki/SI" title="SI">
        SI
       </a>
       system of measurement.
      </p>
      <p>
       An
       <b>
        International Terrestrial Reference Frame
       </b>
       (
       <b>
        ITRF
       </b>
       ) is a realization of the ITRS. New ITRF solutions are produced every few years, using the latest mathematical and surveying techniques to attempt to realize the ITRS as precisely as possible. Due to
       <a class="mw-redirect" href="/wiki/Experimental_error" title="Experimental error">
        experimental error
       </a>
       , any given ITRF will differ very slightly from any other realization of the ITRF. Also, the difference between the latest
       <a href="/wiki/World_Geodetic_System" title="World Geodetic System">
        WGS84
       </a>
       and the latest ITRF is only a few centimeters.
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
      </p>
      <p>
       Practical navigation systems are in general referenced to a specific ITRF solution, or to their own coordinate systems which are then referenced to an ITRF solution.
      </p>
      <p>
       The
       <i>
        Galileo Terrestrial Reference Frame
       </i>
       (
       <i>
        GTRF
       </i>
       ) is used for the
       <a href="/wiki/Galileo_(satellite_navigation)" title="Galileo (satellite navigation)">
        Galileo
       </a>
       navigation system; currently defined as ITRF2005.
      </p>
      <p>
       The ITRS and ITRF solutions are maintained by the
       <a href="/wiki/International_Earth_Rotation_and_Reference_Systems_Service" title="International Earth Rotation and Reference Systems Service">
        International Earth Rotation and Reference Systems Service
       </a>
       (
       <a class="mw-redirect" href="/wiki/IERS" title="IERS">
        IERS
       </a>
       ). GTRF is defined by the
       <a href="/wiki/European_Space_Agency" title="European Space Agency">
        European Space Agency
       </a>
       (ESA).
      </p>

      </p>
