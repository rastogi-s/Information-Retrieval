      <p>
       The
       <b>
        acute accent
       </b>
       ( ´ ) is a
       <a href="/wiki/Diacritic" title="Diacritic">
        diacritic
       </a>
       used in many modern written languages with
       <a href="/wiki/Alphabet" title="Alphabet">
        alphabets
       </a>
       based on the
       <a href="/wiki/Latin_alphabet" title="Latin alphabet">
        Latin
       </a>
       ,
       <a href="/wiki/Cyrillic_script" title="Cyrillic script">
        Cyrillic
       </a>
       , and
       <a href="/wiki/Greek_alphabet" title="Greek alphabet">
        Greek
       </a>
       scripts.
      </p>

      <h2>
       <span class="mw-headline" id="Uses">
        Uses
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Acute_accent&amp;action=edit&amp;section=1" title="Edit section: Uses">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <h3>
       <span class="mw-headline" id="History">
        History
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Acute_accent&amp;action=edit&amp;section=2" title="Edit section: History">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       An early precursor of the acute accent was the
       <a href="/wiki/Apex_(diacritic)" title="Apex (diacritic)">
        apex
       </a>
       , used in
       <a class="mw-redirect" href="/wiki/Latin_language" title="Latin language">
        Latin
       </a>
       inscriptions to mark
       <a href="/wiki/Vowel_length" title="Vowel length">
        long vowels
       </a>
       .
      </p>
      <h3>
       <span class="mw-headline" id="Pitch">
        Pitch
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Acute_accent&amp;action=edit&amp;section=3" title="Edit section: Pitch">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <h4>
       <span class="mw-headline" id="Greek">
        Greek
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Acute_accent&amp;action=edit&amp;section=4" title="Edit section: Greek">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <div class="hatnote navigation-not-searchable" role="note">
       See also:
       <a href="/wiki/Ancient_Greek_accent" title="Ancient Greek accent">
        Ancient Greek accent
       </a>
      </div>
      <p>
       The acute accent was first used in the
       <a href="/wiki/Greek_diacritics" title="Greek diacritics">
        polytonic orthography
       </a>
       of
       <a href="/wiki/Ancient_Greek" title="Ancient Greek">
        Ancient Greek
       </a>
       , where it indicated a syllable with a high
       <a href="/wiki/Pitch_accent" title="Pitch accent">
        pitch
       </a>
       . In Modern Greek, a
       <a href="/wiki/Stress_(linguistics)" title="Stress (linguistics)">
        stress accent
       </a>
       has replaced the pitch accent, and the acute marks the stressed syllable of a word. The Greek name of the accented syllable was and is
       <span lang="grc" xml:lang="grc">
        <a class="extiw" href="https://en.wiktionary.org/wiki/%E1%BD%80%CE%BE%CE%B5%E1%BF%96%CE%B1#Ancient_Greek" title="wikt:ὀξεῖα">
         ὀξεῖα
        </a>
       </span>
       (
       <i>
        oxeîa
       </i>
       , Modern Greek
       <i>
        oxía
       </i>
       ) "sharp" or "high", which was
       <a href="/wiki/Calque" title="Calque">
        calqued
       </a>
       (loan-translated) into
       <a href="/wiki/Latin" title="Latin">
        Latin
       </a>
       as
       <i lang="la" xml:lang="la">
        <a class="extiw" href="https://en.wiktionary.org/wiki/acutus#Latin" title="wikt:acutus">
         acūta
        </a>
       </i>
       "sharpened".
      </p>
      <h3>
       <span class="mw-headline" id="Stress">
        Stress
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Acute_accent&amp;action=edit&amp;section=5" title="Edit section: Stress">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       The acute accent marks the
       <a href="/wiki/Stress_(linguistics)" title="Stress (linguistics)">
        stressed vowel
       </a>
       of a word in several languages:
      </p>
      <ul>
       <li>
        <a href="/wiki/Blackfoot_language" title="Blackfoot language">
         Blackfoot
        </a>
        uses acute accents to show the place of stress in a word: soyópokistsi "leaves".
       </li>
       <li>
        <a href="/wiki/Bulgarian_language" title="Bulgarian language">
         Bulgarian
        </a>
        : stress, which is variable in Bulgarian, is not usually indicated in Bulgarian except in dictionaries and sometimes in homonyms that are distinguished only by stress. However, Bulgarian usually uses the
        <a href="/wiki/Grave_accent#Stress" title="Grave accent">
         grave accent
        </a>
        to mark the vowel in a stressed syllable, unlike Russian, which uses the acute accent.
       </li>
       <li>
        <a href="/wiki/Catalan_language" title="Catalan language">
         Catalan
        </a>
        uses it in stressed vowels:
        <i>
         é
        </i>
        ,
        <i>
         í
        </i>
        ,
        <i>
         ó
        </i>
        ,
        <i>
         ú
        </i>
        .
       </li>
       <li>
        <a href="/wiki/Dutch_language" title="Dutch language">
         Dutch
        </a>
        uses it to mark stress (
        <i>
         vóórkomen
        </i>
        –
        <i>
         voorkómen
        </i>
        , meaning
        <i>
         occur
        </i>
        and
        <i>
         prevent
        </i>
        respectively) or a more
        <a href="/wiki/Vowel#Height" title="Vowel">
         closed
        </a>
        vowel (
        <i>
         hé
        </i>
        –
        <i>
         hè
        </i>
        , equivalent to English
        <i>
         hey
        </i>
        and
        <i>
         heh
        </i>
        ) if it is not clear from context. Sometimes, it is simply used for disambiguation, as in
        <i>
         één
        </i>
        –
        <i>
         een
        </i>
        , meaning "one" and "a(n)".
       </li>
       <li>
        <a href="/wiki/Galician_language" title="Galician language">
         Galician
        </a>
       </li>
       <li>
        <a href="/wiki/Italian_language" title="Italian language">
         Italian
        </a>
        The accent is used to indicate the stress in a word, or whether the vowel is "open" or "wide", or "closed", or "narrow". For example,
        <i>
         pèsca
        </i>
        = Peach ("open" or "wide" vowel, = pehscah, as in "pen"); while
        <i>
         pésca
        </i>
        = fishing ("closed" or "narrow" vowel payscah, as in "pain").
       </li>
       <li>
        <a href="/wiki/Lakota_language" title="Lakota language">
         Lakota
        </a>
        . For example,
        <i>
         kákhi
        </i>
        "in that direction" but
        <i>
         kakhí
        </i>
        "take something to someone back there".
       </li>
       <li>
        <a class="mw-redirect" href="/wiki/Leonese_language" title="Leonese language">
         Leonese
        </a>
        uses it for marking stress or disambiguation.
       </li>
       <li>
        <a href="/wiki/Modern_Greek" title="Modern Greek">
         Modern Greek
        </a>
        marks the stressed vowel of every
        <a href="/wiki/Syllable" title="Syllable">
         polysyllabic
        </a>
        word: ά (
        <i>
         á
        </i>
        ), έ (
        <i>
         é
        </i>
        ), ή (
        <i>
         í
        </i>
        ), ί (
        <i>
         í
        </i>
        ), ό (
        <i>
         ó
        </i>
        ), ύ (
        <i>
         ý
        </i>
        ), ώ (
        <i>
         ó
        </i>
        ).
       </li>
       <li>
        <a href="/wiki/Hopi" title="Hopi">
         Hopi
        </a>
        has acute to mark a higher tone.
       </li>
       <li>
        <a href="/wiki/Navajo_language" title="Navajo language">
         Navajo
        </a>
        where the acute marks a higher tone.
       </li>
       <li>
        <a href="/wiki/Occitan_language" title="Occitan language">
         Occitan
        </a>
       </li>
       <li>
        <a href="/wiki/Portuguese_language" title="Portuguese language">
         Portuguese
        </a>
        :
        <i>
         á
        </i>
        ,
        <i>
         é
        </i>
        ,
        <i>
         í
        </i>
        ,
        <i>
         ó
        </i>
        ,
        <i>
         ú
        </i>
        . It may also indicate height (see below).
       </li>
       <li>
        <a href="/wiki/Russian_language" title="Russian language">
         Russian
        </a>
        . Stress is irregular in Russian, and in reference and teaching materials (dictionaries and books for children or foreigners), stress is indicated by an acute accent above the stressed vowel. The acute accent can be used both in the Cyrillic and sometimes in the romanised text.
       </li>
       <li>
        <a href="/wiki/Spanish_language" title="Spanish language">
         Spanish
        </a>
        marks stressed syllables in words that deviate from the
        <a href="/wiki/Spanish_orthography#Stress_and_accentuation" title="Spanish orthography">
         standardized stress patterns
        </a>
        . It is also used to distinguish minimal pairs such as
        <i>
         el
        </i>
        (the) and
        <i>
         él
        </i>
        (he).
       </li>
       <li>
        <a href="/wiki/Norwegian_language" title="Norwegian language">
         Norwegian
        </a>
        ,
        <a href="/wiki/Swedish_language" title="Swedish language">
         Swedish
        </a>
        and
        <a href="/wiki/Danish_language" title="Danish language">
         Danish
        </a>
        use the acute accent to indicate that a terminal syllable with the
        <i>
         e
        </i>
        is stressed and is often omitted if it does not change the meaning:
        <i>
         armen
        </i>
        (first syllable stressed) means "the arm" while
        <i>
         armé(e)n
        </i>
        means "the army";
        <i>
         ide
        </i>
        (first syllable stressed) means "bear's den" while
        <i>
         idé
        </i>
        means "idea". Also stress-related are the different spellings of the words en/én and et/ét (the indefinite article and the word "one" in Danish and Norwegian). Then, the acute points out that there is one and only one of the object, which derives from the obsolete spelling(s) een and eet. Some loanwords, mainly from French, are also written with the acute accent, such as Norwegian and Swedish
        <i>
         kafé
        </i>
        and Danish
        <i>
         café
        </i>
        (also
        <i>
         cafe
        </i>
        ).
       </li>
       <li>
        <a href="/wiki/Ukrainian_language" title="Ukrainian language">
         Ukrainian
        </a>
        : marks the stress, but in regular typography is only used when it can help to distinguish between
        <a href="/wiki/Homograph" title="Homograph">
         homographs
        </a>
        :
        <i>
         за́мок
        </i>
        (castle) vs.
        <i>
         замо́к
        </i>
        (lock). Commonly used in dictionaries and some children books.
       </li>
       <li>
        <a href="/wiki/Welsh_language" title="Welsh language">
         Welsh
        </a>
        : word stress usually falls on the penultimate syllable, but one way of indicating stress on a final (short) vowel is by the use of the acute accent. In the
        <a href="/wiki/Welsh_orthography" title="Welsh orthography">
         Welsh orthography
        </a>
        , it can be on any vowel:
        <i>
         á
        </i>
        ,
        <i>
         é
        </i>
        ,
        <i>
         í
        </i>
        ,
        <i>
         ó
        </i>
        ,
        <i>
         ú
        </i>
        ,
        <i>
         ẃ
        </i>
        , or
        <i>
         ý
        </i>
        . Examples:
        <i>
         casáu
        </i>
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         <a href="/wiki/Help:IPA/Welsh" title="Help:IPA/Welsh">
          [kaˈsaɨ, kaˈsai]
         </a>
        </span>
        "to hate",
        <i>
         sigarét
        </i>
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         <a href="/wiki/Help:IPA/Welsh" title="Help:IPA/Welsh">
          [sɪɡaˈrɛt]
         </a>
        </span>
        "cigarette",
        <i>
         ymbarél
        </i>
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         <a href="/wiki/Help:IPA/Welsh" title="Help:IPA/Welsh">
          [əmbaˈrɛl]
         </a>
        </span>
        "umbrella".
       </li>
      </ul>
      <h3>
       <span class="mw-headline" id="Height">
        Height
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Acute_accent&amp;action=edit&amp;section=6" title="Edit section: Height">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       The acute accent marks the
       <a class="mw-redirect" href="/wiki/Vowel_height" title="Vowel height">
        height
       </a>
       of some stressed vowels in various
       <a href="/wiki/Romance_languages" title="Romance languages">
        Romance languages
       </a>
       .
      </p>
      <ul>
       <li>
        To mark high vowels:
        <ul>
         <li>
          <a class="mw-redirect" href="/wiki/Bislama_language" title="Bislama language">
           Bislama
          </a>
          . The acute is used only on
          <i>
           é
          </i>
          , but only in one of the two orthographies. It distinguishes
          <i>
           é
          </i>
          <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
           [e]
          </span>
          from
          <i>
           e
          </i>
          <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
           [ɛ]
          </span>
          .
          <sup class="reference" id="cite_ref-1">
           <a href="#cite_note-1">
            [1]
           </a>
          </sup>
          The orthography after 1995 (which has no diacritics), does not distinguish these sounds.
         </li>
         <li>
          <a href="/wiki/Catalan_language" title="Catalan language">
           Catalan
          </a>
          . The acute marks the quality of the vowels
          <i>
           é
          </i>
          <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
           [e]
          </span>
          (as opposed to
          <i>
           è
          </i>
          <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
           [ɛ]
          </span>
          ), and
          <i>
           ó
          </i>
          <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
           [o]
          </span>
          (as opposed to
          <i>
           ò
          </i>
          <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
           [ɔ]
          </span>
          ).
         </li>
         <li>
          <a href="/wiki/French_language" title="French language">
           French
          </a>
          . The acute is used only on
          <i>
           é
          </i>
          . It is known as
          <i>
           accent aigu
          </i>
          , in contrast to the
          <i>
           <a href="/wiki/Grave_accent" title="Grave accent">
            accent grave
           </a>
          </i>
          which is the accent sloped the other way. It distinguishes
          <i>
           é
          </i>
          <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
           [e]
          </span>
          from
          <i>
           è
          </i>
          <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
           [ɛ]
          </span>
          ,
          <i>
           ê
          </i>
          <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
           [ɛ]
          </span>
          , and
          <i>
           e
          </i>
          <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
           [ə]
          </span>
          . Unlike other Romance languages, the accent marks do not imply stress in French.
         </li>
         <li>
          <a href="/wiki/Italian_language" title="Italian language">
           Italian
          </a>
          . The acute accent (sometimes called
          <i>
           accento chiuso
          </i>
          , "closed accent" in Italian) is compulsory only in words of more than one syllable stressed on their final vowel (and a few other words). Words ending in stressed -o are never marked with an acute accent (
          <i>
           ó
          </i>
          ), but with a
          <a href="/wiki/Grave_accent" title="Grave accent">
           grave accent
          </a>
          (
          <i>
           ò
          </i>
          ). Therefore, only
          <i>
           é
          </i>
          and
          <i>
           è
          </i>
          are normally contrasted, typically in words ending in
          <i>
           -ché
          </i>
          , such as
          <i>
           perché
          </i>
          ("why/because"); in the conjugated

         </li>

          <i>

          <i>

          </i>
          <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">

          <i>
           ò
          </i>
          <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
           [ɔː]
          </span>

          <i>
           é
          </i>

          </i>

          <i>

          </i>

          The other vowels (

          </i>

       </li>

          </i>

          <i>

          </i>

           </a>

            e
           </a>
           /

          </i>

            o

However, the accent is only used in words whose stressed syllable is in an unpredictable location within the word: where the location of the stressed syllable is predictable, no accent is used, and the height of the stressed vowel cannot then usually be determined solely from the word's spelling.

        </a>

        </i>

        </a>

        </i>

        </i>

        <i>

        </i>

      <h3>
       <span class="mw-headline" id="Palatalization">
        Palatalization
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Acute_accent&amp;action=edit&amp;section=10" title="Edit section: Palatalization">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       A graphically similar, but not identical, mark is indicative of a
       <a href="/wiki/Palatalization_(phonetics)" title="Palatalization (phonetics)">
        palatalized
       </a>
       sound in several languages.
      </p>
      <p>
       In
       <a href="/wiki/Polish_language" title="Polish language">
        Polish
       </a>
       , such a mark is known as a
       <i>
        <b>
         kreska
        </b>
       </i>
       (
       <i>
        English:
        <span lang="en" xml:lang="en">
         stroke
        </span>
       </i>
       ) and is an integral part of several letters: four consonants and one vowel. When appearing in consonants, it indicates
       <a href="/wiki/Palatalization_(phonetics)" title="Palatalization (phonetics)">
        palatalization
       </a>
       , similar to the use of the
       <i>
        <a class="mw-redirect" href="/wiki/H%C3%A1%C4%8Dek" title="Háček">
         háček
        </a>
       </i>
       in
       <a href="/wiki/Czech_language" title="Czech language">
        Czech
       </a>
       and other Slavic languages (e.g.
       <i>
        sześć
       </i>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        <a href="/wiki/Help:IPA/Polish" title="Help:IPA/Polish">
         [ˈʂɛɕtɕ]
        </a>
       </span>
       "six"). However, in contrast to the
       <i>
        háček
       </i>
       which is usually used for
       <a href="/wiki/Postalveolar_consonant" title="Postalveolar consonant">
        postalveolar consonants
       </a>
       , the
       <i>
        kreska
       </i>
       denotes
       <a href="/wiki/Alveolo-palatal_consonant" title="Alveolo-palatal consonant">
        alveolo-palatal consonants
       </a>
       . In traditional Polish
       <a href="/wiki/Typography" title="Typography">
        typography
       </a>
       , the
       <i>
        kreska
       </i>
       is more nearly vertical than the acute accent, and placed slightly right of center.
       <sup class="reference" id="cite_ref-4">
        <a href="#cite_note-4">
         [4]
        </a>
       </sup>
       A similar rule applies to the Belarusian Latin alphabet
       <a class="mw-redirect" href="/wiki/Lacinka" title="Lacinka">
        Lacinka
       </a>
       . However, for computer use,
       <a href="/wiki/Unicode" title="Unicode">
        Unicode
       </a>
       conflates the
       <a class="mw-redirect" href="/wiki/Codepoint" title="Codepoint">
        codepoints
       </a>
       for these letters with those of the accented Latin letters of similar appearance.
      </p>
      <p>
       In
       <a href="/wiki/Serbo-Croatian" title="Serbo-Croatian">
        Serbo-Croatian
       </a>
       , as in Polish, the letter
       <i>
        ć
       </i>
       is used to represent a palatalized
       <i>
        t
       </i>
       .
      </p>
      <p>
       In the
       <a href="/wiki/Romanization" title="Romanization">
        romanization
       </a>
       of
       <a href="/wiki/Macedonian_language" title="Macedonian language">
        Macedonian
       </a>
       ,
       <i>
        ǵ
       </i>
       and
       <i>
        ḱ
       </i>
       represent the
       <a href="/wiki/Cyrillic_script" title="Cyrillic script">
        Cyrillic
       </a>
       letters
       <a href="/wiki/Gje" title="Gje">
        ѓ
       </a>
       and
       <a href="/wiki/Kje" title="Kje">
        ќ
       </a>
       , which stand for
       <a href="/wiki/Palatal_consonant" title="Palatal consonant">
        palatal
       </a>
       or
       <a href="/wiki/Alveolo-palatal_consonant" title="Alveolo-palatal consonant">
        alveolo-palatal
       </a>
       consonants, though
       <i>
        gj
       </i>
       and
       <i>
        kj
       </i>
       (or
       <i>
        đ
       </i>
       and
       <i>
        ć
       </i>
       ) are more commonly used for this purpose
       <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
        [
        <i>
         <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
          <span title="This claim needs references to reliable sources. (August 2012)">
           citation needed
          </span>
         </a>
        </i>
        ]
       </sup>
       . The same two letters are used to transcribe the postulated
       <a href="/wiki/Proto-Indo-European_language" title="Proto-Indo-European language">
        Proto-Indo-European
       </a>
       <a href="/wiki/Phoneme" title="Phoneme">
        phonemes
       </a>
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        /ɡʲ/
       </span>
       and
       <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
        /kʲ/
       </span>
       .
      </p>
      <h3>
       <span class="mw-headline" id="Tone">
        Tone
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Acute_accent&amp;action=edit&amp;section=11" title="Edit section: Tone">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       In the
       <a href="/wiki/Vietnamese_alphabet" title="Vietnamese alphabet">
        Quốc Ngữ
       </a>
       system for
       <a href="/wiki/Vietnamese_language" title="Vietnamese language">
        Vietnamese
       </a>
       , the Yale romanization for
       <a href="/wiki/Cantonese" title="Cantonese">
        Cantonese
       </a>
       and the
       <a href="/wiki/Pinyin" title="Pinyin">
        Pinyin
       </a>
       <a href="/wiki/Romanization" title="Romanization">
        romanization
       </a>
       for
       <a href="/wiki/Standard_Chinese" title="Standard Chinese">
        Mandarin Chinese
       </a>
       , the acute accent indicates a rising
       <a href="/wiki/Tone_(linguistics)" title="Tone (linguistics)">
        tone
       </a>
       . In Mandarin, the alternative to the acute accent is the number 2 after the syllable: lái = lai2. In Cantonese Yale, the acute accent is either tone 2, or tone 5 if the vowel(s) are followed by 'h' (if the number form is used, 'h' is omitted): má = ma2, máh - ma5.
      </p>
      <p>
       In
       <a class="mw-redirect" href="/wiki/African_languages" title="African languages">
        African languages
       </a>
       and
       <a href="/wiki/Athabaskan_languages" title="Athabaskan languages">
        Athabaskan languages
       </a>
       , it frequently marks a high tone, e.g.,
       <a href="/wiki/Yoruba_language" title="Yoruba language">
        Yoruba
       </a>
       <i>
        apá
       </i>
       'arm',
       <a href="/wiki/Nobiin_language" title="Nobiin language">
        Nobiin
       </a>
       <i>
        féntí
       </i>
       'sweet date',
       <a class="mw-redirect" href="/wiki/Ekoti_language" title="Ekoti language">
        Ekoti
       </a>
       <i>
        kaláwa
       </i>
       'boat',
       <a href="/wiki/Navajo_language" title="Navajo language">
        Navajo
       </a>
       <i>
        t’áá
       </i>
       'just'.
      </p>
      <p>
       The acute accent is used in Serbo-Croatian dictionaries and linguistic publications to indicate a high-rising accent. It is not used in everyday writing.
      </p>

        </span>
        <a href="/w/index.php?title=Acute_accent&amp;action=edit&amp;section=12" title="Edit section: Disambiguation">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       The acute accent is used to disambiguate certain words which would otherwise be
       <a href="/wiki/Homograph" title="Homograph">
        homographs
       </a>
       in the following languages:
      </p>
      <ul>
       <li>
        Catalan. Examples:
        <i>
         són
        </i>
        "they are" vs.
        <i>
         son
        </i>
        "tiredness",
        <i>
         més
        </i>
        "more" vs.
        <i>
         mes
        </i>
        "month".
       </li>
       <li>
        <a href="/wiki/Danish_language" title="Danish language">
         Danish
        </a>
        . Examples:
        <i>
         én
        </i>
        "one" vs.
        <i>
         en
        </i>
        "a/an";
        <i>
         fór
        </i>
        "went" vs.
        <i>
         for
        </i>
        "for";
        <i>
         véd
        </i>
        "know(s)" vs.
        <i>
         ved
        </i>
        "by";
        <i>
         gǿr
        </i>
        "bark(s)" vs.
        <i>
         gør
        </i>
        "do(es)";
        <i>
         dǿr
        </i>
        "die(s)" vs.
        <i>
         dør
        </i>
        "door";
        <i>
         allé
        </i>
        "alley" vs.
        <i>
         alle
        </i>
        "everybody". Furthermore, it is also used for the imperative form of verbs ending in
        <i>
         -ere
        </i>
        , which lose their final
        <i>
         e
        </i>
        and might be mistaken for plurals of a noun (which most often end in
        <i>
         -er
        </i>
        ):
        <i>
         analysér
        </i>
        is the imperative form of
        <i>
         at analysere
        </i>
        "to analyse",
        <i>
         analyser
        </i>
        is "analyses", plural of the noun
        <i>
         analyse
        </i>
        "analysis". Using an acute accent is always optional, never required.
       </li>
       <li>
        <a href="/wiki/Dutch_language" title="Dutch language">
         Dutch
        </a>
        . Examples:
        <i>
         één
        </i>
        "one" vs.
        <i>
         een
        </i>
        "a/an";
        <i>
         vóór
        </i>
        "before" vs.
        <i>
         voor
        </i>
        "for";
        <i>
         vóórkomen
        </i>
        "to exist/to happen" vs.
        <i>
         voorkómen
        </i>
        "to prevent/to avoid". Using an acute accent is mostly optional.
       </li>
       <li>
        <a href="/wiki/Modern_Greek" title="Modern Greek">
         Modern Greek
        </a>
        . Although all polysyllabic words have an acute accent on the stressed syllable, in monosyllabic words the presence or absence of an accent may disambiguate. The most common case is η, the feminine definite article ("the"), versus ή, meaning "or". Other cases include που ("who"/"which") versus πού ("where") and πως ("that", as in "he told me
        <i>
         that
        </i>
        ...") versus πώς ("how").
       </li>
       <li>
        <a href="/wiki/Norwegian_language" title="Norwegian language">
         Norwegian
        </a>
        . It is used to indicate stress on a vowel otherwise not expected to have stress. Most words are stressed on the first syllable and diacritical marks are rarely used. Although incorrect, it is frequently used to mark the imperative form of verbs ending in
        <i>
         -ere
        </i>
        as it is in Danish:
        <i>
         kontrollér
        </i>
        is the imperative form of "to control",
        <i>
         kontroller
        </i>
        is the noun "controls". The simple past of the verb
        <i>
         å fare
        </i>
        , "to travel", can optionally be written
        <i>
         fór
        </i>
        , to distinguish it from
        <i>
         for
        </i>
        (preposition "for" as in English),
        <i>
         fôr
        </i>
        "feed"
        <i>
         n.
        </i>
        /"lining", or
        <i>
         fòr
        </i>
        (only in
        <a href="/wiki/Nynorsk" title="Nynorsk">
         Nynorsk
        </a>
        ) "narrow ditch, trail by plow
        <i>
         (all the diacritics in these examples are optional.
         <sup class="reference" id="cite_ref-5">
          <a href="#cite_note-5">
           [5]
          </a>
         </sup>
         )
        </i>
       </li>
       <li>
        <a href="/wiki/Portuguese_language" title="Portuguese language">
         Portuguese
        </a>
        . Examples:
        <i>
         avô
        </i>
        "grandfather" vs.
        <i>
         avó
        </i>
        "grandmother",
        <i>
         nós
        </i>
        "subject pronoun
        <i>
         we
        </i>
        " vs.
        <i>
         nos
        </i>
        "oblique case".
       </li>
       <li>
        <a href="/wiki/Spanish_language" title="Spanish language">
         Spanish
        </a>
        . Covers various question word / relative pronoun pairs where the first is stressed and the second is a
        <a href="/wiki/Clitic" title="Clitic">
         clitic
        </a>
        , such as
        <i>
         cómo
        </i>
        (interrogative "how") and
        <i>
         como
        </i>
        (non-interrogative "how", comparative "like", "I eat"
        <sup class="reference" id="cite_ref-6">
         <a href="#cite_note-6">
          [6]
         </a>
        </sup>
        ), differentiates
        <i>
         qué
        </i>
        (what) from
        <i>
         que
        </i>
        (that),
        <i>
         dónde
        </i>
        and
        <i>
         donde
        </i>
        "where", and some other words such as
        <i>
         tú
        </i>
        "you" and
        <i>
         tu
        </i>
        "your,"
        <i>
         té
        </i>
        "tea" and
        <i>
         te
        </i>
        "you" (direct/indirect object),
        <i>
         él
        </i>
        "he/him" and
        <i>
         el
        </i>
        ("the", masculine),
        <i>
         sólo
        </i>
        "only" (as in "solamente") and
        <i>
         solo
        </i>
        "alone". This usage of the acute accent is called
        <i>
         tilde diacrítica
        </i>
        .
       </li>
       <li>
        <a href="/wiki/Russian_language" title="Russian language">
         Russian
        </a>
        . Acute accents (technically,
        <a href="#Stress">
         stress marks
        </a>
        ) are used in dictionaries to indicate the stressed syllable. They may also be optionally used to disambiguate both between
        <a href="/wiki/Minimal_pair" title="Minimal pair">
         minimal pairs
        </a>
        , such as за́мок (read as zámak, means "castle") and замо́к (read as zamók, means "lock"), and between
        <a href="/wiki/Interrogative_word" title="Interrogative word">
         question words
        </a>
        and
        <a href="/wiki/Relative_pronoun" title="Relative pronoun">
         relative pronouns
        </a>
        such as что ("what", stressed, or "that", unstressed), similarly to Spanish. This is rare, however, as usually meaning is determined by context and no stress mark is written. The same rules apply to
        <a href="/wiki/Ukrainian_language" title="Ukrainian language">
         Ukrainian
        </a>
        ,
        <a href="/wiki/Rusyn_language" title="Rusyn language">
         Rusyn
        </a>
        ,
        <a href="/wiki/Belarusian_language" title="Belarusian language">
         Belarusian
        </a>
        and
        <a href="/wiki/Bulgarian_language" title="Bulgarian language">
         Bulgarian
        </a>
        .
       </li>
      </ul>
      <h3>
       <span class="mw-headline" id="Emphasis">
        Emphasis
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Acute_accent&amp;action=edit&amp;section=13" title="Edit section: Emphasis">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       In Dutch, the acute accent can also be used to emphasize an individual word within a sentence. For example,
       <i>
        Dit is ónze auto, niet die van jullie
       </i>
       , "This is
       <i>
        our
       </i>
       car, not yours." In this example,
       <i>
        ónze
       </i>
       is merely an emphasized form of
       <i>
        onze
       </i>
       . Also in family names like Piét, Piél, Plusjé, Hofsté.
      </p>
      <p>
       In
       <a href="/wiki/Danish_language" title="Danish language">
        Danish
       </a>
       , the acute accent can also be used for
       <a href="/wiki/Stress_(linguistics)" title="Stress (linguistics)">
        emphasis
       </a>
       , especially on the word
       <i>
        der
       </i>
       (there), as in
       <i>
        Der kan ikke være mange mennesker dér
       </i>
       , meaning "There can't be many people
       <i>
        there
       </i>
       " or
       <i>
        Dér skal vi hen
       </i>
       meaning "
       <i>
        That's
       </i>
       where we're going".
      </p>

          </a>

          <i>

          </i>

          <a href="/wiki/Open_front_unrounded_vowel" title="Open front unrounded vowel">

         </dd>

      <h3>
       <span class="mw-headline" id="Other_uses">
        Other uses
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Acute_accent&amp;action=edit&amp;section=15" title="Edit section: Other uses">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <ul>
       <li>
        In
        <a href="/wiki/Emilian-Romagnol_language" title="Emilian-Romagnol language">
         Emilian-Romagnol
        </a>
        ,
        <i>
         é ó
        </i>
        denote both length and height. In
        <a href="/wiki/Romagnol_dialect" title="Romagnol dialect">
         Romagnol
        </a>
        they represent [eː, oː], while in
        <a href="/wiki/Emilian_dialect" title="Emilian dialect">
         Emilian
        </a>
        they represent [e, o].
       </li>
       <li>
        Many
        <a href="/wiki/Norwegian_language" title="Norwegian language">
         Norwegian
        </a>
        words of French origin retain an acute accent, such as
        <i>
         allé
        </i>
        ,
        <i>
         kafé
        </i>
        ,
        <i>
         idé
        </i>
        ,
        <i>
         komité
        </i>
        . Popular usage can be sketchy and often neglects the accent, or results in the grave accent erroneously being used in its place. Likewise, in
        <a href="/wiki/Swedish_language" title="Swedish language">
         Swedish
        </a>
        , the acute accent is used only for the letter
        <i>
         e
        </i>
        , mostly in words of French origin and in some names. It is used both to indicate a change in vowel quantity as well as quality and that the stress should be on this, normally unstressed, syllable. Examples include
        <i>
         café
        </i>
        ("café") and
        <i>
         resumé
        </i>
        ("résumé", noun). There are two pairs of
        <a href="/wiki/Homograph" title="Homograph">
         homographs
        </a>
        that are differentiated only by the accent:
        <i>
         armé
        </i>
        ("army") versus
        <i>
         arme
        </i>
        ("poor; pitiful", masculine gender) and
        <i>
         idé
        </i>
        ("idea") versus
        <i>
         ide
        </i>
        ("winter quarters").
       </li>
       <li>
        <i>
         Ǵǵ
        </i>
        and
        <i>
         Źź
        </i>
        are used in
        <a class="mw-redirect" href="/wiki/Pashto_language" title="Pashto language">
         Pashto
        </a>
        in the Latin alphabet, equivalent to ږ and ځ, respectively.
       </li>
       <li>
        In
        <a class="mw-redirect" href="/wiki/Northern_S%C3%A1mi" title="Northern Sámi">
         Northern Sámi
        </a>
        , an acute accent was placed over the corresponding
        <a href="/wiki/Latin_alphabet" title="Latin alphabet">
         Latin
        </a>
        letter to represent the letters peculiar to this language (
        <i>
         Áá, Čč, Đđ, Ŋŋ, Šš, Ŧŧ, Žž
        </i>
        ) when typing when there was no way of entering these letters correctly otherwise.
        <sup class="reference" id="cite_ref-Svonni_7-0">
         <a href="#cite_note-Svonni-7">
          [7]
         </a>
        </sup>
       </li>
       <li>
        In transliterating texts written in
        <a href="/wiki/Cuneiform_script" title="Cuneiform script">
         Cuneiform
        </a>
        , an acute accent over the vowel indicates that the original sign is the second representing that value in the canonical lists. Thus
        <i>
         su
        </i>
        is used to transliterate the first sign with the phonetic value
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         /su/
        </span>
        , while
        <i>
         sú
        </i>
        transliterates the second sign with the value
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         /su/
        </span>
        .
       </li>
       <li>
        In some
        <a href="/wiki/Basque_language" title="Basque language">
         Basque
        </a>
        texts predating
        <a href="/wiki/Standard_Basque" title="Standard Basque">
         Standard Basque
        </a>
        , the letters
        <b>
         r
        </b>
        and
        <b>
         l
        </b>
        carry acute accents (an invention by
        <a href="/wiki/Sabino_Arana" title="Sabino Arana">
         Sabino Arana
        </a>
        <sup class="reference" id="cite_ref-Trask_8-0">
         <a href="#cite_note-Trask-8">
          [8]
         </a>
        </sup>
        ), which are otherwise indicated by double letters. In such cases,
        <i>
         <b>
          ŕ
         </b>
        </i>
        is used to represent
        <i>
         <b>
          rr
         </b>
        </i>
        (a trilled r, this spelling is used even at the end of a syllable,
        <sup class="reference" id="cite_ref-Arana_9-0">
         <a href="#cite_note-Arana-9">
          [9]
         </a>
        </sup>
        to differentiate from -
        <i>
         <b>
          r
         </b>
        </i>
        -, an alveolar tap–in Basque
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         /r/
        </span>
        in word-final positions is always trilled) and
        <i>
         <b>
          ĺ
         </b>
        </i>
        for
        <i>
         <b>
          ll
         </b>
        </i>
        (a palatalized
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         /l/
        </span>
        ).
       </li>
       <li>
        In
        <a href="/wiki/Indonesian_language" title="Indonesian language">
         Indonesian
        </a>
        dictionaries,
        <i>
         <b>
          é
         </b>
        </i>
        is used to represent
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         /e/
        </span>
        , while
        <i>
         <b>
          e
         </b>
        </i>
        is used to represent
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         /ə/
        </span>
        .
       </li>
      </ul>
      <h3>
       <span class="mw-headline" id="English">
        English
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Acute_accent&amp;action=edit&amp;section=16" title="Edit section: English">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       As with other diacritical marks, a number of (usually
       <a href="/wiki/French_language" title="French language">
        French
       </a>
       )
       <a class="mw-redirect" href="/wiki/Loanwords" title="Loanwords">
        loanwords
       </a>
       are sometimes spelled in English with an acute accent as used in the original language: these include
       <i>
        attaché
       </i>
       ,
       <i>
        blasé
       </i>
       ,
       <i>
        canapé
       </i>
       ,
       <i>
        cliché
       </i>
       ,
       <i>
        communiqué
       </i>
       ,
       <i>
        café
       </i>
       ,
       <i>
        décor
       </i>
       ,
       <i>
        déjà vu
       </i>
       ,
       <i>
        détente
       </i>
       ,
       <i>
        élite
       </i>
       ,
       <i>
        entrée
       </i>
       ,
       <i>
        exposé
       </i>
       ,
       <i>
        mêlée
       </i>
       ,
       <i>
        fiancé
       </i>
       ,
       <i>
        fiancée
       </i>
       ,
       <i>
        papier-mâché
       </i>
       ,
       <i>
        passé
       </i>
       ,
       <i>
        pâté
       </i>
       ,
       <i>
        piqué
       </i>
       ,
       <i>
        plié
       </i>
       ,
       <i>
        repoussé
       </i>
       ,
       <i>
        résumé
       </i>
       ,
       <i>
        risqué
       </i>
       ,
       <i>
        sauté
       </i>
       ,
       <i>
        roué
       </i>
       ,
       <i>
        séance
       </i>
       ,
       <i>
        naïveté
       </i>
       ,
       <i>
        toupée
       </i>
       and
       <i>
        touché
       </i>
       . Retention of the accent is common only in the French ending
       <i>
        é
       </i>
       or
       <i>
        ée
       </i>
       , as in these examples, where its absence would tend to suggest a different pronunciation. Thus the French word
       <i>
        résumé
       </i>
       is commonly seen in English as
       <i>
        resumé
       </i>
       , with only one accent (but also with both or none).
      </p>
      <p>
       Acute accents are sometimes added to loanwords where a final
       <i>
        e
       </i>
       is not
       <a href="/wiki/Silent_e" title="Silent e">
        silent
       </a>
       , for example,
       <i>
        <a href="/wiki/Yerba_mate" title="Yerba mate">
         maté
        </a>
       </i>
       from Spanish
       <i>
        mate,
       </i>
       the Maldivian capital
       <i>
        <a href="/wiki/Mal%C3%A9" title="Malé">
         Malé
        </a>
        ,
       </i>
       <i>
        <a class="mw-redirect" href="/wiki/Sak%C3%A9" title="Saké">
         saké
        </a>
        ,
       </i>
       and
       <i>
        <a href="/wiki/Pok%C3%A9mon" title="Pokémon">
         Pokémon
        </a>
       </i>
       from the Japanese compound for
       <i>
        pocket monster,
       </i>
       the last three from languages which do not use the Roman alphabet, and where transcriptions do not normally use acute accents.
      </p>
      <p>
       For foreign terms used in English that have not been assimilated into English or are not in general English usage,
       <a href="/wiki/Italic_type" title="Italic type">
        italics
       </a>
       are generally used with the appropriate accents: for example,
       <i>
        <a href="/wiki/Coup_d%27%C3%A9tat" title="Coup d'état">
         coup d'état
        </a>
       </i>
       ,
       <i>
        <a href="/wiki/Pi%C3%A8ce_de_r%C3%A9sistance" title="Pièce de résistance">
         pièce de résistance
        </a>
       </i>
       ,
       <i>
        <a href="/wiki/Cr%C3%A8me_br%C3%BBl%C3%A9e" title="Crème brûlée">
         crème brûlée
        </a>
       </i>
       and
       <i>
        <a class="mw-redirect" href="/wiki/Ancien_r%C3%A9gime" title="Ancien régime">
         ancien régime
        </a>
       </i>
       .
      </p>
      <p>
       The acute accent is sometimes (though rarely) used for poetic purposes:
      </p>
      <ul>
       <li>
        It can mark stress on an unusual syllable: for example,
        <i>
         caléndar
        </i>
        to indicate
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         [kəˈlɛn.dɚ]
        </span>
        (rather than the standard
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         [ˈkæl.ən.dɚ]
        </span>
        ).
       </li>
       <li>
        It can disambiguate stress where the distinction is metrically important: for example,
        <i>
         rébel
        </i>
        (as opposed to
        <i>
         rebél
        </i>
        ), or
        <i>
         áll trádes
        </i>
        , to show that the phrase is pronounced as a
        <a href="/wiki/Spondee" title="Spondee">
         spondee
        </a>
        , rather than the more natural
        <a href="/wiki/Iamb_(poetry)" title="Iamb (poetry)">
         iamb
        </a>
        .
       </li>
       <li>
        It can indicate the sounding of an ordinarily silent letter: for example,
        <i>
         pickéd
        </i>
        to indicate the pronunciation
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         [ˈpɪkɪd]
        </span>
        , rather than standard
        <span class="IPA" title="Representation in the International Phonetic Alphabet (IPA)">
         [pɪkt]
        </span>
        (the
        <a href="/wiki/Grave_accent" title="Grave accent">
         grave accent
        </a>
        is more common for this last purpose).
       </li>
      </ul>
      <p>
       The layout of some European PC keyboards, combined with problematic keyboard-driver semantics, causes some users to use an acute accent or a grave accent instead of an apostrophe when typing in English (e.g. typing John`s or John´s instead of John's).
       <sup class="reference" id="cite_ref-10">
        <a href="#cite_note-10">
         [10]
        </a>
       </sup>
      </p>

      <h3>
       <span class="mw-headline" id="Microsoft_Windows">
        Microsoft Windows
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Acute_accent&amp;action=edit&amp;section=18" title="Edit section: Microsoft Windows">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       On Windows computers, letters with acute accents can be created by holding down the alt key and typing in a three-number code on the number pad to the right of the keyboard before releasing the Alt key. Before the appearance of Spanish keyboards, Spanish speakers had to learn these codes if they wanted to be able to write acute accents, though some preferred using the Microsoft Word spell checker to add the accent for them. Some young computer users got in the habit of not writing accented letters at all.
       <sup class="reference" id="cite_ref-11">
        <a href="#cite_note-11">
         [11]
        </a>
       </sup>
       The codes (which come from the
       <a href="/wiki/Code_page_437" title="Code page 437">
        IBM PC encoding
       </a>
       ) are:
      </p>
      <ul>
       <li>
        160 for á
       </li>
       <li>
        130 for é
       </li>
       <li>
        161 for í
       </li>
       <li>
        162 for ó
       </li>
       <li>
        163 for ú
       </li>
      </ul>
      <p>
       On some non-US keyboard layouts (e.g. Hiberno-English), these letters can also be made by holding Ctrl+Alt (or Alt Gr) and the desired letter.
      </p>
      <h4>
       <span class="mw-headline" id="Microsoft_Office">
        Microsoft Office
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Acute_accent&amp;action=edit&amp;section=19" title="Edit section: Microsoft Office">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <p>
       To input an accented letter in a
       <a href="/wiki/Microsoft_Office" title="Microsoft Office">
        Microsoft Office
       </a>
       software (Word, Powerpoint, Excel, Access, etc.), hold the Ctrl key, press the apostrophe (') key once, release the Ctrl key, and then press the desired letter.
      </p>

      <h3>
       <span class="mw-headline" id="Keyboards">
        Keyboards
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Acute_accent&amp;action=edit&amp;section=21" title="Edit section: Keyboards">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       Because
       <a class="mw-redirect" href="/wiki/Keyboard_(computing)" title="Keyboard (computing)">
        keyboards
       </a>
       have only a limited number of keys, English keyboards do not have keys for accented characters. The concept of
       <a href="/wiki/Dead_key" title="Dead key">
        dead key
       </a>
       , a key that modified the meaning of the next key press, was developed to overcome this problem. This acute accent key was already present on typewriters where it typed the accent without moving the carriage, so a normal letter could be written on the same place.
      </p>

       Some sites, such as Wikipedia or the

       <sup class="reference" id="cite_ref-12">

      </p>
      <h3>
       <span class="mw-headline" id="Limitations">
        Limitations
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Acute_accent&amp;action=edit&amp;section=23" title="Edit section: Limitations">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       In the
       <a href="/wiki/Dutch_language" title="Dutch language">
        Dutch language
       </a>
       , emphasis is expressed with an acute accent, and accents should be used on both letters of a compound vowel. However, some words when emphasized, such as
       <i>
        blíj́f
       </i>
       ,
       <i>
        míj́
       </i>
       ,
       <i>
        zíj́
       </i>
       , and
       <i>
        wíj́ten
       </i>
       , require an accent on the j, which is often omitted for technical convenience, giving
       <i>
        blíjf
       </i>
       ,
