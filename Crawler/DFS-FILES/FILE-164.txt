      <p>
       A
       <b>
        spatial reference system
       </b>
       (
       <b>
        SRS
       </b>
       ) or
       <b>
        coordinate reference system
       </b>
       (
       <b>
        CRS
       </b>
       ) is a
       <a href="/wiki/Coordinate_system" title="Coordinate system">
        coordinate-based
       </a>
       local, regional or global system used to locate geographical entities. A spatial reference system defines a specific
       <a href="/wiki/Map_projection" title="Map projection">
        map projection
       </a>
       , as well as transformations between different spatial reference systems. Spatial reference systems are defined by the
       <a href="/wiki/Open_Geospatial_Consortium" title="Open Geospatial Consortium">
        OGC
       </a>
       's
       <a class="mw-redirect" href="/wiki/Simple_feature_access" title="Simple feature access">
        Simple feature access
       </a>
       using
       <a href="/wiki/Well-known_text" title="Well-known text">
        well-known text
       </a>
       , and support has been implemented by several
       <a href="/wiki/Technical_standard" title="Technical standard">
        standards-based
       </a>
       <a href="/wiki/Geographic_information_system" title="Geographic information system">
        geographic information systems
       </a>
       . Spatial reference systems can be referred to using a
       <a class="mw-redirect" href="/wiki/SRID" title="SRID">
        SRID
       </a>
       <a href="/wiki/Integer" title="Integer">
        integer
       </a>
       , including EPSG codes defined by the
       <a class="mw-redirect" href="/wiki/International_Association_of_Oil_and_Gas_Producers" title="International Association of Oil and Gas Producers">
        International Association of Oil and Gas Producers
       </a>
       . It is specified in ISO 19111:2007 Geographic information—Spatial referencing by coordinates, also published as
       <a href="/wiki/Open_Geospatial_Consortium" title="Open Geospatial Consortium">
        OGC
       </a>
       Abstract Specification, Topic 2: Spatial referencing by coordinate.
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
      </p>

      <h2>
       <span class="mw-headline" id="Identifiers">
        Identifiers
        <span id="Identifier">
        </span>
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Spatial_reference_system&amp;action=edit&amp;section=2" title="Edit section: Identifiers">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       A
       <b>
        Spatial Reference System Identifier
       </b>
       (
       <b>
        SRID
       </b>
       ) is a unique value used to unambiguously identify projected, unprojected, and local spatial coordinate system definitions. These coordinate systems form the heart of all
       <a href="/wiki/Geographic_information_system" title="Geographic information system">
        GIS
       </a>
       applications.
      </p>
      <p>
       Virtually all major spatial vendors have created their own SRID implementation or refer to those of an authority, such as the
       <a class="mw-redirect" href="/wiki/EPSG" title="EPSG">
        European Petroleum Survey Group (EPSG)
       </a>
       . (NOTE: As of 2005 the EPSG SRID values are now maintained by the International Association of
       <a class="external text" href="http://www.epsg.org/" rel="nofollow">
        Oil &amp; Gas Producers (OGP) Surveying &amp; Positioning Committee
       </a>
       ).
      </p>
      <p>
       SRIDs are the primary key for the
       <a href="/wiki/Open_Geospatial_Consortium" title="Open Geospatial Consortium">
        Open Geospatial Consortium (OGC)
       </a>
       <b>
        spatial_ref_sys
       </b>
       metadata table for the
       <a href="/wiki/Simple_Features" title="Simple Features">
        Simple Features for SQL Specification, Versions 1.1 and 1.2
       </a>
       , which is defined as follows:
      </p>
      <div class="mw-highlight mw-content-ltr" dir="ltr">
       <pre>
<span class="k">CREATE</span> <span class="k">TABLE</span> <span class="n">SPATIAL_REF_SYS</span>
<span class="p">(</span>
    <span class="n">SRID</span>      <span class="nb">INTEGER</span>   <span class="k">NOT</span> <span class="k">NULL</span> <span class="k">PRIMARY</span> <span class="k">KEY</span><span class="p">,</span>
    <span class="n">AUTH_NAME</span> <span class="nb">CHARACTER</span> <span class="nb">VARYING</span><span class="p">(</span><span class="mi">256</span><span class="p">),</span>
    <span class="n">AUTH_SRID</span> <span class="nb">INTEGER</span><span class="p">,</span>
    <span class="n">SRTEXT</span>    <span class="nb">CHARACTER</span> <span class="nb">VARYING</span><span class="p">(</span><span class="mi">2048</span><span class="p">)</span>
<span class="p">)</span>
</pre>
      </div>
      <p>
       In spatially enabled databases (such as
       <a class="mw-redirect" href="/wiki/IBM_DB2" title="IBM DB2">
        IBM DB2
       </a>
       ,
       <a href="/wiki/IBM_Informix" title="IBM Informix">
        IBM Informix
       </a>
       ,
       <a href="/wiki/Microsoft_SQL_Server" title="Microsoft SQL Server">
        Microsoft SQL Server
       </a>
       ,
       <a href="/wiki/MySQL" title="MySQL">
        MySQL
       </a>
       ,
       <a class="mw-redirect" href="/wiki/Oracle_RDBMS" title="Oracle RDBMS">
        Oracle RDBMS
       </a>
       ,
       <a href="/wiki/Teradata" title="Teradata">
        Teradata
       </a>
       ,
       <a href="/wiki/PostGIS" title="PostGIS">
        PostGIS
       </a>
       and
       <a href="/wiki/SQL_Anywhere" title="SQL Anywhere">
        SQL Anywhere
       </a>
       ), SRIDs are used to uniquely identify the coordinate systems used to define columns of spatial data or individual spatial objects in a spatial column (depending on the spatial implementation). SRIDs are typically associated with a well known text (WKT) string definition of the coordinate system (SRTEXT, above). From the
       <a href="/wiki/Well-known_text#Spatial_reference_systems" title="Well-known text">
        Well Known Text
       </a>
       Wikipedia page, “A WKT string for a spatial reference system describes the
       <a class="mw-redirect" href="/wiki/Datum_(geodesy)" title="Datum (geodesy)">
        datum
       </a>
       ,
       <a href="/wiki/Geoid" title="Geoid">
        geoid
       </a>
       ,
       <a href="/wiki/Coordinate_system" title="Coordinate system">
        coordinate system
       </a>
       , and
       <a href="/wiki/Map_projection" title="Map projection">
        map projection
       </a>
       of the spatial objects”. Here are two common coordinate systems with their EPSG SRID value followed by their well known text:
      </p>
      <p>
       UTM, Zone 17N, NAD27 — SRID 2029:
      </p>
      <div class="mw-highlight mw-content-ltr" dir="ltr">
       <pre>

    <span class="nx">GEOGCS</span><span class="p">[</span><span class="s2">"NAD27(76)"</span><span class="p">,</span>

        <span class="nx">PRIMEM</span><span class="p">[</span><span class="s2">"Greenwich"</span><span class="p">,</span><span class="mi">0</span><span class="p">,</span>
            <span class="nx">AUTHORITY</span><span class="p">[</span><span class="s2">"EPSG"</span><span class="p">,</span><span class="s2">"8901"</span><span class="p">]],</span>

            <span class="nx">AUTHORITY</span><span class="p">[</span><span class="s2">"EPSG"</span><span class="p">,</span><span class="s2">"9122"</span><span class="p">]],</span>
        <span class="nx">AUTHORITY</span><span class="p">[</span><span class="s2">"EPSG"</span><span class="p">,</span><span class="s2">"4608"</span><span class="p">]],</span>

      <p>
       SRID values associated with spatial data can be used to constrain spatial operations — for instance, spatial operations cannot be performed between spatial objects with differing SRIDs in some systems, or trigger coordinate system transformations between spatial objects in others.
