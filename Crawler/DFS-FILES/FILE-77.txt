   <h1 class="firstHeading" id="firstHeading" lang="en">
    Fundamental plane (spherical coordinates)
   </h1>
   <div class="mw-body-content" id="bodyContent">
    <div class="noprint" id="siteSub">
     From Wikipedia, the free encyclopedia
    </div>
    <div id="contentSub">
    </div>
    <div class="mw-jump" id="jump-to-nav">

     <a href="#mw-head">

    <div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en">
     <div class="mw-parser-output">
      <div class="hatnote navigation-not-searchable" role="note">
       For the concept in extragalactic astronomy, see
       <a href="/wiki/Fundamental_plane_(elliptical_galaxies)" title="Fundamental plane (elliptical galaxies)">
        Fundamental plane (elliptical galaxies)
       </a>
       .
      </div>
      <p>
       The
       <b>
        fundamental plane
       </b>
       in a
       <a href="/wiki/Spherical_coordinate_system" title="Spherical coordinate system">
        spherical coordinate system
       </a>
       is a
       <a href="/wiki/Plane_of_reference" title="Plane of reference">
        plane of reference
       </a>
       that divides the
       <a href="/wiki/Sphere" title="Sphere">
        sphere
       </a>
       into two
       <a href="/wiki/Sphere" title="Sphere">
        hemispheres
       </a>
       . The
       <a href="/wiki/Latitude" title="Latitude">
        latitude
       </a>
       of a
       <a href="/wiki/Point_(geometry)" title="Point (geometry)">
        point
       </a>
       is then the
       <a href="/wiki/Angle" title="Angle">
        angle
       </a>
       between the fundamental plane and the line joining the point to the centre of the sphere.
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
      </p>
      <p>
       For a
       <a href="/wiki/Geographic_coordinate_system" title="Geographic coordinate system">
        geographic coordinate system
       </a>
       of the Earth, the fundamental plane is the
       <a href="/wiki/Equator" title="Equator">
        Equator
       </a>
       .
       <a href="/wiki/Celestial_coordinate_system" title="Celestial coordinate system">
        Celestial coordinate systems
       </a>
       have varying fundamental planes:
       <sup class="reference" id="cite_ref-2">
        <a href="#cite_note-2">
         [2]
        </a>
       </sup>
      </p>
      <ul>
       <li>
        The
        <a href="/wiki/Equatorial_coordinate_system" title="Equatorial coordinate system">
         equatorial coordinate system
        </a>
        uses the
        <a href="/wiki/Celestial_equator" title="Celestial equator">
         celestial equator
        </a>
        .
       </li>
       <li>
        The
        <a href="/wiki/Ecliptic_coordinate_system" title="Ecliptic coordinate system">
         ecliptic coordinate system
        </a>
        uses the
        <a href="/wiki/Ecliptic" title="Ecliptic">
         ecliptic
        </a>
        .
       </li>
       <li>
        The
        <a href="/wiki/Galactic_coordinate_system" title="Galactic coordinate system">
         galactic coordinate system
        </a>
        uses the
        <a href="/wiki/Milky_Way" title="Milky Way">
         Milky Way
        </a>
        's glactic equator.
       </li>
       <li>
        The
        <a href="/wiki/Horizontal_coordinate_system" title="Horizontal coordinate system">
         horizontal coordinate system
        </a>
        uses the observer's
        <a href="/wiki/Horizon" title="Horizon">
         horizon
        </a>
        .
       </li>
       <li>
        The Besselian coordinate system uses Earth's
        <a href="/wiki/Terminator_(solar)" title="Terminator (solar)">
         terminator
        </a>
        (day/night boundary). This is a
        <a href="/wiki/Cartesian_coordinate_system" title="Cartesian coordinate system">
         Cartesian coordinate system
        </a>
        (
        <i>
         x
        </i>
        ,
        <i>
         y
        </i>
        ,
        <i>
         z
        </i>
        ).
       </li>
      </ul>
