   <h1 class="firstHeading" id="firstHeading" lang="en">
    Accusative case
   </h1>
   <div class="mw-body-content" id="bodyContent">
    <div class="noprint" id="siteSub">
     From Wikipedia, the free encyclopedia
    </div>
    <div id="contentSub">
    </div>
    <div class="mw-jump" id="jump-to-nav">
     Jump to:
     <a href="#mw-head">
      navigation
     </a>
     ,
     <a href="#p-search">
      search
     </a>
    </div>
    <div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en">
     <div class="mw-parser-output">
      <p>
       The
       <b>
        accusative case
       </b>
       (
       <a href="/wiki/List_of_glossing_abbreviations" title="List of glossing abbreviations">
        abbreviated
       </a>
       <span class="smallcaps" style="font-variant:small-caps;">
        <b>
         acc
        </b>
       </span>
       ) of a
       <a href="/wiki/Noun" title="Noun">
        noun
       </a>
       is the
       <a href="/wiki/Grammatical_case" title="Grammatical case">
        grammatical case
       </a>
       used to mark the
       <a class="mw-redirect" href="/wiki/Direct_object" title="Direct object">
        direct object
       </a>
       of a
       <a href="/wiki/Transitive_verb" title="Transitive verb">
        transitive verb
       </a>
       . The same case is used in many languages for the objects of (some or all)
       <a class="mw-redirect" href="/wiki/Preposition" title="Preposition">
        prepositions
       </a>
       . It is a noun that is having something done to it, usually used together (such as in
       <a href="/wiki/Latin" title="Latin">
        Latin
       </a>
       ) with the
       <a href="/wiki/Nominative_case" title="Nominative case">
        nominative case
       </a>
       . For example, "they" in English is nominative; "them" is accusative. The sentence "They like them" shows the nominative case and accusative case working in conjunction using the same base word. The syntactic functions of the accusative consist of designating the immediate object of an action, the intended result, the goal of a motion, and the extent of an action.
       <sup class="reference" id="cite_ref-Accusative_case_and_its_various_substitutes_in_language_1-0">
        <a href="#cite_note-Accusative_case_and_its_various_substitutes_in_language-1">
         [1]
        </a>
       </sup>
      </p>
      <p>
       The accusative case existed in
       <a href="/wiki/Proto-Indo-European_language" title="Proto-Indo-European language">
        Proto-Indo-European
       </a>
       and is present in some
       <a href="/wiki/Indo-European_languages" title="Indo-European languages">
        Indo-European languages
       </a>
       (including
       <a href="/wiki/Latin" title="Latin">
        Latin
       </a>
       ,
       <a href="/wiki/Sanskrit" title="Sanskrit">
        Sanskrit
       </a>
       ,
       <a href="/wiki/Greek_language" title="Greek language">
        Greek
       </a>
       ,
       <a href="/wiki/German_language" title="German language">
        German
       </a>
       ,
       <a href="/wiki/Icelandic_language" title="Icelandic language">
        Icelandic
       </a>
       ,
       <a href="/wiki/Bulgarian_language" title="Bulgarian language">
        Bulgarian
       </a>
       ,
       <a href="/wiki/Polish_language" title="Polish language">
        Polish
       </a>
       ,
       <a href="/wiki/Czech_language" title="Czech language">
        Czech
       </a>
       ,
       <a href="/wiki/Serbian_language" title="Serbian language">
        Serbian
       </a>
       ,
       <a href="/wiki/Romanian_language" title="Romanian language">
        Romanian
       </a>
       ,
       <a href="/wiki/Russian_language" title="Russian language">
        Russian
       </a>
       ,
       <a href="/wiki/Ukrainian_language" title="Ukrainian language">
        Ukrainian
       </a>
       ), in the
       <a href="/wiki/Uralic_languages" title="Uralic languages">
        Uralic languages
       </a>
       , in
       <a href="/wiki/Altaic_languages" title="Altaic languages">
        Altaic languages
       </a>
       and in
       <a href="/wiki/Semitic_languages" title="Semitic languages">
        Semitic languages
       </a>
       (such as
       <a class="mw-redirect" href="/wiki/Hebrew" title="Hebrew">
        Hebrew
       </a>
       and
       <a href="/wiki/Classical_Arabic" title="Classical Arabic">
        Classical Arabic
       </a>
       ).
       <a href="/wiki/Finnic_languages" title="Finnic languages">
        Finnic languages
       </a>
       , such as
       <a href="/wiki/Finnish_language" title="Finnish language">
        Finnish
       </a>
       and
       <a href="/wiki/Estonian_language" title="Estonian language">
        Estonian
       </a>
       , have two cases to mark objects, the accusative and the
       <a href="/wiki/Partitive_case" title="Partitive case">
        partitive case
       </a>
       . In
       <a href="/wiki/Morphosyntactic_alignment" title="Morphosyntactic alignment">
        morphosyntactic alignment
       </a>
       terms, both perform the accusative function, but the accusative object is
       <a href="/wiki/Telicity" title="Telicity">
        telic
       </a>
       , while the partitive is not.
      </p>
      <p>
       Modern
       <a href="/wiki/English_language" title="English language">
        English
       </a>
       , which almost entirely lacks
       <a href="/wiki/Declension" title="Declension">
        declension
       </a>
       in its nouns, does not have an explicitly marked accusative case even in the pronouns. Such forms as
       <i>
        whom
       </i>
       ,
       <i>
        them
       </i>
       , and
       <i>
        her
       </i>
       derive rather from the old Germanic
       <a href="/wiki/Dative_case" title="Dative case">
        dative
       </a>
       forms, of which the -m and -r endings are characteristic. This conflation of the old accusative, dative, and (after prepositions) genitive cases is the
       <i>
        <a href="/wiki/Oblique_case" title="Oblique case">
         oblique case
        </a>
       </i>
       . Most modern English grammarians no longer use the Latin accusative/dative model, though they tend to use the terms
       <i>
        objective
       </i>
       for oblique,
       <i>
        subjective
       </i>
       for nominative, and
       <i>
        possessive
       </i>
       for genitive
       <i>
        (see
        <a class="mw-redirect" href="/wiki/Declension_in_English" title="Declension in English">
         declension in English
        </a>
        ).
       </i>
       <i>
        Hine
       </i>
       , a true accusative masculine third person singular pronoun, is attested in some northern English dialects as late as the 19th century.
       <sup class="reference" id="cite_ref-2">
        <a href="#cite_note-2">
         [2]
        </a>
       </sup>
      </p>

      </p>

      <h2>
       <span class="mw-headline" id="Etymology">
        Etymology
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Accusative_case&amp;action=edit&amp;section=1" title="Edit section: Etymology">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       The
       <a href="/wiki/English_language" title="English language">
        English
       </a>
       name "accusative (case)" is an
       <a href="/wiki/Anglicisation" title="Anglicisation">
        Anglicisation
       </a>
       of the
       <a href="/wiki/Latin" title="Latin">
        Latin
       </a>
       <i>
        accūsātīvus
       </i>
       (
       <i>
        cāsus
       </i>
       ),
       <sup class="reference" id="cite_ref-3">
        <a href="#cite_note-3">
         [3]
        </a>
       </sup>
       which was translated from
       <a href="/wiki/Ancient_Greek" title="Ancient Greek">
        Ancient Greek
       </a>
       αἰτιατικὴ (πτῶσις),
       <i>
        aitiatikē (ptôsis)
       </i>
       .
       <sup class="reference" id="cite_ref-4">
        <a href="#cite_note-4">
         [4]
        </a>
       </sup>
       The Greek term can mean either "(inflection) for something caused" or "for an accusation".
       <sup class="reference" id="cite_ref-5">
        <a href="#cite_note-5">
         [5]
        </a>
       </sup>
       The intended meaning was likely the first, which would be translated as Latin
       <i>
        causātīvus
       </i>
       <sup class="reference" id="cite_ref-6">
        <a href="#cite_note-6">
         [6]
        </a>
       </sup>
       or
       <i>
        effectīvus
       </i>
       ,
       <sup class="reference" id="cite_ref-7">
        <a href="#cite_note-7">
         [7]
        </a>
       </sup>
       but the Latin term was a translation of the second. Compare
       <a href="/wiki/Russian_language" title="Russian language">
        Russian
       </a>
       <a class="extiw" href="https://en.wiktionary.org/wiki/%D0%B2%D0%B8%D0%BD%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9#Russian" title="wikt:винительный">
        вини́тельный
       </a>
       <i>
        vinítel’nyj
       </i>
       , from
       <a class="extiw" href="https://en.wiktionary.org/wiki/%D0%B2%D0%B8%D0%BD%D0%B8%D1%82%D1%8C#Russian" title="wikt:винить">
        винить
       </a>
       <i>
        vinít’
       </i>
       "to blame".
      </p>
      <h2>
       <span class="mw-headline" id="Description">
        Description
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Accusative_case&amp;action=edit&amp;section=2" title="Edit section: Description">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       In the sentence
       <i>
        <b>
         He
        </b>
        sees the woman
       </i>
       , "he" is the subject of the sentence, while in
       <i>
        The woman sees
        <b>
         him
        </b>
       </i>
       , "him" is the object. In English the two uses are distinguished by different forms of the pronoun: he/him. If, however, instead of a pronoun, we use a noun, we make no such distinction in the form of the word. Thus, we use the same word "man" in both
       <i>
        <b>
         The man
        </b>
        sees the woman
       </i>
       and
       <i>
        The woman sees
        <b>
         the man
        </b>
       </i>
       . In many languages, however, different forms of the word are used not only for pronouns, but for nouns too. For example, in Latin
       <i>
        The man sees the woman
       </i>
       =
       <i>
        <b>
         Vir
        </b>
        feminam videt
       </i>
       , while
       <i>
        The woman sees the man
       </i>
       =
       <i>
        Femina
        <b>
         virum
        </b>
        videt
       </i>
       . For "man", Latin uses "vir" for the subject, and "virum" for the object. Likewise, in the same pair of sentences, we have "femina" for a subject and "feminam" for object. The form used for the direct object ("him", "virum", "feminam") is known as the "accusative case", while the form used for the subject ("he", "vir", "femina") is known as the
       <a href="/wiki/Nominative_case" title="Nominative case">
        nominative case
       </a>
       .
      </p>
      <p>
       Just as with pronouns and nouns, many inflected languages also make distinctions between cases in their adjectives and (for languages that have them) articles. Thus in German, "the giant" as the subject of a sentence may be expressed as
       <i>
        der Riese
       </i>
       : nominative case. As the object of a verb, this becomes
       <i>
        <b>
         den
        </b>
        Riesen
       </i>
       , the accusative.
      </p>

      <h4>
       <span class="mw-headline" id="Latin">
        Latin
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Accusative_case&amp;action=edit&amp;section=5" title="Edit section: Latin">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <p>
       In Latin, nouns, adjectives, or pronouns in the accusative case (
       <i>
        accusativus
       </i>
       ) can be used
      </p>
      <ul>
       <li>
        as
        <a href="/wiki/Object_(grammar)" title="Object (grammar)">
         direct object
        </a>
        .
       </li>
       <li>
        to indicate duration of time. E.g.,
        <i>
         multos annos
        </i>
        , "for many years";
        <i>
         ducentos annos
        </i>
        , "for 200 years." This is known as the
        <b>
         accusative of time
        </b>
        . Compare the
        <a class="mw-redirect" href="/wiki/Ablative_of_time" title="Ablative of time">
         ablative of time
        </a>
        , which was used for most points in time except—in classical Latin—for dates of the form
        <i>
         ante diem N. Kal./Non./Id.
        </i>
        , which followed
        <a class="mw-redirect" href="/wiki/Ancient_Greek_language" title="Ancient Greek language">
         Greek
        </a>
        in taking the accusative.
       </li>
       <li>
        to indicate direction towards which. E.g.
        <i>
         domum
        </i>
        , "homewards";
        <i>
         Romam
        </i>
        , "to Rome" with no preposition needed. This is known as the
        <b>
         accusative of place to which
        </b>
        , and is equivalent to the
        <a href="/wiki/Lative_case" title="Lative case">
         lative case
        </a>
        found in some other languages.
       </li>
       <li>
        as the subject of an
        <a class="mw-redirect" href="/wiki/Indirect_statement" title="Indirect statement">
         indirect statement
        </a>
        (e.g.
        <i>
         Dixit
         <b>
          me
         </b>
         fuisse saevum
        </i>
        , "He said that
        <b>
         I
        </b>
        had been cruel;" in later Latin works, such as the
        <a href="/wiki/Vulgate" title="Vulgate">
         Vulgate
        </a>
        , such a construction is replaced by
        <i>
         quod
        </i>
        and a regularly structured sentence, having the subject in the nominative: e.g.,
        <i>
         Dixit quod
         <b>
          ego
         </b>
         fueram saevus
        </i>
        ).
       </li>
       <li>
        with case-specific prepositions such as "per" (through), "ad" (to/toward), and "trans" (across).
       </li>
       <li>
        in exclamations, such as
        <i>
         me miseram,
        </i>
        "wretched me" (spoken by
        <a href="/wiki/Circe" title="Circe">
         Circe
        </a>
        to
        <a href="/wiki/Odysseus" title="Odysseus">
         Ulysses
        </a>
        in
        <a href="/wiki/Ovid" title="Ovid">
         Ovid
        </a>
        's
        <i>
         Remedium Amoris;
        </i>
        note that this is feminine: the masculine form would be me miser
        <ins>
         um
        </ins>
        ).
       </li>
      </ul>
      <p>
       For the accusative endings, see
       <a href="/wiki/Latin_declension" title="Latin declension">
        Latin declension
       </a>
       .
      </p>
      <h5>
       <span class="mw-headline" id="Latin_prepositions">
        Latin prepositions
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Accusative_case&amp;action=edit&amp;section=6" title="Edit section: Latin prepositions">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h5>
      <p>
       Some Latin
       <a class="mw-redirect" href="/wiki/Preposition" title="Preposition">
        prepositions
       </a>
       take a noun in the accusative. A few prepositions may take either an accusative or an
       <a href="/wiki/Ablative_case" title="Ablative case">
        ablative
       </a>
       , in which case the accusative indicates motion, and the ablative indicates no motion. E.g. (ablative)
       <i>
        in casā
       </i>
       , "in the cottage"; (accusative)
       <i>
        in casam
       </i>
       , "into the cottage".
       <sup class="reference" id="cite_ref-8">
        <a href="#cite_note-8">
         [8]
        </a>
       </sup>
      </p>
      <p>
       This
       <i>
        aide-memoire
       </i>
       was taught in schools when Latin was on the curriculum:
       <br/>
       <i>
        Ante, apud, ad, adversus,
       </i>
       <br/>
       <i>
        Circum, circa, citra, cis,
       </i>
       <br/>
       <i>
        Contra, inter, erga, extra,
       </i>
       <br/>
       <i>
        Infra, intra, iuxta, ob,
       </i>
       <br/>
       <i>
        Penes, pone, post,
       </i>
       and
       <i>
        praeter,
       </i>
       <br/>
       <i>
        Prope, propter, per, secundum,
       </i>
       <br/>
       <i>
        Supra, versus, ultra, trans:
       </i>
      </p>
      <ul>
       <li>
        <ul>
         <li>
          Add
          <i>
           super, subter, sub,
          </i>
          and
          <i>
           in,
          </i>
         </li>
        </ul>
       </li>
      </ul>
      <p>
       When 'motion' 'tis, not 'state' they mean.**
       <sup class="reference" id="cite_ref-9">
        <a href="#cite_note-9">
         [9]
        </a>
       </sup>
      </p>
      <p>
       Or try: **
       <i>
        And unto these if motion be intended,
       </i>
       <br/>
       <i>
        Let In, Sub, Super, Subter be appended
       </i>
      </p>
      <h4>
       <span class="mw-headline" id="German">
        German
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Accusative_case&amp;action=edit&amp;section=7" title="Edit section: German">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <p>
       <a href="/wiki/German_language" title="German language">
        German
       </a>
       uses the accusative to mark direct objects and objects of certain
       <a href="/wiki/Preposition_and_postposition" title="Preposition and postposition">
        prepositions
       </a>
       , or
       <a href="/wiki/Adverb" title="Adverb">
        adverbs
       </a>
       relating to time. The accusative is only marked for masculine
       <a href="/wiki/Article_(grammar)" title="Article (grammar)">
        articles
       </a>
       ,
       <a href="/wiki/Pronoun" title="Pronoun">
        pronouns
       </a>
       ,
       <a href="/wiki/Adjective" title="Adjective">
        adjectives
       </a>
       , and weak nouns.
      </p>
      <h5>
       <span class="mw-headline" id="German_articles">
        German articles
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">

        </span>
       </span>
      </h5>
      <p>
       The masculine forms for
       <a href="/wiki/German_articles" title="German articles">
        German articles
       </a>
       , e.g.,
       <i>
        der
       </i>
       (the),
       <i>
        ein
       </i>
       (a/an),
       <i>
        mein
       </i>
       (my), etc., change in the accusative case: they always end in
       <i>

       </i>
       . Feminine, neuter and plural articles do not change in the accusative.

      <h5>
       <span class="mw-headline" id="German_prepositions">
        German prepositions
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">

       </span>

      <p>
       The accusative case is also used after particular German prepositions. These include
       <i>
        bis
       </i>
       (until),
       <i>
        durch
       </i>
       (through),
       <i>
        für
       </i>
       (for),
       <i>
        gegen
       </i>
       (against),
       <i>
        ohne
       </i>
       (without),
       <i>
        um
       </i>
       (around, by), after which the accusative case is always used, and
       <i>
        an
       </i>
       (at),
       <i>
        auf
       </i>
       (on),
       <i>
        entlang
       </i>
       (along),
       <i>
        hinter
       </i>
       (behind),
       <i>
        in
       </i>
       (in, into),
       <i>
        neben
       </i>
       (beside, next to),
       <i>
        über
       </i>
       (over, across),
       <i>
        unter
       </i>
       (under, below),
       <i>
        vor
       </i>
       (in front of),
       <i>
        zwischen
       </i>
       (among, between) which can govern either the accusative or the dative. These prepositions are also used in conjunction with certain verbs, in which case it is the verb in question which governs whether the accusative or dative should be used.

       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>

        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h5>
      <p>
       In German, the accusative case is also used for some adverbial expressions, mostly temporal ones, as in "
       <i>
        Diesen Abend bleibe ich daheim
       </i>
       " (This evening I'm staying at home), where "
       <i>
        diesen Abend
       </i>
       " is marked as accusative, although not a direct object.
      </p>
      <h4>
       <span class="mw-headline" id="Russian">
        Russian
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Accusative_case&amp;action=edit&amp;section=13" title="Edit section: Russian">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <p>
       In Russian, the accusative (винительный) is used not only to indicate the direct object of an action, but also to indicate the destination or goal of motion. It is also used with some prepositions. The prepositions
       <i>
        в
       </i>
       and
       <i>
        на
       </i>
       can both take the accusative in situations where they indicate the goal of a motion.
      </p>
      <p>
       In the
       <a href="/wiki/Grammatical_gender" title="Grammatical gender">
        masculine
       </a>
       , Russian also distinguishes between
       <a href="/wiki/Animacy" title="Animacy">
        animate and inanimate
       </a>
       nouns with regard to the accusative; only the animates carry a
       <a href="/wiki/Marker_(linguistics)" title="Marker (linguistics)">
        marker
       </a>
       in this case.
      </p>
      <p>
       Over time, Russian has almost lost the real PIE accusative case, since only singular nouns ending in 'a' have a distinct form. Other words use the forms of the genitive case or the nominative case in place of the accusative, depending on their
       <a href="/wiki/Animacy" title="Animacy">
        animacy
       </a>
       .
      </p>
      <h4>
       <span class="mw-headline" id="Armenian">
        Armenian
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Accusative_case&amp;action=edit&amp;section=14" title="Edit section: Armenian">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <p>
       While the
       <a href="/wiki/Armenian_language" title="Armenian language">
        Armenian
       </a>
       dialects both have a
       <a href="/wiki/De_facto" title="De facto">
        de facto
       </a>
       accusative case, Eastern Armenian uses an accusative marker for transitive verbs
       <sup class="reference" id="cite_ref-10">
        <a href="#cite_note-10">
         [10]
        </a>
       </sup>
      </p>
      <p>
       Example:
      </p>
      <p>
       գիրք - girkh - book (Nominative)
       <br/>
       ուսուցիչ - usuchičh - teacher (Nominative)
      </p>
      <p>
       Արամը վերցրեց գիրք
       <u>
        ը
       </u>
       :
       <br/>
       Aramë verchrech girkh
       <u>
        ë
       </u>
       <br/>
       Aram took the book.
      </p>
      <p>
       Արամը սիրում է իր ուսուցչ
       <u>
        ին
       </u>
       :
       <br/>
       Aramë sirum ē ir usuchčh
       <u>
        in
       </u>
       <br/>
       Aram loves his teacher.
      </p>
      <h4>
       <span class="mw-headline" id="Greek">
        Greek
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Accusative_case&amp;action=edit&amp;section=15" title="Edit section: Greek">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <p>
       In both
       <a href="/wiki/Ancient_Greek_grammar" title="Ancient Greek grammar">
        Ancient
       </a>
       and
       <a href="/wiki/Modern_Greek_grammar" title="Modern Greek grammar">
        Modern Greek
       </a>
       , nouns, adjectives, verb participles, articles and pronouns are used in the accusative case, when they indicate a direct object or if they are preceded by a preposition. There is a wide variety of accusative markers depending on gender, number and declension. Like in Latin, all neuter names yield the same form in both the nominative and the accusative case in Ancient Greek. In its modern successor, this rule also extends to most feminine nouns, except these ending to -ος.
      </p>
      <p>
       Example: "
       <i>
        He was also calumniating
        <a href="/wiki/Socrates" title="Socrates">
         Socrates
        </a>
        .
       </i>
       "
      </p>
      <ul>
       <li>
        In Ancient Greek:
        <i>
         "Διέβαλλε δὲ καὶ
         <b>
          τὸν
         </b>
         Σωκράτ
         <b>
          ην
         </b>
         ." (Diéballe de kai
         <b>
          ton
         </b>
         Sōkrát
         <b>
          ēn
         </b>
         .)
        </i>
       </li>
       <li>
        In Modern Greek:
        <i>
         "Διέβαλλε και
         <b>
          τον
         </b>
         Σωκράτ
         <b>
          η
         </b>
         ." (Diévalle ke
         <b>
          ton
         </b>
         Sokrát
         <b>
          i
         </b>
         .)
        </i>
       </li>
      </ul>

       <span class="mw-headline" id="Constructed_languages">
        Constructed languages

       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Accusative_case&amp;action=edit&amp;section=16" title="Edit section: Constructed languages">

      <h4>
       <span class="mw-headline" id="Esperanto">
        Esperanto
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Accusative_case&amp;action=edit&amp;section=17" title="Edit section: Esperanto">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <p>
       <a href="/wiki/Esperanto_grammar" title="Esperanto grammar">
        Esperanto grammar
       </a>
       involves only two cases, a nominative and an accusative. The accusative is formed by the addition of
       <i>
        -n
       </i>
       to the nominative form, and is the case used for direct objects. (Example:
       <i>
        knabo
       </i>
       ("boy") →
       <i>
        knabon
       </i>
       ("boy.ACC")) Other case functions, including dative functions, are achieved with prepositions, all of which normally take the nominative case. Direction of motion can be expressed either by the accusative case, or by the preposition
       <i>
        al
       </i>
       (to) with the nominative.
      </p>
      <h4>
       <span class="mw-headline" id="Ido">
        Ido
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Accusative_case&amp;action=edit&amp;section=18" title="Edit section: Ido">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <p>
       In
       <a class="mw-redirect" href="/wiki/Ido_(language)" title="Ido (language)">
        Ido
       </a>
       the
       <i>
        -n
       </i>
       suffix is optional, as
       <a href="/wiki/Subject%E2%80%93verb%E2%80%93object" title="Subject–verb–object">
        subject–verb–object
       </a>
       order is assumed when it is not present. Note that this is sometimes done in Esperanto
       <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">
        [
        <i>
         <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">
          <span title="This claim needs references to reliable sources. (June 2015)">
           citation needed
          </span>
         </a>
        </i>
        ]
       </sup>
       , especially by beginners, but it is considered incorrect while in Ido it is the norm.
      </p>
      <h3>
       <span class="mw-headline" id="Uralic_languages">
        Uralic languages
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Accusative_case&amp;action=edit&amp;section=19" title="Edit section: Uralic languages">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <h4>
       <span class="mw-headline" id="Finnish">
        Finnish
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Accusative_case&amp;action=edit&amp;section=20" title="Edit section: Finnish">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <p>
       According to traditional Finnish grammars, the accusative is the case of a total object, while the case of a partial object is the
       <a href="/wiki/Partitive_case" title="Partitive case">
        partitive
       </a>
       . The negative forms of verbs always take the partial object, whereas in positive sentences it depends on the nature of the action, the main rule being that incomplete or indefinite action requires a partial object.
      </p>
      <p>
       The accusative singular is identical either to the
       <a href="/wiki/Nominative_case" title="Nominative case">
        nominative
       </a>
       (often called nominative-accusative) or the
       <a href="/wiki/Genitive_case" title="Genitive case">
        genitive
       </a>
       (genitive-accusative). In plural, only nominative-accusative exists. The active verb forms usually require the total object in the genitive-accusative and passive forms take the nominative-accusative. The only exceptions to this rule are imperative first and second persons, and the rarely used third infinitive in instructive, which take the total object in nominative-accusative.
      </p>
      <p>
       The
       <a href="/wiki/Personal_pronoun" title="Personal pronoun">
        personal pronouns
       </a>
       and the personal
       <a class="mw-redirect" href="/wiki/Interrogative_pronoun" title="Interrogative pronoun">
        interrogative pronoun
       </a>
       <i>
        kuka/ken
       </i>
       have a special accusative form ending in
       <i>
        -t
       </i>
       which is used in place of both nominative-accusative and genitive-accusative. For example, the accusative form of
       <i>
        hän
       </i>
       (he/she) is
       <i>
        hänet
       </i>
       , and the accusative form of
       <i>
        kuka
       </i>
       (or
       <i>
        ken
       </i>
       ) is
       <i>
        kenet
       </i>
       .
      </p>
      <p>
       The major new Finnish grammar,
       <i>
        <a href="/wiki/Iso_suomen_kielioppi" title="Iso suomen kielioppi">
         Iso suomen kielioppi
        </a>
       </i>
       , breaks with the traditional classification by limiting the accusative case to the special case of the personal pronouns and
       <i>
        kuka/ken
       </i>
       . The new grammar considers other total objects as being in the nominative or genitive case.
      </p>
      <h4>
       <span class="mw-headline" id="Hungarian">
        Hungarian
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Accusative_case&amp;action=edit&amp;section=21" title="Edit section: Hungarian">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <p>
       The accusative case in Hungarian applies to nouns, pronouns; even to adjectives and numerals when either of them stands alone in the sense of direct object.
      </p>
      <p>
       Accusative is formed by the suffix
       <i>
        <b>
         -t
        </b>
       </i>
       . In many cases,
       <i>
        -t
       </i>
       is preceded by a suffix-initial vowel, primarily based on specific
       <a href="/wiki/Vowel_harmony" title="Vowel harmony">
        vowel harmony
       </a>
       , resulting in
       <i>
        -et
       </i>
       ,
       <i>
        -ot
       </i>
       , or
       <i>
        -öt
       </i>
       . The rules are complex, also involve consonants, and have exceptions. Thus: k
       <i>
        e
       </i>
       rt
       <i>
        et
       </i>
       "garden", k
       <i>
        é
       </i>
       k
       <i>
        et
       </i>
       "blue"; h
       <i>
        a
       </i>
       t
       <i>
        o
       </i>
       t "six"; p
       <i>
        o
       </i>
       lc
       <i>
        ot
       </i>
       "shelf"; k
       <i>
        ö
       </i>
       d
       <i>
        öt
       </i>
       "fog".
      </p>
      <p>
       In some words, a low vowel
       <i>
        a
       </i>
       or
       <i>
        e
       </i>
       appears instead of the expected harmonic vowel: e.g. fal
       <i>
        at
       </i>
       (ˣfal
       <i>
        ot
       </i>
       ) "wall"; nyolc
       <i>
        at
       </i>
       (ˣnyolc
       <i>
        ot
       </i>
       ) "eight"; könyv
       <i>
        et
       </i>
       (ˣkönyv
       <i>
        öt
       </i>
       ) "book".
      </p>
      <p>
       In fewer cases, the root of the word is also affected. Word endings
       <i>
        -a
       </i>
       or
       <i>
        -e
       </i>
       will (even if they are the endings of a preceding suffix) change to
       <i>
        -á
       </i>
       and
       <i>
        -é
       </i>
       , respectively, before
       <i>
        -t
       </i>
       . E.g.: f
       <i>
        a
       </i>
       (tree) -&gt; f
       <i>
        <b>
         át
        </b>
       </i>
       . The long vowel of a one-syllable word may get shortened. E.g.: úr (lord) -&gt;
       <i>
        u
       </i>
       r
       <i>
        <b>
         at
        </b>
       </i>
       . But: b
       <i>
        ú
       </i>
       r (Boer) -&gt; b
       <i>
        ú
       </i>
       r
       <i>
        <b>
         t
        </b>
       </i>
       . If a word has more than one syllable and the last syllable ends in a consonant, the vowel of the last syllable may drop. E.g.: köröm (fingernail) -&gt;
       <i>
        körm
        <b>
         öt
        </b>
       </i>
       . But: kör
       <i>
        öm
       </i>
       (
       <i>
        my
       </i>
       circle) -&gt; köröm
       <i>
        <b>
         et
        </b>
       </i>
       . Notably, the first-person and second-person personal pronouns have quite unique accusative forms (indeed, as indicated in the table, in the singular case the ending
       <i>
        -et
       </i>
       is rather optional, even considered archaic).
      </p>
      <table class="wikitable">
       <tr>
        <th>
        </th>
        <th>
         Nominative
        </th>
        <th>
         Accusative
        </th>
       </tr>
       <tr>
        <td>
         first-person singular (I)
        </td>
        <td>
         én
        </td>
        <td>
         <i>
          engem(et)
         </i>
        </td>
       </tr>
       <tr>
        <td>
         second-person singular (you)
        </td>
        <td>
         te
        </td>
        <td>
         <i>
          téged(et)
         </i>
        </td>
       </tr>
       <tr>
        <td>
         third-person singular (he/she/it)
        </td>
        <td>
         ő
        </td>
        <td>
         őt
        </td>
       </tr>
       <tr>
        <td>
         first-person plural (we)
        </td>
        <td>
         mi
        </td>
        <td>
         <i>
          minket
         </i>
        </td>
       </tr>
       <tr>
        <td>
         second-person plural (you)
        </td>
        <td>
         ti
        </td>
        <td>
         <i>
          titeket
         </i>
        </td>
       </tr>
       <tr>
        <td>
         third-person plural (they)
        </td>
        <td>

        </td>
        <td>
         őket
        </td>
       </tr>
      </table>

      </p>

       <span class="mw-headline" id="Akkadian">
        Akkadian
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Accusative_case&amp;action=edit&amp;section=23" title="Edit section: Akkadian">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <dl>
       <dd>
        Nominative:
        <i>
         awīlum
        </i>
        (a/the man)
       </dd>
       <dd>
        Accusative:
        <i>

      <h4>
       <span class="mw-headline" id="Classical_Arabic">
        Classical Arabic
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Accusative_case&amp;action=edit&amp;section=24" title="Edit section: Classical Arabic">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h4>
      <p>
       In Arabic, the accusative case (also the subjunctive mood) is called
       <i>
        an-naṣb
       </i>
       (
       <span lang="ar" xml:lang="ar">
        نصب
       </span>
       ), and a word in the accusative case (also a verb in the subjunctive) is called
       <i>
        al-manṣūb
       </i>
       (‏
       <span dir="rtl" lang="ar" xml:lang="ar">
        <a class="extiw" href="https://en.wiktionary.org/wiki/%D9%85%D9%86%D8%B5%D9%88%D8%A8#Arabic" title="wikt:منصوب">
         المنصوب
        </a>
       </span>
       ‎), both from the verb
       <i>
        naṣaba
       </i>
       (
       <span lang="ar" xml:lang="ar">
        نصب
       </span>
       ) "set up". The accusative is used in 51 places,
       <sup class="reference" id="cite_ref-11">
        <a href="#cite_note-11">
         [11]
        </a>
       </sup>
       but mainly to mark the object of a verb and to form adverbs and prepositions.
      </p>
      <dl>
       <dd>
        Nominative:
        <i>
         rajulun
        </i>
        "a man",
        <i>
         ar-rajulu
        </i>
        "the man"
       </dd>
       <dd>
        Accusative:
        <i>
         ʾasʾalu rajulan
        </i>
        "I ask a man",
        <i>
         ʾasʾalu r-rajula
        </i>
        "I ask the man"
       </dd>
      </dl>
