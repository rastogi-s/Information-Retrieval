      <p>
       The
       <b>
        United States National Grid
       </b>
       (USNG) is a point reference system of grid references commonly used in the
       <a href="/wiki/United_States" title="United States">
        United States
       </a>
       . It provides a nationally consistent language of location in a user friendly format. It is similar in design to the
       <a class="mw-redirect" href="/wiki/National_grid_reference_system" title="National grid reference system">
        national grid reference systems
       </a>
       used throughout other nations. The USNG was developed by the
       <a href="/wiki/Federal_Geographic_Data_Committee" title="Federal Geographic Data Committee">
        Federal Geographic Data Committee
       </a>
       .
      </p>

      </p>

      <h2>
       <span class="mw-headline" id="Overview">
        Overview
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=United_States_National_Grid&amp;action=edit&amp;section=1" title="Edit section: Overview">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       The purpose of the USNG according to the FGDC, is to
       <i>
        "create a more interoperable environment for developing
        <a href="/wiki/Location-based_service" title="Location-based service">
         location-based services
        </a>
        within the United States and to increase the interoperability of location services appliances with printed map products by establishing a nationally consistent grid reference system as the preferred grid for NSDI applications. The U.S. National Grid is based on universally-defined coordinate and grid systems and can, therefore, be easily extended for use worldwide as a universal grid reference system."
       </i>
       <sup class="reference" id="cite_ref-1">
        <a href="#cite_note-1">
         [1]
        </a>
       </sup>
      </p>
      <p>
       It resembles the
       <a href="/wiki/Military_Grid_Reference_System" title="Military Grid Reference System">
        Military Grid Reference System
       </a>
       (MGRS). When the
       <a class="mw-redirect" href="/wiki/WGS84" title="WGS84">
        WGS84
       </a>
       <a class="mw-redirect" href="/wiki/Datum_(geodesy)" title="Datum (geodesy)">
        datum
       </a>
       or NAD83 datum is used, USNG and MGRS coordinates are "equivalent".
       <sup class="reference" id="cite_ref-2">
        <a href="#cite_note-2">
         [2]
        </a>
       </sup>
      </p>
      <p>
       In addition, it has the advantage of being easily plotted on USGS topographic and other properly gridded maps by using a simple "read right and then up" convention where the user measures to the East and then North in linear increments. The coordinates are easily translated to distance as they are actually in meters rather than the more complex degree based increments of latitude and longitude. Thus the distance between two coordinates can quickly be determined in the field.
      </p>
      <p>
       USNG 1 km
       <sup>
        2
       </sup>
       grid statewide polygons for select states can be downloaded from
       <a class="external free" href="http://usngcenter.org" rel="nofollow">
        http://usngcenter.org
       </a>
       . The website also provides more detailed information about the US National Grid.
      </p>
      <p>
       USNG allows a common system based on the US Military Grid system using landmarks like hills or streams, anything permanent. It allows these maps to be printed anywhere by responders totally unfamiliar with the local area. The degree of precision is determined by the number of digits in the coordinates, 23 67 = 1000 meters, 23451 67345 = 1 meter.
      </p>
