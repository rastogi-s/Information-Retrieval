      <p>
       This is a list of named
       <a href="/wiki/Lunar_craters" title="Lunar craters">
        lunar craters
       </a>
       . The large majority of these features are
       <a href="/wiki/Impact_crater" title="Impact crater">
        impact craters
       </a>
       . The
       <a href="/wiki/Planetary_nomenclature" title="Planetary nomenclature">
        crater nomenclature
       </a>
       is governed by the
       <a href="/wiki/International_Astronomical_Union" title="International Astronomical Union">
        International Astronomical Union
       </a>
       , and this listing only includes features that are officially recognized by that scientific society.
      </p>
