      <p>
       The following is a
       <b>
        list of
        <a class="mw-redirect" href="/wiki/Solar_variation" title="Solar variation">
         solar cycles
        </a>
       </b>
       (sometimes called
       <a href="/wiki/Sunspot" title="Sunspot">
        sunspot
       </a>
       cycles), tracked since 1755 following the original numbering proposed by Rudolf Wolf in the mid-19th century
       <sup class="reference" id="cite_ref-Kane_1-0">
        <a href="#cite_note-Kane-1">
         [1]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-2">
        <a href="#cite_note-2">
         [2]
        </a>
       </sup>
       The source data are the revised International Sunspot Numbers (ISN v2.0), as available at SILSO.
       <sup class="reference" id="cite_ref-3">
        <a href="#cite_note-3">
         [3]
        </a>
       </sup>
       Sunspot number counts exist since 1610
       <sup class="reference" id="cite_ref-4">
        <a href="#cite_note-4">
         [4]
        </a>
       </sup>
       but the cycle numbering is not well defined during the Maunder minimum
       <sup class="reference" id="cite_ref-5">
        <a href="#cite_note-5">
         [5]
        </a>
       </sup>
       <sup class="noprint Inline-Template noprint noexcerpt Template-Fact" style="white-space:nowrap;">
        [
        <i>
         <a class="mw-redirect" href="/wiki/Wikipedia:NOTRS" title="Wikipedia:NOTRS">
          <span title="per WP:CIRCULAR (November 2017)">
           better source needed
          </span>
         </a>
        </i>
        ]
       </sup>
       . It was proposed that one cycle might have been lost in the late 18th century
       <sup class="reference" id="cite_ref-6">
        <a href="#cite_note-6">
         [6]
        </a>
       </sup>
       , but this still remains not fully confirmed.
      </p>
      <p>
       The smoothing was done using the traditional SIDC smoothing formula.
       <sup class="reference" id="cite_ref-7">
        <a href="#cite_note-7">
         [7]
        </a>
       </sup>
       Other smoothing formulas exist, and they usually give slightly different values for the amplitude and timings of the solar cycles. An example is the Meeus smoothing formula,
       <sup class="reference" id="cite_ref-8">
        <a href="#cite_note-8">
         [8]
        </a>
       </sup>
       with related solar cycles characteristics available in this STCE news item.
       <sup class="reference" id="cite_ref-9">
        <a href="#cite_note-9">
         [9]
        </a>
       </sup>
      </p>
      <p>
       In the table below, the number of spotless days is the number between the maximum of the previous solar cycle and the maximum of the new solar cycle. As an example, there were 817 spotless days during the transit from solar cycle 23 to solar cycle 24.
      </p>
