      <p>
       An
       <b>
        orbital node
       </b>
       is either of the two points where an
       <a href="/wiki/Orbit" title="Orbit">
        orbit
       </a>
       <a class="new" href="/w/index.php?title=Intersection_(Euclidian_geometry)&amp;action=edit&amp;redlink=1" title="Intersection (Euclidian geometry) (page does not exist)">
        intersects
       </a>
       a
       <a href="/wiki/Plane_of_reference" title="Plane of reference">
        plane of reference
       </a>
       to which it is inclined.
       <sup class="reference" id="cite_ref-col_1-0">
        <a href="#cite_note-col-1">
         [1]
        </a>
       </sup>
       A
       <a href="/wiki/Non-inclined_orbit" title="Non-inclined orbit">
        non-inclined orbit
       </a>
       , which is
       <a href="/wiki/Coplanarity" title="Coplanarity">
        contained
       </a>
       in the reference plane, has no nodes.
      </p>
      <p>
      </p>

      <h2>
       <span class="mw-headline" id="Planes_of_reference">
        Planes of reference
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Orbital_node&amp;action=edit&amp;section=1" title="Edit section: Planes of reference">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       Common
       <a href="/wiki/Plane_of_reference" title="Plane of reference">
        planes of reference
       </a>
       include the following:
      </p>
      <ul>
       <li>
        For a
        <a href="/wiki/Geocentric_orbit" title="Geocentric orbit">
         geocentric orbit
        </a>
        ,
        <a href="/wiki/Earth" title="Earth">
         Earth
        </a>
        's
        <a class="mw-redirect" href="/wiki/Equatorial_plane" title="Equatorial plane">
         equatorial plane
        </a>
        . In this case, non-inclined orbits are called
        <i>
         equatorial
        </i>
        .
        <sup class="reference" id="cite_ref-darlinglon_2-0">
         <a href="#cite_note-darlinglon-2">
          [2]
         </a>
        </sup>
       </li>
       <li>
        For a
        <a href="/wiki/Heliocentric_orbit" title="Heliocentric orbit">
         heliocentric orbit
        </a>
        , the
        <a href="/wiki/Ecliptic" title="Ecliptic">
         ecliptic
        </a>
        or
        <a href="/wiki/Invariable_plane" title="Invariable plane">
         invariable plane
        </a>
        . In this case, non-inclined orbits are called
        <i>
         ecliptic
        </i>
        .
        <sup class="reference" id="cite_ref-darlinglon_2-1">
         <a href="#cite_note-darlinglon-2">
          [2]
         </a>
        </sup>
       </li>
       <li>
        For an orbit outside the
        <a href="/wiki/Solar_System" title="Solar System">
         Solar System
        </a>
        , the plane through the
        <a href="/wiki/Primary_(astronomy)" title="Primary (astronomy)">
         primary
        </a>
        perpendicular to a line through the observer and the primary (called the
        <i>
         <a class="new" href="/w/index.php?title=Plane_of_the_sky&amp;action=edit&amp;redlink=1" title="Plane of the sky (page does not exist)">
          plane of the sky
         </a>
        </i>
        ).
        <sup class="reference" id="cite_ref-3">
         <a href="#cite_note-3">
          [3]
         </a>
        </sup>
       </li>
      </ul>
      <h2>
       <span class="mw-headline" id="Node_distinction">
        Node distinction
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Orbital_node&amp;action=edit&amp;section=2" title="Edit section: Node distinction">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <div class="thumb tright">
       <div class="thumbinner" style="width:242px;">
        <a class="image" href="/wiki/File:Planet_orbit_nodes_2_animation.gif">
         <img alt="" class="thumbimage" data-file-height="300" data-file-width="800" height="90" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b9/Planet_orbit_nodes_2_animation.gif/240px-Planet_orbit_nodes_2_animation.gif" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b9/Planet_orbit_nodes_2_animation.gif/360px-Planet_orbit_nodes_2_animation.gif 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b9/Planet_orbit_nodes_2_animation.gif/480px-Planet_orbit_nodes_2_animation.gif 2x" width="240"/>
        </a>
        <div class="thumbcaption">
         <div class="magnify">
          <a class="internal" href="/wiki/File:Planet_orbit_nodes_2_animation.gif" title="Enlarge">
          </a>
         </div>
         Animation about nodes of two elliptic trajectories. (Click onto image.)
        </div>
       </div>
      </div>
      <p>
       If a reference direction from one side of the plane of reference to the other is defined, the two nodes can be distinguished. For geocentric and heliocentric orbits, the
       <b>
        ascending node
       </b>
       (or
       <b>
        north node
       </b>
       ) is where the orbiting object moves north through the plane of reference, and the
       <b>
        descending node
       </b>
       (or
       <b>
        south node
       </b>
       ) is where it moves south through the plane.
       <sup class="reference" id="cite_ref-darlingasc_4-0">
        <a href="#cite_note-darlingasc-4">
         [4]
        </a>
       </sup>
       In the case of objects outside the Solar System, the ascending node is the node where the orbiting secondary passes away from the observer, and the descending node is the node where it moves towards the observer.
       <sup class="reference" id="cite_ref-5">
        <a href="#cite_note-5">
         [5]
        </a>
       </sup>
       <sup>
        , p. 137.
       </sup>
      </p>
      <p>
       The position of the node may be used as one of a set of parameters, called
       <i>
        <a href="/wiki/Orbital_elements" title="Orbital elements">
         orbital elements
        </a>
       </i>
       , which describe the orbit. This is done by specifying the
       <a href="/wiki/Longitude_of_the_ascending_node" title="Longitude of the ascending node">
        longitude of the ascending node
       </a>
       (or, sometimes, the
       <a class="mw-redirect" href="/wiki/Longitude_of_the_node" title="Longitude of the node">
        longitude of the node
       </a>
       .)
      </p>
      <p>
       The
       <b>
        line of nodes
       </b>
       is the intersection of the object's orbital plane with the plane of reference. It passes through the two nodes.
       <sup class="reference" id="cite_ref-darlinglon_2-2">
        <a href="#cite_note-darlinglon-2">
         [2]
        </a>
       </sup>
      </p>
      <h2>
       <span class="mw-headline" id="Symbols_and_nomenclature">
        Symbols and nomenclature
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Orbital_node&amp;action=edit&amp;section=3" title="Edit section: Symbols and nomenclature">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       The symbol of the ascending node is
       <a class="image" href="/wiki/File:Northnode-symbol.png">
        <img alt="Northnode-symbol.png" data-file-height="20" data-file-width="20" height="20" src="//upload.wikimedia.org/wikipedia/commons/b/b5/Northnode-symbol.png" width="20"/>
       </a>
       (
       <a href="/wiki/Unicode" title="Unicode">
        Unicode
       </a>
       : U+260A, ☊), and the symbol of the descending node is
       <a class="image" href="/wiki/File:Southnode-symbol.png">
        <img alt="Southnode-symbol.png" data-file-height="20" data-file-width="20" height="20" src="//upload.wikimedia.org/wikipedia/commons/e/e2/Southnode-symbol.png" width="20"/>
       </a>
       (
       <a href="/wiki/Unicode" title="Unicode">
        Unicode
       </a>
       : U+260B, ☋). In medieval and early modern times the ascending and descending nodes were called the
       <i>
        dragon's head
       </i>
       (
       <a href="/wiki/Latin" title="Latin">
        Latin
       </a>
       :
       <i>
        caput draconis
       </i>
       ,
       <a href="/wiki/Arabic" title="Arabic">
        Arabic
       </a>
       :
       <i>
        ra's al-jauzahar
       </i>
       ) and
       <i>
        dragon's tail
       </i>
       (
       <a class="mw-redirect" href="/wiki/Latin_language" title="Latin language">
        Latin
       </a>
       :
       <i>
        <span lang="la" xml:lang="la">
         cauda draconis
        </span>
       </i>
       ), respectively.
       <sup class="reference" id="cite_ref-6">
        <a href="#cite_note-6">
         [6]
        </a>
       </sup>
       <sup>
        , p. 141;
       </sup>
       <sup class="reference" id="cite_ref-7">
        <a href="#cite_note-7">
         [7]
        </a>
       </sup>
       <sup>
        , p. 245.
       </sup>
       These terms originally referred to the times when the
       <a href="/wiki/Moon" title="Moon">
        Moon
       </a>
       crossed the apparent path of the sun in the sky. Also, corruptions of the Arabic term such as
       <i>
        ganzaar
       </i>
       ,
       <i>
        genzahar
       </i>
       ,
       <i>
        geuzaar
       </i>
       and
       <i>
        zeuzahar
       </i>
       were used in the medieval West to denote either of the nodes.
       <sup class="reference" id="cite_ref-8">
        <a href="#cite_note-8">
         [8]
        </a>
       </sup>
       <sup>
        , pp. 196–197;
       </sup>
       <sup class="reference" id="cite_ref-9">
        <a href="#cite_note-9">
         [9]
        </a>
       </sup>
       <sup>
        , p. 65;
       </sup>
       <sup class="reference" id="cite_ref-10">
        <a href="#cite_note-10">
         [10]
        </a>
       </sup>
       <sup>
        , pp. 95–96.
       </sup>
       The
       <a href="/wiki/Greek_language" title="Greek language">
        Greek
       </a>
       terms αναβιβάζων and καταβιβάζων were also used for the ascending and descending nodes, giving rise to the
       <a href="/wiki/English_language" title="English language">
        English
       </a>
       words
       <i>
        anabibazon
       </i>
       and
       <i>
        catabibazon
       </i>
       .
       <sup class="reference" id="cite_ref-11">
        <a href="#cite_note-11">
         [11]
        </a>
       </sup>
       <sup>
        ;
       </sup>
       <sup class="reference" id="cite_ref-12">
        <a href="#cite_note-12">
         [12]
        </a>
       </sup>
       <sup>
        , ¶27.
       </sup>
      </p>
      <h2>
       <span class="mw-headline" id="Lunar_nodes">
        Lunar nodes
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Orbital_node&amp;action=edit&amp;section=4" title="Edit section: Lunar nodes">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <div class="hatnote navigation-not-searchable" role="note">
       Main article:
       <a href="/wiki/Lunar_node" title="Lunar node">
        Lunar node
       </a>
      </div>
      <p>
       For the
       <a href="/wiki/Orbit_of_the_Moon" title="Orbit of the Moon">
        orbit of the Moon
       </a>
       around
       <a href="/wiki/Earth" title="Earth">
        Earth
       </a>
       , the
       <a href="/wiki/Plane_of_reference" title="Plane of reference">
        plane
       </a>
       is taken to be the
       <a href="/wiki/Ecliptic" title="Ecliptic">
        ecliptic
       </a>
       , not the
       <a href="/wiki/Celestial_equator" title="Celestial equator">
        equatorial plane
       </a>
       . The
       <a href="/wiki/Gravity" title="Gravity">
        gravitational
       </a>
       pull of the
       <a href="/wiki/Sun" title="Sun">
        Sun
       </a>
       upon the Moon causes
       <a href="/wiki/Lunar_node" title="Lunar node">
        its nodes
       </a>
       to gradually
       <a href="/wiki/Lunar_precession" title="Lunar precession">
        precess
       </a>
       westward, completing a cycle in approximately 18.6 years.
