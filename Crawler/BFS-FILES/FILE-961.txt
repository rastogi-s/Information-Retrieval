   <h1 class="firstHeading" id="firstHeading" lang="en">
    Steady state
   </h1>
   <div class="mw-body-content" id="bodyContent">
    <div class="noprint" id="siteSub">

      <p>
       In
       <a href="/wiki/Systems_theory" title="Systems theory">
        systems theory
       </a>
       , a
       <a href="/wiki/System" title="System">
        system
       </a>
       or a
       <a href="/wiki/Process_theory" title="Process theory">
        process
       </a>
       is in a
       <b>
        steady state
       </b>
       if the variables (called
       <a href="/wiki/State_variable" title="State variable">
        state variables
       </a>
       ) which define the behavior of the system or the process are unchanging in time.
       <sup class="reference" id="cite_ref-:0_1-0">
        <a href="#cite_note-:0-1">
         [1]
        </a>
       </sup>
       In
       <a class="mw-redirect" href="/wiki/Continuous_time" title="Continuous time">
        continuous time
       </a>
       , this means that for those properties

        p
       </i>
       of the system, the

        partial derivative
       </a>
       with respect to time is zero and remains so:

      <p>
       The concept of a steady state has relevance in many fields, in particular
       <a href="/wiki/Thermodynamics" title="Thermodynamics">
        thermodynamics
       </a>
       ,
       <a class="mw-redirect" href="/wiki/Steady_state_economy" title="Steady state economy">
        economics
       </a>
       , and
       <a href="/wiki/Engineering" title="Engineering">
        engineering
       </a>
       . If a system is in a steady state, then the recently observed behavior of the system will continue into the future.
       <sup class="reference" id="cite_ref-:0_1-1">
        <a href="#cite_note-:0-1">
         [1]
        </a>
       </sup>
       In
       <a href="/wiki/Stochastic" title="Stochastic">
        stochastic
       </a>
       systems, the probabilities that various states will be repeated will remain constant. See for example
       <a href="/wiki/Linear_difference_equation#Conversion_to_homogeneous_form" title="Linear difference equation">
        Linear difference equation#Conversion to homogeneous form
       </a>
       for the derivation of the steady state.
      </p>
      <p>
       In many systems, a steady state is not achieved until some time after the system is started or initiated. This initial situation is often identified as a
       <a href="/wiki/Transient_state" title="Transient state">
        transient state
       </a>
       , start-up or warm-up period.
       <sup class="reference" id="cite_ref-:0_1-2">
        <a href="#cite_note-:0-1">
         [1]
        </a>
       </sup>
       <sup class="reference" id="cite_ref-Ricardo_Antunes_2-0">
        <a href="#cite_note-Ricardo_Antunes-2">
         [2]
        </a>
       </sup>
       For example, while the flow of
       <a href="/wiki/Fluid" title="Fluid">
        fluid
       </a>
       through a tube or electricity through a network could be in a steady state because there is a constant flow of fluid or electricity, a tank being drained or filled with fluid is a system in transient state, because its volume of fluid changes with time.
      </p>
      <p>
       Often, a steady state is approached
       <a class="mw-redirect" href="/wiki/Asymptotically" title="Asymptotically">
        asymptotically
       </a>
       . An unstable system is one that diverges from the steady state. See for example
       <a href="/wiki/Linear_difference_equation#Stability" title="Linear difference equation">
        Linear difference equation#Stability
       </a>
       .
      </p>
      <p>
       In
       <a href="/wiki/Chemistry" title="Chemistry">
        chemistry
       </a>
       , a steady state is a more general situation than
       <a href="/wiki/Dynamic_equilibrium" title="Dynamic equilibrium">
        dynamic equilibrium
       </a>
       . While a dynamic equilibrium occurs when two or more
       <a href="/wiki/Reversible_reaction" title="Reversible reaction">
        reversible processes
       </a>
       occur at the same rate, and such a system can be said to be in a steady state, a system that is in a steady state may not necessarily be in a state of dynamic equilibrium, because some of the processes involved are not reversible.
      </p>
      <p>
      </p>

      <h3>
       <span class="mw-headline" id="Economics">
        Economics
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Steady_state&amp;action=edit&amp;section=2" title="Edit section: Economics">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <div class="hatnote navigation-not-searchable" role="note">
       Main article:
       <a class="mw-redirect" href="/wiki/Steady_state_economy" title="Steady state economy">
        Steady state economy
       </a>
      </div>
      <p>
       A
       <i>
        steady state economy
       </i>
       is an economy (especially a national economy but possibly that of a city, a region, or the world) of stable size featuring a stable population and stable
       <a href="/wiki/Consumption_(economics)" title="Consumption (economics)">
        consumption
       </a>
       that remain at or below
       <a href="/wiki/Carrying_capacity" title="Carrying capacity">
        carrying capacity
       </a>
       . In the
       <a href="/wiki/Economic_growth" title="Economic growth">
        economic growth
       </a>
       model of
       <a href="/wiki/Robert_Solow" title="Robert Solow">
        Robert Solow
       </a>
       and
       <a href="/wiki/Trevor_Swan" title="Trevor Swan">
        Trevor Swan
       </a>
       , the steady state occurs when gross
       <a href="/wiki/Investment_(macroeconomics)" title="Investment (macroeconomics)">
        investment
       </a>
       in
       <a href="/wiki/Physical_capital" title="Physical capital">
        physical capital
       </a>
       equals
       <a href="/wiki/Depreciation_(economics)" title="Depreciation (economics)">
        depreciation
       </a>
       and the economy reaches
       <a href="/wiki/Economic_equilibrium" title="Economic equilibrium">
        economic equilibrium
       </a>
       , which may occur during a period of growth.
      </p>
      <h3>
       <span class="mw-headline" id="Electronics">
        Electronics
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Steady_state&amp;action=edit&amp;section=3" title="Edit section: Electronics">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <div class="hatnote navigation-not-searchable" role="note">
       Main article:
       <a href="/wiki/Steady_state_(electronics)" title="Steady state (electronics)">
        Steady state (electronics)
       </a>
      </div>
      <p>
       In electronics,
       <i>
        steady state
       </i>
       is an equilibrium condition of a circuit or network that occurs as the effects of
       <a class="mw-redirect" href="/wiki/Transient_(electricity)" title="Transient (electricity)">
        transients
       </a>
       are no longer important.
      </p>
      <p>
       Steady state determination is an important topic, because many design specifications of electronic systems are given in terms of the steady-state characteristics. Periodic steady-state solution is also a prerequisite for small signal dynamic modeling. Steady-state analysis is therefore an indispensable component of the design process.
      </p>
      <p>
       In some cases, it is useful to consider
       <a href="/wiki/Constant_envelope" title="Constant envelope">
        constant envelope
       </a>
       vibration—vibration that never settles down to motionlessness, but continues to move at constant amplitude—a kind of steady-state condition.
      </p>

      <p>
       In
       <a href="/wiki/Chemistry" title="Chemistry">
        chemistry
       </a>
       ,
       <a href="/wiki/Thermodynamics" title="Thermodynamics">
        thermodynamics
       </a>
       , and other
       <a href="/wiki/Chemical_engineering" title="Chemical engineering">

       , a
       <i>
        steady state
       </i>
       is a situation in which all
       <a class="mw-redirect" href="/wiki/Thermodynamic_variable" title="Thermodynamic variable">
        state variables
       </a>
       are constant in spite of ongoing processes that strive to change them. For an entire system to be at steady state, i.e. for all state variables of a system to be constant, there must be a flow through the system (compare
       <a href="/wiki/Mass_balance" title="Mass balance">
        mass balance
       </a>
       ). One of the simplest examples of such a system is the case of a bathtub with the tap open but without the bottom plug: after a certain time the water flows in and out at the same rate, so the water level (the state variable being Volume) stabilizes and the system is at steady state. Of course the Volume stabilizing inside the tub depends on the size of the tub, the diameter of the exit hole and the flowrate of water in. Since the tub can overflow, eventually a steady state can be reached where the water flowing in equals the overflow plus the water out through the drain.
      </p>
      <p>
       A steady state flow process requires conditions at all points in an apparatus remain constant as time changes. There must be no accumulation of mass or energy over the time period of interest. The same mass flow rate will remain constant in the flow path through each element of the system.
       <sup class="reference" id="cite_ref-3">
        <a href="#cite_note-3">
         [3]
        </a>
       </sup>
       Thermodynamic properties may vary from point to point, but will remain unchanged at any given point.
       <sup class="reference" id="cite_ref-4">
        <a href="#cite_note-4">
         [4]
        </a>
       </sup>
      </p>
      <h3>
       <span class="mw-headline" id="Electrical_engineering">
        Electrical engineering
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Steady_state&amp;action=edit&amp;section=5" title="Edit section: Electrical engineering">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <div class="hatnote navigation-not-searchable" role="note">
       Main article:
       <a href="/wiki/Steady_state_(electronics)" title="Steady state (electronics)">
        Steady state (electronics)
       </a>
      </div>
      <p>
       Sinusoidal Steady State Analysis is a method for analyzing
       <a href="/wiki/Alternating_current" title="Alternating current">
        alternating current
       </a>
       circuits using the same techniques as for solving DC circuits.
       <sup class="reference" id="cite_ref-5">
        <a href="#cite_note-5">
         [5]
        </a>
       </sup>
      </p>
      <p>
       The ability of an electrical machine or
       <a href="/wiki/Electric_power_system" title="Electric power system">
        power system
       </a>
       to regain its original/previous state is called Steady State Stability.
       <sup class="reference" id="cite_ref-6">
        <a href="#cite_note-6">
         [6]
        </a>
       </sup>
      </p>
      <p>
       The stability of a system refers to the ability of a system to return to its steady state when subjected to a disturbance. As mentioned before, power is generated by synchronous generators that operate in synchronism with the rest of the system. A generator is synchronized with a bus when both of them have same
       <a href="/wiki/Frequency" title="Frequency">
        frequency
       </a>
       ,
       <a href="/wiki/Voltage" title="Voltage">
        voltage
       </a>
       and
       <a href="/wiki/Polyphase_system" title="Polyphase system">
        phase sequence
       </a>
       . We can thus define the power system stability as the ability of the power system to return to steady state without losing synchronicity. Usually power system stability is categorized into
       <b>
        Steady State, Transient and Dynamic Stability
       </b>
      </p>
      <p>
       Steady State Stability studies are restricted to small and gradual changes in the system operating conditions. In this we basically concentrate on restricting the bus voltages close to their nominal values. We also ensure that phase angles between two buses are not too large and check for the overloading of the power equipment and transmission lines. These checks are usually done using power flow studies.
      </p>
      <p>
       Transient Stability involves the study of the power system following a major disturbance. Following a large disturbance in the synchronous alternator the machine power (load) angle changes due to sudden acceleration of the rotor shaft. The objective of the transient stability study is to ascertain whether the load angle returns to a steady value following the clearance of the disturbance.
      </p>
      <p>
       The ability of a power system to maintain stability under continuous small disturbances is investigated under the name of Dynamic Stability (also known as small-signal stability). These small disturbances occur due random fluctuations in loads and generation levels. In an interconnected power system, these random variations can lead catastrophic failure as this may force the rotor angle to increase steadily.
      </p>
      <h3>
       <span class="mw-headline" id="Mechanical_engineering">
        Mechanical engineering
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Steady_state&amp;action=edit&amp;section=6" title="Edit section: Mechanical engineering">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <p>
       When a periodic force is applied to a mechanical system, it will typically reach steady state after going through some transient behavior. This is often observed in
       <a href="/wiki/Vibration" title="Vibration">
        vibrating
       </a>
       systems, such as a
       <a href="/wiki/Pendulum_clock" title="Pendulum clock">
        clock pendulum
       </a>
       , but can happen with any type of
       <a class="mw-redirect mw-disambig" href="/wiki/Stability_(disambiguation)#Engineering" title="Stability (disambiguation)">
        stable
       </a>
       or semi-stable dynamic system. The length of the transient state will depend on the initial conditions of the system. Given certain initial conditions a system may be in steady state from the beginning.
      </p>
      <h3>
       <span class="mw-headline" id="Physiology">
        Physiology
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Steady_state&amp;action=edit&amp;section=7" title="Edit section: Physiology">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h3>
      <div class="hatnote navigation-not-searchable" role="note">
       Main article:
       <a href="/wiki/Homeostasis" title="Homeostasis">
        Homeostasis
       </a>
      </div>
      <p>
       <b>
        Homeostasis
       </b>
       (from
       <a href="/wiki/Ancient_Greek" title="Ancient Greek">
        Greek
       </a>
       ὅμοιος,
       <i>
        hómoios
       </i>
       , "similar" and στάσις,
       <i>
        stásis
       </i>
       , "standing still") is the property of a system that regulates its internal environment and tends to maintain a stable, constant condition. Typically used to refer to a living
       <a href="/wiki/Organism" title="Organism">
        organism
       </a>
       , the concept came from that of
       <a class="mw-redirect" href="/wiki/Milieu_interieur" title="Milieu interieur">
        milieu interieur
       </a>
       that was created by
       <a href="/wiki/Claude_Bernard" title="Claude Bernard">
        Claude Bernard
       </a>
       and published in 1865. Multiple dynamic equilibrium adjustment and regulation mechanisms make homeostasis possible.
      </p>
