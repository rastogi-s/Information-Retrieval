      <p>
       The
       <b>
        particle number
       </b>
       (or
       <b>
        number of particles
       </b>
       ) of a
       <a href="/wiki/Thermodynamic_system" title="Thermodynamic system">
        thermodynamic system
       </a>
       , conventionally indicated with the letter
       <i>
        N
       </i>
       , is the number of constituent particles in that system.
       <sup class="reference" id="cite_ref-HoP_1-0">
        <a href="#cite_note-HoP-1">
         [1]
        </a>
       </sup>
       The particle number is a fundamental parameter in
       <a href="/wiki/Thermodynamics" title="Thermodynamics">
        thermodynamics
       </a>
       which is
       <a href="/wiki/Conjugate_variables_(thermodynamics)" title="Conjugate variables (thermodynamics)">
        conjugate
       </a>
       to the
       <a href="/wiki/Chemical_potential" title="Chemical potential">
        chemical potential
       </a>
       . Unlike most
       <a class="mw-redirect" href="/wiki/Physical_quantities" title="Physical quantities">
        physical quantities
       </a>
       , particle number is a
       <a href="/wiki/Dimensionless_quantity" title="Dimensionless quantity">
        dimensionless quantity
       </a>
       . It is an
       <a class="mw-redirect" href="/wiki/Extensive_parameter" title="Extensive parameter">
        extensive parameter
       </a>
       , as it is directly proportional to the size of the system under consideration, and thus meaningful only for
       <a href="/wiki/Thermodynamic_system#Closed_system" title="Thermodynamic system">
        closed systems
       </a>
       .
      </p>
      <p>
       A
       <b>
        constituent particle
       </b>
       is one that cannot be broken into smaller pieces at the scale of
       <a href="/wiki/Energy" title="Energy">
        energy
       </a>
       <i>
        k·T
       </i>
       involved in the process (where
       <i>
        k
       </i>
       is the
       <a href="/wiki/Boltzmann_constant" title="Boltzmann constant">
        Boltzmann constant
       </a>
       and
       <i>
        T
       </i>
       is the
       <a href="/wiki/Temperature" title="Temperature">
        temperature
       </a>
       ). For example, for a thermodynamic system consisting of a
       <a href="/wiki/Piston" title="Piston">
        piston
       </a>
       containing water vapour, the particle number is the number of water molecules in the system. The meaning of constituent particle, and thereby of particle number, is thus temperature-dependent.
      </p>

      </p>

      <h2>
       <span class="mw-headline" id="Determining_the_particle_number">
        Determining the particle number
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Particle_number&amp;action=edit&amp;section=1" title="Edit section: Determining the particle number">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       The concept of particle number has a main role in
       <a href="/wiki/Theoretical_physics" title="Theoretical physics">
        theoretical
       </a>
       considerations. In situations where the actual particle number of a given thermodynamical system needs to be determined, mainly in
       <a href="/wiki/Chemistry" title="Chemistry">
        chemistry
       </a>
       , it is not practically possible to measure it directly by counting the particles. If the material is homogeneous and has a known
       <a href="/wiki/Amount_of_substance" title="Amount of substance">
        amount of substance
       </a>
       <i>
        n
       </i>
       expressed in
       <a href="/wiki/Mole_(unit)" title="Mole (unit)">
        moles
       </a>
       , the particle number
       <i>
        N
       </i>
       can be found by the relation
      </p>
      <dl>
       <dd>
        <i>
         N
        </i>
        =
        <i>
         nN
         <sub>
          A
         </sub>
        </i>
        ,
       </dd>
      </dl>
      <p>
       where
       <i>
        N
        <sub>
         A
        </sub>
       </i>
       is the
       <a href="/wiki/Avogadro_constant" title="Avogadro constant">
        Avogadro constant
       </a>
       .
       <sup class="reference" id="cite_ref-HoP_1-1">
        <a href="#cite_note-HoP-1">
         [1]
        </a>
       </sup>
      </p>
      <h2>
       <span class="mw-headline" id="Particle_number_density">
        Particle number density
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Particle_number&amp;action=edit&amp;section=2" title="Edit section: Particle number density">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       A related
       <a class="mw-redirect" href="/wiki/Intensive_parameter" title="Intensive parameter">
        intensive system parameter
       </a>
       is given by the
       <b>
        particle
        <a href="/wiki/Number_density" title="Number density">
         number density
        </a>
       </b>
       , obtained by dividing the particle number of a system by its
       <a href="/wiki/Volume" title="Volume">
        volume
       </a>
       . This parameter is often denoted by a lower-case letter
       <i>
        n
       </i>
       .
      </p>
      <h2>
       <span class="mw-headline" id="In_quantum_mechanics">
        In quantum mechanics
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Particle_number&amp;action=edit&amp;section=3" title="Edit section: In quantum mechanics">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       In
       <a class="mw-redirect" href="/wiki/Quantum_mechanical" title="Quantum mechanical">
        quantum mechanical
       </a>
       processes the total number of particles may not be preserved. The concept is therefore generalized to the
       <a href="/wiki/Particle_number_operator" title="Particle number operator">
        particle number operator
       </a>
       , that is, the
       <a href="/wiki/Observable" title="Observable">
        observable
       </a>
       that counts the number of constituent particles.
       <sup class="reference" id="cite_ref-2">
        <a href="#cite_note-2">
         [2]
        </a>
       </sup>
       In
       <a href="/wiki/Quantum_field_theory" title="Quantum field theory">
        quantum field theory
       </a>
       , the particle number operator (see
       <a href="/wiki/Fock_state" title="Fock state">
        Fock state
       </a>
       ) is conjugate to the phase of the
       <i>
        classical
       </i>
       wave (see
       <a class="mw-redirect" href="/wiki/Coherent_state" title="Coherent state">
        coherent state
       </a>
       ).
      </p>
      <h2>
       <span class="mw-headline" id="In_air_quality">
        In air quality
       </span>
       <span class="mw-editsection">
        <span class="mw-editsection-bracket">
         [
        </span>
        <a href="/w/index.php?title=Particle_number&amp;action=edit&amp;section=4" title="Edit section: In air quality">
         edit
        </a>
        <span class="mw-editsection-bracket">
         ]
        </span>
       </span>
      </h2>
      <p>
       One measure of
       <a href="/wiki/Air_pollution" title="Air pollution">
        air pollution
       </a>
       used in air quality standards is the atmospheric concentration of
       <a class="mw-redirect" href="/wiki/Atmospheric_particulate_matter" title="Atmospheric particulate matter">
        particulate matter
       </a>
       . This measure is usually expressed in μg/m
       <sup>
        3
       </sup>
       (
       <a class="mw-redirect" href="/wiki/Micrograms" title="Micrograms">
        micrograms
       </a>
       per cubic metre). In the upcoming emission norm for non-road mobile machinery a new particle number legislation is registered. In this case PN expresses a quantity of particles.
      </p>
